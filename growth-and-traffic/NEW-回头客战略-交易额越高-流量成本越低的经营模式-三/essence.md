# 回头客战略：交易额越高，流量成本越低的经营模式

> **原书名**: Delivering Happiness — A Path to Profits, Passion, and Purpose
> **作者**: 谢家华 (Tony Hsieh) — Zappos创始人兼CEO
> **中文版**: 文汇出版社，2017年

---

## 核心理念

**更幸福的员工 = 更幸福的顾客 = 赚钱的企业 + 有意义的生活**

Zappos的商业模式核心：把本应投到广告上的资金运用到提高客户服务和顾客体验上，让顾客的口碑为我们做宣传。促进Zappos发展的最大推动力是**回头客和口碑**。

## 第一部分：客户服务即营销

### 1. 品牌承诺的演变

| 年份 | 品牌承诺 |
|------|----------|
| 1999 | 提供鞋界种类最繁多的鞋子 |
| 2003 | 客户服务 |
| 2005 | 以文化和核心价值观作为平台 |
| 2007 | 个人情感的连接 |
| 2009 | 分享幸福 |

### 2. 客户服务的核心实践

- **免费双向运送**：客户可订5双试穿，不合适的免费退回——当作市场开发费用
- **365天退货政策**：消除客户决策风险
- **电话号码放在网站每页最上**：一天24小时、一周7天接电话，真正希望和客户讲话
- **不计算通话时间**：最长通话10小时29分钟，不推销，不设预设台词
- **建立个人情感联系（PEC）**：让每次通话都发挥员工个性
- **帮客户找竞争对手的产品**：卖完时帮客户在3个其他网站查找，宁可失去单笔交易
- **意外升级运送**：选择免费标准运送的忠实客户，升级为隔夜送达
- **仓库24×7运作**：最大化客户体验而非取货效率

### 3. 电话是最被低估的品牌工具

> "电话仍然是建立品牌的最好工具。你至少得到了顾客5~10分钟的全部注意力。"

- 大多数电话不带来订单，但每位客户一生至少打一次电话
- 利用这个机会给客户建立永久记忆
- 有时客户打电话只是需要人陪聊天、问着装意见

### 4. 客户服务十大方法

1. 将客户服务作为整个公司优先考虑的事，不仅是部门的事
2. 把"WOW！"变为动词，成为公司日常词汇
3. 授权和信任客服代表
4. 敢于"炒掉"辱骂员工的客户
5. 不计通话时间，不强制推销，不用预设台词
6. 不隐瞒免费电话号码
7. 每次通话都视为品牌投资，而非要减少的开销
8. 全公司庆祝特别好的服务
9. 寻找并雇用对客户服务有激情的人
10. 给每个人（客户、员工、供应商）最好的服务

## 第二部分：公司文化

### 1. 文化 > 一切

> "品牌只是反映公司文化的滞后指标。你的文化就是你的品牌。"

**BCP战略**：品牌(Brand)、文化(Culture)、培训(Pipeline)——唯一可持续的竞争优势。其他一切都能且最终都会被复制。

### 2. Zappos十大核心价值观

1. **通过服务让人们感到惊叹：WOW！** — 做出超越预期、以情动人的事
2. **拥抱并驱动变革** — 变化是常态，自下而上推动
3. **创造欢乐及一点点搞怪** — 鼓励个性和创新思维
4. **勇于冒险，敢于创新，开放思想** — 不怕犯错，犯错说明冒了足够的险
5. **积极进取和不断学习** — 释放内在潜力
6. **通过沟通建立开放和诚实的关系** — 倾听与沟通同样重要
7. **建立积极的团队，塑造家庭精神** — 最好的领导是公仆
8. **追求事半功倍** — "好"是"极好"的敌人
9. **充满激情和决断力** — 不接受"不"的答案
10. **虚怀若谷** — 宁牺牲短期利益保护长期文化

### 3. 文化建设实践

- **文化书**：每年出版，员工自由投稿描述对公司文化的理解，不编辑不删改
- **双重面试**：一套考技术能力，一套考文化匹配，必须同时通过
- **培训期付费请离**：新员工培训第1周出价$2000请不匹配者离开（仅<1%接受）
- **认脸游戏**：登录系统时随机显示员工照片，要求选出姓名
- **全员进出经由中心接待区**：增加偶然碰面交流的机会
- **每月《什么都可以问》简报**：匿名问答，促进透明

### 4. 招聘原则

- **宁愿牺牲短期利益**拒绝有才能但文化不合的人
- "虚怀若谷"是影响最终用人决定最大的一条
- 为每条核心价值观制定对应面试提问方案

## 第三部分：幸福框架

### 1. 幸福四要素

| 要素 | 说明 | 商业应用 |
|------|------|----------|
| 控制力 | 感到对生活有掌控 | 技能项目让员工自行决定薪资增长 |
| 进步 | 感到在不断前进 | 18个月晋升拆成3次6个月微调 |
| 连通性 | 关系网的数量和深度 | 强调文化，建立深厚同事关系 |
| 愿景/意义 | 成为更广阔事业的一部分 | 超越金钱的更高使命 |

### 2. 幸福三层级（应用马斯洛需求理论）

**客户**：满足期望 → 满足欲望 → 满足潜在需求
- 例：收到正确产品 → 免费运送 → 意外升级隔夜送达

**员工**：金钱 → 认可 → 意义

**投资者**：交易一致 → 关系一致 → 遗产

### 3. 三种幸福形式（持续时间由短到长）

1. **乐趣**（摇滚明星式）——持续最短，刺激消失即消退
2. **激情/飘然**——最佳表现结合最深参与（"在状态"）
3. **更高使命/意义**——持续最久，成为比自己更广阔事业的一部分

> **正确策略**：先追求使命，再追求激情，最后加上乐趣（而非相反）

### 4. 幸福分形体

- 个人层面：乐趣 → 激情 → 使命
- 公司层面：利润 → 激情 → 使命

两者构成分形体——放大缩小结构相同。以幸福为组织原则，简单公式可推导出复杂体系。

## 第四部分：商业战略与教训

### 1. 从德州扑克学到的商业智慧

- **选择牌桌是最重要的决策**：进入正确行业比完美执行更重要
- **区分正确决定和个别结果**：不要因单次输赢改变长期策略
- **确保有足够的资金**：只玩你输得起的游戏
- **不断换桌**：不要因惯性停留在不适合的牌桌

### 2. 成长中的关键转折

| 阶段 | 关键决策 |
|------|----------|
| 2000 | 裁员+减薪，聚焦现有客户成为回头客 |
| 2003 | 关停第三方出货，宁牺牲收入保证服务质量 |
| 2003 | 迁总部至拉斯维加斯，文化成为第一要务 |
| 2003 | 获得富国银行$600万信贷额度，绝处逢生 |
| 2008 | 透明裁员8%，公开邮件发给全员+博客 |
| 2009 | 亚马逊$12亿收购，保持独立运营和文化 |

### 3. 核心教训

- **不要外包核心部门**：从电子物流的惨痛教训学到
- **文化认同要覆盖所有利益相关者**：包括投资者和董事会
- **体验比物质重要**：把钱花在体验上而非物质上
- **建立友谊而非人脉**：纯粹的友谊两三年后自然带来商业机会
- **每天进步1%**：年底改进37倍（复利效应）

### 4. Zappos增长轨迹

| 年份 | 销售额 |
|------|--------|
| 1999 | ~$0 |
| 2000 | $160万 |
| 2001 | $860万 |
