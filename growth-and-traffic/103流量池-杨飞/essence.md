# 流量池 — 杨飞

> 核心命题：流量思维是"获取流量→变现"，流量池思维是"获取流量→存储运营→发掘更多流量"。以存量找增量，以高频带高频。

---

## 一、流量之困

- **流量少**：移动互联网红利消失，头部App（微信、今日头条等）垄断用户时间，微信公众号打开率跌至2%。
- **流量贵**：获客成本从几角飙升至几百甚至上千元；百度广告主平均消费三年涨近两倍。
- **流量陷阱**：流量欺诈成全球性问题，百度推广每天过滤千万级无效点击（5%人工作弊，49%~65%机器作弊）。
- **突围方向**：线上成本高企→"流量下乡"——线下门店（小米之家、京东之家）、分众电梯、地推重新成为流量洼地。

---

## 二、品牌是最稳定的流量池

### 品牌即流量
- 品牌通过**关注**（注意力经济）和**粉丝**（口碑自传播）源源不断地带来免费流量。
- 品牌成本边际递减，成熟品牌即使减少投放也能稳步增长。

### 三种犀利定位法

| 方法 | 核心逻辑 | 典型案例 |
|------|----------|----------|
| **对立型定位** | 与头部竞品形成鲜明差异，"人无我有/人有我强" | 神州专车"更安全的专车"、瓜子"没有中间商赚差价"、农夫山泉"大自然的搬运工" |
| **USP定位** | 聚焦单一功能利益，场景化口号"…就用…" | 斯达舒"胃痛胃酸胃胀"、OPPO"充电5分钟通话2小时"、红牛"困了累了喝红牛" |
| **升维定位** | 创造新品类/新需求，不做同维度纠缠 | 小米/乐视"互联网电视"、RIO"预调果酒" |

### 品牌符号系统

**视觉符号**：LOGO简化（纯文字LOGO成趋势）、主题色抢占（神州金、luckin蓝、摩拜橙）、特殊产品造型（可口可乐曲线瓶）、辅助图形（椰树牌椰汁）、人物代言（企业家/员工/IP均可成为符号）。

**听觉符号**：口号要说人话（"送礼就送脑白金""饿了别叫妈叫饿了么"）；Jingle音效（英特尔"灯等灯等灯"、滴滴"滴滴一下马上出发"）。

## 三、品牌广告如何做出实效

### 品效合一原则
- 移动端"轻轻一点"即可完成购买，品牌广告必须带转化按钮。
- **不浪费每一次不易获得的流量**——微信长文加购买按钮、户外广告加二维码。

### 场景营销
- 场景是品牌落地的"扳机"。神州专车切入6大场景：接送机、会务用车、夜晚加班（"第三高峰"）、孕妇、带子出行、异地出差。
- 核心策略：**避重就轻，集中优势占领关键场景**，而非全面硬拼。

### 品牌接触点
- 用户第一次接触品牌往往不是广告，而是产品本身——包装、App、服务人员。
- 神州专车：司机SOP话术 + 金领司机着装 + App金色主题色 + "叮"的叫车成功音。

### 传统广告转化技巧
1. **明确投放场景，素材简单直接**——分众电梯广告要"一瞥钟情"，不要绕。
2. **提供互动方式**——"传统广告4件套"：客服电话、二维码、关注微信、搜索关键词。
3. **产品活动带品牌**——品牌作为下方背书，产品/促销信息放主位。
4. **广告公关化**——配合事件营销避免"墙纸效应"（如神州"Beat U"）。

## 四、裂变营销：最低成本的获客之道

### 核心理念
- 社交关系链是移动互联网最强大的护城河。
- 裂变 = **分享** + **后付奖励**（广告成本 = 老用户拉新奖励 + 新用户注册奖励）。
- **AARRR模型**：获客→活跃→留存→变现→自传播。

### 裂变六大技巧

**App裂变**：
1. 拉新奖励（神州"邀请有礼"，新用户占70%）
2. 裂变红包（美团/饿了么分享红包）
3. IP裂变（影视IP+红包，神州《鬼吹灯》单次获40万新用户）
4. 储值裂变（亲情账户，10天获118万新用户）
5. 个体福利裂变（喜马拉雅"分享免费听"）
6. 团购裂变（拼多多"通过分享获得让利"）

**微信裂变**：
1. 分销裂变（二级复利海报，神州U+司机招募一周10万报名）
2. 众筹裂变（砍价/拼团）
3. 微信卡券（朋友共享优惠券）
4. 微信礼品卡（星巴克"用星说"）

**线下裂变**：
1. 包装裂变（味全"拼字瓶"、可口可乐"歌词瓶"）
2. O2O积分/红包（青岛啤酒一瓶一码）
3. 产品社交化设计（奥利奥音乐盒）

### 裂变三要素
- **种子用户**：质量>数量，活跃度高、影响力大。
- **裂变诱饵**：福利补贴为主，创意内容为辅。
- **分享趣味**：提供互动谈资、塑造个人形象。

### 游戏化留存（PBL）
- **Points**（点数）：积分体系，激励收集欲望。
- **Badges**（徽章）：可视化成就，身份标签。
- **Leaderboards**（排行榜）：激活攀比心理（微信运动）。

### 核心公式
> **流量裂变 = 平台 + 创意 + 福利 + 技术**
> 
> **存量找增量，高频带高频**——没有绝对低频的产品，关键是转化福利频次。

## 五、微信社会化营销

### 微信服务号 > 订阅号
- 服务号不被折叠，直达用户首屏，月4次精品推送优于每日平庸内容。
- 定位为**超级App**：承载业务功能 + 客服 + 营销告知。

### 10万+的三要素
- **创意驱动**：有趣、好玩儿→自传播。
- **技术驱动**：H5交互、裂变系统。
- **利益驱动**：福利是分享催化剂。神州专车"花式发券"期期10万+。

### 社会化营销三字诀：轻、快、有网感
- **轻**：内容轻量，争取多次试错机会。
- **快**：速度超越大众预期，天下功夫唯快不破。
- **有网感**：说人话，有同理心，人格化运营（支付宝公众号"神经病"风格也能篇篇10万+）。

### 一图流 > 图文流
- 图片是最快速、最讨好的传播方式。
- 底部带二维码/小程序即可品效合一。

### 微信模板消息
- 用于用户提频：注册提醒、积分变动、卡券到期、消费提醒、配送提醒。
- 终极目标：将交互变成用户习惯。

### 小程序
- 关键词：线下、低频、场景、轻量。
- "无须安装，无处不在，触手可及，用完就走"。

## 六、事件营销："轻快爆"

### 原则
- **轻**：内容轻、媒介轻（线上为主），快速出街。
- **快**：反应速度 > 大众预期，14天从创意到上线。
- **爆**：5个关键爆点——热点、爆点（关键词/符号）、卖点（核心利益）、槽点（引导吐槽）、节点（周二至周四，避开周末）。

### 典型案例
- **闪送"我们是谁"**：24小时内真人版海报+线下广告牌，两天App下载超3万。
- **丧茶快闪店**：低成本线下改造→线上爆发，菜单成为刷屏素材。
- **神州"Beat U"**：争议性造势→日新增用户增长10倍→道歉券5分钟10万+。

### 转化警告
- 事件火了但品牌没人知道=失败（斯巴达勇士、逃离北上广）。
- **火力必须集中在品牌或核心卖点上**。

## 七、数字广告投放

### 防作弊：全程数据监测
- 三端数据必须打通：**媒体端**（展现/点击）+ **网站端**（PV/UV/跳出率）+ **销售端**（留资/订单/成单）。
- KPI应多定在潜客量、订单量、成单量，而非展现量/点击量。

### "神庙系统"（神州DMP实践）
- 打通三端数据→排除40%无效广告费→成单成本连续5月下降。
- SEO+SEM互补：品牌词用SEO承接老用户，通用词/竞品词用SEM拉新。

### 投放形式评估

| 形式 | 特点 | 靠谱度 |
|------|------|--------|
| **SEM** | 搜索竞价，精准但越来越贵（年涨15%） | ★★★★ |
| **信息流/原生广告** | "信息找人"，原生感强，点击率高 | ★★★★ |
| **微信朋友圈广告** | 最贵信息流（100-150元/CPM），但精准优质 | ★★★★ |
| **DSP** | 碎片化标签投放，黑洞多，垃圾流量严重 | ★★ |
| **企业DMP** | 第一方数据管理，用户画像+精准定向+效果优化 | ★★★★★（长期） |

### 信息流广告投放要点
1. 核心卖点明确
2. 标题极强吸引力（欢迎"标题党"）
3. 配图精美+生活化（不像广告的广告）

## 八、搜索入口的大流量获取

### SEO（搜索引擎优化）
- 免费点击，一次优化半年有效。
- 四大优化：结构优化（TDK/301/404/目录层级）、内容优化（高质量伪原创）、内链优化、外链优化。

### ASO（应用商店优化）
- 65%的iOS下载通过搜索实现。
- 关键词拦截：高热度行为词、品牌词、竞品词。

### 电商搜索优化
- 标题关键词优化、数据螺旋上升、无线端权重提升。
- 付费推广：直通车（CPC）、钻石展位（CPM）、淘宝客（CPS）。

## 九、落地页：第一生产力

### 核心原则
- 落地页不是创意稿，是转化终端。
- "尽可能收集有效用户信息"为第一要义。

### 六大要素
1. 核心卖点和品牌/活动信息
2. 品牌整体印象与产品口碑（从众心理："已有×万人注册"）
3. 消费者益处（不是规格型号，是"能给我什么"）
4. 权威认证/社会证言
5. 用户留资（电话/微信）
6. 索取最少有效信息

### 外部素材原则
- **图片**：场景式（原生感）> 福利式（促销刺激）> 告知式（硬广）。
- **文案**：用户视角 > 客户视角；一条文案一个卖点；简单易懂；使用地域/年龄/日期等关联信息；激发主观动机而非情感绑架。

## 十、直播营销

### 电商直播四大优势
1. 全面直观传递商品信息
2. 一对多降低售前成本
3. 聚集人气营造团购氛围
4. 直播中直接跳转购买页

### IMBT方法论
- **I**dea & IP（创意和IP）
- **M**edia（媒介）：电商直播为主+综合直播为辅
- **B**enefit（福利）：红包雨、秒杀、买送
- **T**echnology（技术）：拍摄/导播/互动/特效

### 案例：神州买买车王祖蓝直播
- 90分钟卖出2717辆科鲁兹，订单金额2.28亿元。
- 关键：综艺节目规格脚本 + 双平台（天猫+一直播）+ 密集福利刺激。

## 十一、跨界营销（BD）

### BD合作四阶段
1. **联合创意**（品牌海报/活动）
2. **内容/平台合作**（丧茶快闪店）
3. **产品/技术合作**（API接入/预装App）
4. **订单交易合作**（联名卡/预装机）

### BD经理四字诀
- **真诚**：先满足对方KPI。
- **务实**：不忽悠，可实现。
- **高调**：让合作被更多人看到。
- **捞过界**：懂产品、技术、法务。

### 两类关键BD资源
- **娱乐资源**：减少传播成本（IP授权、明星资源）。
- **金融资源**：增加销售收益（联名卡、银行采购收入）。

## 全书核心公式总结

```
流量池思维 = 获取流量 + 存储运营 + 发掘更多流量

品牌是最稳定的流量池（认知→认同→认购）

裂变 = 平台 + 创意 + 福利 + 技术

品效合一 = 品牌声量 + 效果销量（移动互联网的每一次传播都应导向购买）

"急功" = 快速建立品牌，切入市场
"近利" = 流量快速转化为销量
```
