# SKILL.md — 华与华超级符号方法实操指南

> 来源：《超级符号就是超级创意》（华杉、华楠）
> 适用场景：品牌创建、品牌升级、产品开发、营销传播、企业战略制定

---

## 方法总纲：华与华方法 = 用创意降低营销传播成本

核心公式：
```
超级符号 + 超级话语 + 超级词语 → 低成本传播 → 品牌资产积累
```

一切工作的出发点：**成本与投资**——降低企业营销成本、消费者选择成本、社会监督成本。

## 场景路由：按任务快速定位

| 场景 | 优先参考技能 | 关键检验 |
|------|------------|---------|
| **品牌命名/更名** | 技能三（超级词语）→ 技能一（超级符号） | 电话测试：5秒能说清？有召唤性？ |
| **视觉识别/LOGO设计** | 技能四（标志设计）→ 技能一（超级符号） | 需要写释义吗？一句话能描述？ |
| **包装设计** | 技能五（包装设计）→ 技能一（超级符号） | 不看大图能看清？撕碎一片能识别？ |
| **广告创意/传播** | 技能六（15秒广告）→ 技能二（超级话语） | 华四条全过？品牌名第一个词出现？ |

## 技能一：打造品牌超级符号

### 步骤
1. **扫描五感通道**：视觉、听觉、嗅觉、味觉、触觉，找到产品最突出的感觉路径
2. **寻找文化原型**：在人类文化中寻找人们本来就记得、熟悉、喜欢的符号
   - 公共符号：红绿灯、交通标志、男女厕所标志
   - 文化符号：金元宝、餐桌绿格子、十二生肖、熊猫、《我爱北京天安门》
3. **嫁接到品牌**：将超级符号嫁接到品牌名称、产品包装、广告形象上
4. **检验标准**：
   - 视觉符号 → "过目不忘"
   - 听觉符号 → "耳熟能详"
   - 综合 → "惊鸿一瞥"也能被认出

### 关键原则
- 做编辑不做原创——传承千锤百炼的老符号（"述而不作"）
- 具象 > 抽象——能描述的符号传播成本低
- 传播的关键在于"传"——设计可口耳相传的视觉（如"绿格子包装那个"）
- 不要盲目追求"高级感"——脱离常识去运用知识会出问题

### 检测方法
> "你做了个什么设计？你说！"——如果描述能让对方脑中浮现准确画面，就是好设计。

### ⚠️ 失败模式
- **自嗨式原创**：设计师追求独特性，造出无人认识的新符号，传播成本无限高
- **文化错配**：嫁接的符号在目标人群文化中无根基（如给下沉市场用抽象艺术符号）
- **五感分散**：视觉一个符号、听觉一个符号、包装又一个符号，没有合力
- **频繁更换**：符号还没积累出资产就换掉，等于每年重新交学费

### 🚫 反模式
| 做法 | 为什么错 | 应该怎么做 |
|------|---------|-----------|
| 请艺术家原创一个"高级"符号 | 没有文化母体，传播成本高 | 从公共符号/文化原型中选 |
| 追求"国际范"抽象图形 | 消费者看不懂、记不住、传不了 | 用具象图形，一句话能描述 |
| 品牌符号和产品无关 | 符号不能强化品类认知 | 符号要一目了然见行业 |
| 把符号注册了就觉得成功 | 商标注册≠消费者记住 | 在消费者心中注册才算数 |

### ✅ 检查点
- [ ] 能用一句话描述这个符号吗？
- [ ] "惊鸿一瞥"（0.5秒扫过）能认出吗？
- [ ] 消费者能口耳相传地描述它吗？（"就是那个绿格子的"）
- [ ] 这个符号有文化根基吗？目标人群本来就熟悉吗？

## 技能二：创作品牌超级话语

### 步骤
1. **明确目的**：不是"说清"自己，不是"说服"消费者，而是"说动"消费者行动
2. **嫁接文化**：找到人类文化中的预制件（歌曲、俗语、成语、经典台词）
3. **用口语写出来**：必须是日常说话的方式，不是书面文案
4. **套话化**：做成顺口溜、韵律句，让人脱口而出
5. **检验三个"动"**：
   - 消费者听到后冲动、行动？
   - 这句话本身能被消费者主动想起并传播？
   - 我们自己的员工会随时挂在嘴边？

### 十二字方针
**一目了然，一见如故，不胫而走**

### 语法规则
- 只用**陈述句**（陈述事实）和**行动句**（要求行动）
- 不要空洞口号，要有具体内容
- 广告语里尽量包含品牌名
- "现量"境界：直截了当，脱口而出，不思而得

### 经典范式
```
"晒足180天，厨邦酱油美味鲜" —— 陈述句，陈述事实
"送长辈，黄金酒" —— 行动句，要求行动
"一个北京城，四个孔雀城" —— 陈述句，品牌战略
"小葵花妈妈课堂开课啦" —— 陈述句，建立品牌资产
```

### ⚠️ 失败模式
- **书面腔**：写出来像作文不像说话，消费者听不懂、记不住
- **只说清没说动**：信息传达了但没有行动指令，消费者知道了但不买
- **不含品牌名**：广告语火了但消费者不知道是哪个品牌的
- **文艺病**：追求"意境""格调"，牺牲了传达效率

### 🚫 反模式
| 做法 | 为什么错 | 应该怎么做 |
|------|---------|-----------|
| "品味非凡，尊享人生" | 空洞口号，任何品牌都能用 | 写具体事实或行动指令 |
| "让生活更美好" | 没有品牌名，没有具体内容 | 包含品牌名+具体购买理由 |
| 用比喻/隐喻/双关 | 增加理解成本，传播变慢 | 直说，陈述句/行动句 |
| 请文案大师写"金句" | 消费者不按文学标准买东西 | 用消费者原话/口语写 |

### ✅ 检查点
- [ ] 是陈述句或行动句吗？（排除感叹句、疑问句、比喻句）
- [ ] 包含品牌名吗？
- [ ] 员工会不会自然地挂在嘴边？
- [ ] 电话里念给朋友听，朋友能听懂并记住吗？
- [ ] 这句话是在"说动"而非仅仅"说清""说服"？

## 技能三：运用超级词语

### 步骤
1. **低成本命名**：用人们本来就熟悉的文化词语（苹果 > 戴尔，娃哈哈 > 乐百氏）
2. **听觉词语优先**：用电话测试法——电话里说名字要花多少时间？
3. **召唤性命名**：名字要有指向性，能召唤价值、召唤顾客
4. **先词语后产品**：先提出一个词语/概念，再用产品实现它
5. **编写《企业词典》**：先定义，后定位——假定编入词典怎么写？

### 命名三原则
| 原则 | 说明 | 反面案例 |
|------|------|----------|
| 成本 | 记忆成本、传达成本、理解成本低 | "兰乔圣菲"（需120字解释） |
| 召唤 | 能召唤品牌价值和顾客 | "兰德酷路泽"（无召唤性） |
| 投资 | 长期积累品牌资产 | 频繁改名浪费投资 |

### 关键规则
- 不要死守注册商标思维——在消费者心中注册比在商标局注册重要
- 有价值的名字再长也比没价值的名字好
- 能把价值做进名字里，就不要放在口号上

### ⚠️ 失败模式
- **注册优先症**：为了能注册商标，选了一个消费者记不住的名字
- **洋名崇拜**：用音译洋名装高端，消费者读不出、记不住、传不了
- **频繁改名**：每换一个领导改一次名，品牌资产清零重来
- **名字无价值**：名字只是代号，不能召唤任何利益或顾客

### 🚫 反模式
| 做法 | 为什么错 | 应该怎么做 |
|------|---------|-----------|
| "Lancôme"式音译名 | 消费者要花成本学习发音 | 用消费者本来就会说的词 |
| 生造词（如"Xperia"） | 没有文化根基，记忆成本高 | 借用已有文化词语 |
| 名字需要30秒解释 | 传播成本太高 | 电话测试：5秒内说清 |
| 先做产品再想名字 | 名字沦为标签，无法承载战略 | 先词语后产品 |

### ✅ 检查点
- [ ] 电话测试：说名字需要几秒？需要解释吗？
- [ ] 名字能召唤品牌价值或目标顾客吗？
- [ ] 名字用的是消费者本来就熟悉的词吗？
- [ ] 能把价值做进名字里（而不只是口号上）吗？

## 技能四：品牌标志设计

### 做标字，不做标志
1. **一目了然见名字**：以品牌名字体为中心设计
2. **一目了然见行业**：食品像食品，药品像药品，银行像银行
3. **使用具象图形**：不要抽象化、艺术化

### 不要做的事
- ❌ 不写标志释义——需要解释的标志就该扔掉
- ❌ 不做"辅助图形"——华与华方法里没有这个词
- ❌ 不轻易改标志——标志承载几十年的投资
- ❌ 不学大品牌的后期做法——要学它们创业期的成功之道

### ⚠️ 失败模式
- **释义依赖**：标志旁边配一段"设计理念"才能看懂，说明设计失败
- **抽象艺术病**：设计师追求几何美学，消费者看不懂是什么
- **辅助图形泛滥**：搞出一堆"延展图形"，增加混乱不增加识别
- **改标上瘾**：每3年换一次标志，品牌资产持续归零

### 🚫 反模式
| 做法 | 为什么错 | 应该怎么做 |
|------|---------|-----------|
| 把品牌名抽象成几何图形 | 消费者看不出是字 | 标志核心就是品牌名字体 |
| 加一堆"品牌元素"辅助图形 | 增加识别负担，分散注意力 | 一个符号打天下 |
| 标志设计完写500字释义 | 需要解释=设计失败 | 不写释义，消费者秒懂 |
| 学苹果/耐克的极简标志 | 那是百亿投资后的结果，不是起点 | 学它们创业期的标志 |

### ✅ 检查点
- [ ] 第一眼能看到品牌名吗？
- [ ] 第一眼能看出是什么行业吗？
- [ ] 需要写释义才能理解吗？（需要就扔掉重做）
- [ ] 有"辅助图形"吗？（有就删掉）

## 技能五：包装设计

### 核心理念
包装 = 信息炸药包，不是商品包

### 步骤
1. **去卖场/货架调研**：先看100%的竞品，再设计自己的1%
2. **建立陈列优势**：包装设计首先是为了在货架上脱颖而出
3. **符号刺激本能反射**：用色彩和符号第一时间抓住注意力
4. **包装文案做导购指南**：严肃、快速、完整地传达购买理由
5. **包装即广告位**：把每个产品都当成媒体来设计

### 检测标准
- "不用点击大图就能看清"——电商时代的货架思维
- 撕成碎片任意拣起一片，也能识别品牌
- 没有任何广告，只要摆上货架就能自己卖出去

### ⚠️ 失败模式
- **设计师桌面美学**：电脑上看很美，货架上看不清品牌名
- **信息不足**：包装上没有购买理由，消费者拿起又放下
- **货架隐身**：设计时只看单品效果，不看放在竞品旁边的效果
- **包装与广告脱节**：包装一套视觉、广告一套视觉，符号不统一

### 🚫 反模式
| 做法 | 为什么错 | 应该怎么做 |
|------|---------|-----------|
| 用电脑屏幕看包装效果 | 电商缩略图/货架距离完全不同 | 用手机缩小图测试 |
| 追求"简约高级"留白 | 货架上被竞品淹没 | 先建陈列优势，再谈美感 |
| 包装上品牌名很小 | 消费者不知道买的什么牌子 | 品牌名要大、要醒目 |
| 包装不含购买理由 | 消费者没有购买动力 | 包装文案=导购指南 |

### ✅ 检查点
- [ ] 把包装缩到手机屏幕大小，还能看清品牌名和品类吗？
- [ ] 撕碎后任意一片能识别品牌吗？
- [ ] 包装上有完整的购买理由吗？
- [ ] 放在竞品旁边，能第一时间跳出来吗？

## 技能六：15秒电视广告创作

### "华四条"标准
1. ✅ 让人记住品牌叫什么名字
2. ✅ 让人记住商品长什么样子
3. ✅ 给人购买理由和冲动
4. ✅ 建立品牌符号和企业战略优势

### 关键规则
- 品牌名出现得越早越好，最好是第一个词
- 产品是一号主角，创意不能抢产品的戏
- 假设观众是第一次听说这个产品
- 不要"故事板"思维，要"耍把戏"/"产品演出"思维
- 停止使用"故事板"这个词

### ⚠️ 失败模式
- **创意抢戏**：广告拍成了微电影，观众记住了故事记不住品牌
- **品牌名晚出**：前10秒铺垫，最后5秒才出现品牌名，观众已经走神
- **华四条没过完**：只满足1-2条，另外2-3条缺失
- **自嗨创意**：团队觉得"有创意"，消费者看不懂在卖什么

### 🚫 反模式
| 做法 | 为什么错 | 应该怎么做 |
|------|---------|-----------|
| 拍感人故事微电影 | 观众记住故事，忘掉品牌 | 产品是主角，创意是配角 |
| 品牌名在最后5秒出现 | 前10秒观众已流失 | 第一个词就是品牌名 |
| 追求"戛纳广告节"水平 | 评委标准≠消费者标准 | 华四条全过就是好广告 |
| 用"故事板"规划广告 | 故事思维让创意偏离产品 | 用"产品演出"思维 |

### ✅ 检查点
- [ ] 品牌名是第一个出现的词吗？
- [ ] 产品是广告的绝对主角吗？
- [ ] 华四条全部满足了吗？逐一核对
- [ ] 假设观众第一次看，能记住品牌名+产品样子+购买理由吗？

## 技能七：产品开发

### 核心方法
```
先有购买理由 → 再有命名 → 再有广告创意 → 再有包装设计 → 最后产品开发实现
```

### 步骤
1. 研究消费者使用体验，发现痛点
2. 提出购买理由（一句话/一个词）
3. 命名产品——把价值做进名字里
4. 创作广告语和包装方案
5. 写成《新产品开发任务书》交给技术部门实现

### 产品四重属性
```
工具（基础功能）→ 器具（技术含量）→ 道具（身份角色）→ 玩具（乐趣享受）
```
任何产品都可以通过超级符号实现超越工具层面的价值。

### ⚠️ 失败模式
- **技术先行**：先研发产品，再想卖给谁、为什么买——本末倒置
- **购买理由模糊**：说不清"消费者为什么要买这个"就开始开发
- **名字后补**：产品做完了才想名字，名字沦为代号无价值
- **任务书缺失**：口头传达需求，技术部门理解偏差，出来的产品不对

### 🚫 反模式
| 做法 | 为什么错 | 应该怎么做 |
|------|---------|-----------|
| 技术部门主导产品开发 | 做出来的不一定是消费者要的 | 营销部门写任务书，技术实现 |
| 先做产品再想购买理由 | 理由是硬凑的，不成立 | 先有购买理由，后有产品 |
| 产品名就是品类名 | 没有差异化，没有品牌资产 | 把价值做进名字里 |
| 口头沟通产品需求 | 信息衰减，出来的产品走样 | 写成正式的《任务书》 |

### ✅ 检查点
- [ ] 购买理由是一个词或一句话吗？
- [ ] 命名、广告语、包装方案在产品开发之前就定好了吗？
- [ ] 有正式的《新产品开发任务书》吗？
- [ ] 产品至少超越了"工具"层面（有道具/玩具属性）吗？

## 技能八：企业战略制定

### 核心等式
```
企业社会责任 = 经营使命 = 企业战略
```

### 步骤
1. **发现社会问题**：你的行业存在什么社会问题？
2. **锁定使命**：你为社会解决什么问题？（永远完不成的才是使命）
3. **设计业务组合**：你的产品结构就是社会问题的解决方案
4. **规划路线图**：金角 → 银边 → 草肚皮
5. **构建话语权**：成为该领域的权威专家和首席知识官

### 三个层次
- 拳头产品 → 代表先进生产力
- 权威专家 → 代表先进文化（首席知识官）
- 梦想化身 → 代表人类梦想

### 竞争原则
- 盯住顾客，不盯住对手
- "泡妞的关键在于妞"
- 竞争是一种幻觉，同行是一种假设
- 行有不得，反求诸己

### ⚠️ 失败模式
- **使命太高大上**：使命变成了口号，和实际业务脱节
- **盯住对手**：把精力花在研究竞品上，忽略消费者真正需求
- **战略悬空**：战略只存在于PPT里，没有植入产品、包装、广告
- **"大方向定了"综合症**：战略方向定了就觉得完了，执行层没有具体指引

### 🚫 反模式
| 做法 | 为什么错 | 应该怎么做 |
|------|---------|-----------|
| 使命是"成为行业第一" | 这是目标不是使命，和消费者无关 | 使命=解决什么社会问题 |
| 花大量预算做竞品分析 | 竞争是幻觉，关键是消费者 | 盯住顾客，不盯住对手 |
| 战略和创意分家 | 没有创意，战略等于零 | 所有事是一件事 |
| 每年换一次战略方向 | 品牌投资需要50年复利 | 选对方向，持续投资 |

### ✅ 检查点
- [ ] 使命和要解决的社会问题对应吗？
- [ ] 使命是永远完不成的吗？（完成了说明使命太小）
- [ ] 战略能植入到产品、包装、广告的每一个细节吗？
- [ ] 100年后这件事还需要做吗？

## 技能九：消费者调研

### 核心理念
- 调研是找参考、找启发，不是找依据
- 一切智慧都在历史，一切秘密都在现场
- 调研即策划，不是独立工作

### 现场调研法
1. **行业竞争史调研**：研究过去10年各品牌的举措和市场消长
2. **售点观察**：在一个店待2小时，观察消费者完整购买过程
3. **店员访谈**：问家常问题，不问科学问题（今天来了几个？卖了多少？）
4. **消费者深访**：只要求回忆场景，不要求总结原因

### 创意测试四问题
1. **是谁？**（品牌识别）
2. **他要你做什么？**（信息传达）
3. **你做不做？**（行动驱动）
4. **为什么？**（理由判断）

### 四大陷阱
1. 提问形式本身就是问题——"为什么"得不到真实答案
2. 定量调查掩盖了观念形成的过程
3. 普通人对问卷课题的理解与设计者不同
4. 颠倒了企业与消费者的责任——创意好不好是企业的事

### ⚠️ 失败模式
- **问卷依赖**：用标准化问卷替代现场观察，得到的是假数据
- **问"为什么"**：消费者事后合理化，答案不是真实动机
- **调研替代判断**：把决策权交给消费者调研，推卸企业责任
- **调研与策划脱节**：调研团队做完报告交给策划团队，信息断裂

### 🚫 反模式
| 做法 | 为什么错 | 应该怎么做 |
|------|---------|-----------|
| 大规模定量问卷调查 | 掩盖观念形成过程，数据是假的 | 深度观察+场景回忆 |
| 问消费者"你为什么买" | 事后合理化，不是真实原因 | 问"当时发生了什么" |
| 调研完出200页报告 | 报告没人看，信息不转化为行动 | 调研即策划，边调研边出方案 |
| 用调研数据决定创意好坏 | 创意好不好是企业的事 | 调研找启发，企业做判断 |

### ✅ 检查点
- [ ] 有没有亲自去售点观察2小时以上？
- [ ] 有没有做行业竞争史调研（过去10年）？
- [ ] 问卷里有没有"为什么"类问题？（有就删掉）
- [ ] 创意测试四问题全部过了吗？

## 技能十：品牌顶层设计

### 核心法则
**所有的事都是一件事，所有人一起做所有事。**

### 一次成型清单
- [ ] 企业战略（社会问题 + 使命 + 业务组合）
- [ ] 产品战略（拳头产品 + 产品结构 + 推出次序）
- [ ] 品牌战略（品牌名 + 超级符号 + 超级话语）
- [ ] 品牌设计（标志 + 包装 + 视觉系统）
- [ ] 广告创意（15秒电视广告 + 平面 + 传播方案）
- [ ] 产品开发（购买理由 + 命名 + 技术实现）

### 原则
- 后工序决定前工序——传播是后工序，以终为始
- 战略不是高高在上，而是植入每一个行动
- 没有创意，战略等于零
- "大方向定了"其实什么都不是

### ⚠️ 失败模式
- **分阶段推进**：先做战略，再做品牌，再做产品——每个阶段信息断裂
- **部门墙**：战略部、品牌部、产品部各做各的，最终不统一
- **"大方向"满足症**：定了方向就觉得完了，没有落到具体设计
- **抄袭后期做法**：学大品牌现在的做法，而不是它们创业期的做法

### 🚫 反模式
| 做法 | 为什么错 | 应该怎么做 |
|------|---------|-----------|
| 战略做完再找设计公司 | 信息断裂，设计不理解战略 | 所有团队从第一天就一起干 |
| 分阶段招标不同供应商 | 每个供应商做自己的理解 | 一个团队一次性做完 |
| 学大品牌当前做法 | 那是百亿投资后的结果 | 学它们创业期的成功之道 |
| 定了战略方向就觉得完成 | 方向≠执行，差十万八千里 | 落到每一个产品/包装/广告细节 |

### ✅ 检查点
- [ ] 战略、品牌、产品、设计、广告是否一次性同步完成？
- [ ] 后工序（传播）是否决定了前工序（战略）的表达？
- [ ] "大方向"是否已经落到每个可执行的细节？
- [ ] 是否在学大品牌的创业期而非成熟期？

## 综合检验清单

| 检验项 | 标准 | 快速判断 |
|--------|------|---------|
| 品牌符号 | 一句话能描述？惊鸿一瞥能认出？ | 说给朋友听，脑中有画面吗？ |
| 广告语 | 员工会挂在嘴边？消费者愿传播？ | 是陈述句/行动句？含品牌名？ |
| 命名 | 电话测试：几秒能说清？ | 5秒内？不用解释？ |
| 标志 | 需要写释义吗？需要就扔掉 | 一眼见名字？一眼见行业？ |
| 包装 | 不看大图能看清？撕碎一片能识别？ | 缩到手机大小还能看清？ |
| 产品 | 购买理由是一个词还是一句话？ | 先有理由后有产品？ |
| 战略 | 100年后这事还需要做吗？ | 使命完不成才对？ |
| 调研 | 创意测试四问题全过了吗？ | 有没有问"为什么"的陷阱？ |

## 推荐实践路径

```
1. 发现社会问题 → 2. 锁定经营使命 → 3. 寻找文化原型/超级符号
→ 4. 创作超级话语/超级词语 → 5. 设计品牌符号系统和包装
→ 6. 开发产品（先有购买理由后有产品） → 7. 创作15秒广告
→ 8. 持续重复投资 → 9. 50年后回收第一年投资的利息
```

### 全流程防错速查

| 阶段 | 最常犯的错 | 一句话提醒 |
|------|-----------|-----------|
| 命名 | 注册优先、洋名崇拜 | 消费者心中注册 > 商标局注册 |
| 符号 | 自嗨原创、抽象艺术 | 做编辑不做原创，具象>抽象 |
| 话语 | 书面腔、文艺病 | 口语写，陈述句/行动句 |
| 标志 | 需要释义、辅助图形 | 不写释义，删掉辅助图形 |
| 包装 | 设计师桌面美学 | 用手机缩小图测试 |
| 广告 | 故事抢戏、品牌名晚出 | 产品是主角，品牌名第一个词 |
| 产品 | 技术先行、购买理由模糊 | 先有理由后有产品 |
| 战略 | 盯对手、使命空洞 | 盯顾客，使命=社会问题 |
| 调研 | 问卷依赖、问"为什么" | 去现场，问场景 |
| 顶层设计 | 分阶段、部门墙 | 所有事是一件事 |
