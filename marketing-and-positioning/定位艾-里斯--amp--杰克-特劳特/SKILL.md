## 一、定位思维核心模型

### 1.1 定位五步法（通用框架）

```
输入：一个产品/品牌/服务/个人/组织
输出：一个在目标受众头脑中的独特位置

步骤：
1. 扫描心智地图：目标受众头脑中已有哪些品牌/概念占据位置？
2. 发现可用空位：哪些有价值的位置尚未被占领？
3. 选择定位策略：争当第一 / 对比定位 / 给对手重新定位 / 找空子
4. 构建极简信息：用一句话概括定位，能被目标受众秒懂
5. 持续固化：年复一年钉在那儿，只变战术不变战略
```

### 1.2 产品阶梯心智模型

```
人脑中每类产品有一个梯子，最多7层：
┌─────────────────────┐
│ 第1层：品类领导者      │  ← 份额最大、利润最高
│ 第2层：第二品牌        │  ← 份额约第一的1/2
│ 第3层：第三品牌        │  ← 份额约第一的1/3
│ ...                   │
│ 第7层：最后一个被记住的  │
└─────────────────────┘

行动指南：
- 你是第一 → 强化"真东西"地位，扩大品类
- 你是第二 → 与第一挂钩（对比定位）
- 你什么都不是 → 找空子 / 给对手重新定位
```

## 二、定位策略工具箱

### 2.1 策略一：争当第一

**适用场景**：品类处于早期、头脑中尚无强品牌；你想成为品类领导者

**操作要点**：
1. 识别一个尚未被占据的品类或细分品类
2. 用品牌名抢占该品类的"通用名称"地位
3. 第一时间大规模传播，建立"印刻效应"

**案例**：柯达（胶卷）、IBM（计算机）、施乐（复印机）

**检验问题**：
- "第一个做X的是谁？" 如果答案是你，策略正确
- 如果不是，考虑换一个更窄的品类去争第一

**品类领导者的护城河**：
- 品类 = 品牌（搜索=谷歌，社交=微信）
- 定义品类规则，让后来者只能跟从
- 不断扩大品类边界，而非品牌延伸

---

### 2.2 策略二：对比/关联定位

**适用场景**：品类已有强大领导者，你要做第二；你的品牌和竞品说不清区别

**操作要点**：
1. 明确承认对手的地位（不是假装它不存在）
2. 把自己与领导者挂钩，在对比中显示差异化价值
3. 利用人们对弱者的天然同情心

**经典公式**：
> "X是第一，但我们更Y"（Y = 努力/专注/实惠/...）

**案例**：
- 艾维斯：在租车业屈居第二 → "我们更努力"
- 七喜：非可乐 → 挂在可口可乐/百事可乐梯子旁边

**反面教训**：
- 艾维斯后来想"当第一" → 失败（FWMTS陷阱）
- 对比广告说"我们比对手强" ≠ 重新定位

---

### 2.3 策略三：给竞争对手重新定位

**适用场景**：头脑中已无空子可钻；行业老大太强找不到突破口

**操作要点**：
1. 找出领导者在消费者心智中的**固有弱点**（不是产品质量问题）
2. 用简明信息指出该弱点，动摇消费者对领导者的信心
3. 消费者会自动寻找替代品——你的品牌就是那个替代品

**案例**：
- 泰诺 vs 阿司匹林："如果你有胃溃疡/哮喘/过敏……阿司匹林会刺激胃黏膜"
- 皇家道尔顿 vs 雷诺克斯："英国瓷器 vs 美国瓷器"
- 智慧牌薯片 vs 品客："土豆/油/盐 vs 脱水土豆/甘油二酸酯"

**关键区别**：
| 对比广告 | 重新定位 |
|---------|---------|
| "我们更强" | "对手有这个问题" |
| 消费者半信半疑 | 消费者开始怀疑对手 |
| 弱效果 | 强效果 |

---

### 2.4 策略四：找空子

**适用场景**：品类成熟，但仍有机会

**空子清单**：

| 维度 | 空子类型 | 经典案例 |
|------|---------|---------|
| 价格 | 高价 | 米什劳啤酒、芝华士、哈根达斯 |
| 价格 | 低价 | 奎普传真机、西南航空 |
| 尺寸 | 小型 | 大众甲壳虫、索尼微型电视 |
| 尺寸 | 大型 | Advent投影电视 |
| 性别 | 男性化 | 万宝路 |
| 性别 | 女性化 | 弗吉尼亚细枝 |
| 年龄 | 儿童 | 目标牌牙膏 |
| 年龄 | 老年人 | 寿星营养液 |
| 时段 | 夜间 | Nyquil感冒药 |
| 渠道 | 超市 | L'eggs连裤袜 |
| 用量 | 大消耗 | "沙弗——喝了还想喝的啤酒" |

**关键原则**：
- 填补的是**头脑中的空当**，不是工厂里的空当
- 要有**逆向思维**：如果人人都向东，你向西找机会
- 空子必须足够大，能支撑一个业务

## 三、命名技能

### 3.1 好名字的特征

```
好名字 = 定位启动器

检验标准：
□ 是否暗示了产品的核心利益/定位？
□ 是否容易发音、记忆？
□ 是否在品类中独一无二？
□ 是否不会过时？
□ 是否听起来比看起来更重要？（耳朵优先于眼睛）
```

**好名字示例**：
- 海飞丝（Head & Shoulders）→ 暗示去屑
- 呵护（Intensive Care）→ 暗示滋润
- 永久（DieHard）→ 暗示电池持久
- 亲近（Close-Up）→ 暗示社交距离

### 3.2 坏名字的类型

| 类型 | 问题 | 案例 |
|------|------|------|
| 区域性 | 限制认知范围 | 东方航空 |
| 含混/多义 | 无法建立清晰联想 | 《时代》 |
| 过于通用 | 失去商标保护 | 米勒"莱特" |
| 无意义缩写 | 无法激发联想 | MBPXL、TRW |
| 产品延伸 | 跷跷板效应 | 低糖百事 |
| 过时 | 与时代脱节 | 《绅士》 |
| 难发音 | 口碑传播受阻 | 含X/Z/Q的名字 |

### 3.3 改名决策树

```
你的名字是否阻碍定位？
├── 是 → 改名（埃克森、尤尼罗亚尔都改过且成功）
│   ├── 起新名还是用缩写？
│   │   ├── 未出名 → 用全名/描述性名字
│   │   └── 已出名 → 可用缩写（IBM、GE）
│   └── 改名时机：越早越好，负面权益只会累积
└── 否 → 保持，但确保名字与定位一致
```

## 四、产品延伸警报系统

### 4.1 产品延伸决策矩阵

```
是否应该延伸品牌名到新产品？

✅ 可以延伸的条件（全部满足）：
□ 竞争对手很弱/没有
□ 你的销量很小
□ 你不想建立定位
□ 你不做广告
□ 没有知名品牌可以切入

❌ 不应延伸的条件（任一满足）：
□ 品类中已有强大品牌
□ 你计划大量广告投入
□ 你想建立品类领导地位
□ 消费者会把品牌名与特定品类强关联
```

### 4.2 购物单检验法

**操作方法**：写下品牌名，让家人去购买

**通过**：买回来的全是原始产品（舒洁=卫生纸，拜耳=阿司匹林）
**失败**：买回来的可能是多种不同产品中的任何一个（斯各特=？卡夫=？）

### 4.3 酒保检验法

**操作方法**：只说品牌名，看酒保给你什么

**通过**："芝华士" → 明确的威士忌
**失败**："卡提" → 是卡提·沙克还是卡提12？

### 4.4 替代方案

当延伸诱惑出现时：
1. **新品牌**：宝洁模式——汰渍、佳洁士、帮宝适各用独立品牌
2. **拓宽用途**：强生模式——婴儿香波卖给成人（同产品、同包装、不同用途）
3. **新概念新品牌**：当新技术出现时，不要用旧品牌（超霸 vs 埃弗雷迪碱性电池）

## 五、竞争对手分析框架

### 5.1 心智地图绘制

```
步骤：
1. 列出品类中的所有主要品牌
2. 用语义分化法调查消费者对每个品牌的认知
   - 属性维度（如：价格高/低、质量好/差、创新/传统...）
   - 评分1-10
3. 在二维/多维地图上绘制每个品牌的位置
4. 识别空白区域（有需求但无品牌占据的区域）
```

### 5.2 强弱分析

```
对每个竞争对手问：
- 它在消费者头脑中代表什么？（一个词/一个概念）
- 这个位置有多强？（稳固/动摇/空虚）
- 它有什么固有弱点？（不是产品质量弱点，是心智弱点）
- 它如果延伸品牌，会在哪里露出破绽？
```

## 六、个人定位技能

### 6.1 个人定位四步

1. **定义自己**：选定一个具体概念与自己联系（"达拉斯最好的税务律师"，不是"达拉斯最好的律师"）
2. **找匹马骑**：公司（朝阳行业）、上司（最有前途的）、朋友（商界人脉）、好点子、信心
3. **起个好名字**：避免缩写、避免与人混淆
4. **坚持你的定位**：不要试图成为"样样都会的通家"

### 6.2 六匹马

| 马 | 策略 |
|----|------|
| 公司 | 加入朝阳行业的好公司 |
| 上司 | 在最精明、最有能耐的人手下干 |
| 朋友 | 维护商界人脉，定期保持联系 |
| 好点子 | 逆势而行，准备好受人奚落 |
| 信心 | 对别人的好点子有信心（麦当劳兄弟卖掉了点子） |
| 自己 | 这是最差的马——别只靠自己 |

## 七、定位执行检查清单

### 7.1 战略层面

- [ ] 我们在目标受众头脑中的当前位置是什么？（调研确认，不是内部猜测）
- [ ] 我们想要的定位是否已被别人占据？
- [ ] 我们选的定位能否用**一句话**说清楚？
- [ ] 这个定位是否与竞争对手形成了**明确区隔**？
- [ ] 我们有足够资金在一个市场把这个定位**打响**吗？

### 7.2 执行层面

- [ ] 品牌名是否能启动定位程序？
- [ ] 所有传播内容是否围绕**同一个**定位概念？
- [ ] 广告是否**简单到自身就是战略**的程度？
- [ ] 我们是否抵制了产品延伸的诱惑？
- [ ] 我们是否从外部（预期客户）角度看问题，而非内部角度？

### 7.3 持续层面

- [ ] 我们是否**年复一年**坚持基本定位战略？
- [ ] 我们是否在战术层面不断创新（而非改变战略）？
- [ ] 我们是否及时拦截了竞争对手的挑战？
- [ ] 我们是否避免了FWMTS陷阱（忘记使我们成功的根本）？

## 八、常见定位错误与对策

| 错误 | 症状 | 对策 |
|------|------|------|
| 由内而外思维 | "我们产品更好，消费者应该选择我们" | 从预期客户头脑开始思考 |
| 人人满意陷阱 | 产品/广告试图覆盖所有人 | 选定一个细分位置，接受有所损失 |
| 产品延伸诱惑 | 用知名品牌名命名新产品 | 新产品用新品牌名 |
| 对比广告误用 | "我们比对手强" | 改为"对手有这个问题" |
| FWMTS陷阱 | 成功后改变定位 | 坚持使你成功的定位 |
| 工厂空当思维 | "填补产品线空白" | 转向"填补消费者心智空白" |
| 简称/缩写热 | 不知名公司用缩写 | 先用全名出名，再考虑缩写 |
| 多样化定位 | "我们是多元化企业" | 找到一个核心定位概念 |

## 九、失败模式与反模式（Failure Modes & Anti-Patterns）

### 9.1 定位失败的六种死法

```
模式1：内部视角病
  症状：决策者用自己的认知代替消费者认知
  信号：会议中频繁出现"我们觉得"、"我们的技术领先"
  后果：定位与消费者心智脱节，传播无效
  纠偏：强制要求每次定位讨论前展示消费者调研数据

模式2：贪大求全病
  症状：定位试图覆盖多个品类或人群
  信号：一句话说不清定位，或者用了"and"连接多个概念
  后果：消费者记不住任何东西，梯子上没有你的位置
  纠偏：砍到只剩一个词/一个概念，接受失去部分市场

模式3：自我感动病
  症状：为定位投入大量创意，但消费者完全get不到
  信号：内部员工觉得广告"有创意"，外部消费者无感
  后果：钱花了，心智位置没建立
  纠偏：用"5秒测试"——5秒内能否理解核心信息

模式4：朝令夕改病
  症状：每1-2年换一次定位
  信号：市场部换人后推翻前任策略，或老板看到新趋势就想变
  后果：消费者心智中品牌形象模糊，信任感丧失
  纠偏：定位变更需通过"三年检验"——能否坚持3年以上

模式5：延伸上瘾病
  症状：成功后用同一品牌名进入新品类
  信号：品牌名出现在3个以上不同品类
  后果：跷跷板效应——新品类的销量来自旧品类的损失
  纠偏：严格执行购物单检验法和酒保检验法

模式6：跟进模仿病
  症状：看到领导者做什么就做什么
  信号：对标企业的产品线、广告风格、定价都与领导者相似
  后果：永远活在领导者阴影下，被定义为"仿品"
  纠偏：反过来想——领导者做什么你不做
```

### 9.2 反模式清单（遇到就警报）

| 反模式 | 识别信号 | 风险等级 | 立即行动 |
|--------|---------|---------|---------|
| 领导者自大 | "我们是第一，不用管对手" | 🔴 高 | 重新审视心智地图，检查是否有品类分化趋势 |
| 品牌搭便车 | "用现有品牌名推出新品" | 🔴 高 | 运行产品延伸决策矩阵（4.1） |
| 定位模糊 | "我们的定位是高品质+创新+服务好" | 🔴 高 | 砍到只剩一个核心概念 |
| 跟风定位 | "竞品打高端我们也打高端" | 🟡 中 | 找领导者的反面空位 |
| 内部命名 | "用公司内部术语做品牌名" | 🟡 中 | 运行命名检验（3.1） |
| 广告万能论 | "多投广告就能建立定位" | 🟡 中 | 先确认定位概念本身是否正确 |
| 短期思维 | "这个定位Q3才能见效，换个快的" | 🟠 中低 | 检查是否陷入朝令夕改模式 |

## 十、定位健康度检查点（Checkpoints）

### 10.1 启动前检查点（Go/No-Go Gate）

```
在投入资源前，必须全部通过：

□ CP-1 消费者心智扫描完成
  → 你能否画出品类梯子？能否说出前3名分别代表什么？

□ CP-2 定位概念一句话测试通过
  → 能否用一句不超过10个字的话说清你的定位？

□ CP-3 竞争区隔确认
  → 你的定位与最近的竞品是否有至少一个明确差异点？

□ CP-4 品牌名匹配度确认
  → 品牌名是否能启动定位程序？（运行3.1检验标准）

□ CP-5 资源匹配确认
  → 你是否有足够资金在目标市场把定位打透？

任一未通过 → 回到第一步重新定位，不要强行推进
```

### 10.2 执行中检查点（季度回顾）

```
□ CP-6 传播一致性
  → 所有渠道的传播是否围绕同一个定位概念？有没有走偏？

□ CP-7 延伸诱惑监控
  → 是否有人提议用现有品牌名进入新品类？如有，运行4.1矩阵

□ CP-8 竞争态势监控
  → 领导者是否在向你的位置移动？是否有新品牌试图抢占你的位置？

□ CP-9 消费者认知验证
  → 最近一次消费者调研中，他们用什么词描述你的品牌？与定位一致吗？
```

### 10.3 年度战略检查点

```
□ CP-10 FWMTS检测
  → 你是否还在坚持当初使你成功的那个核心定位？
  → 如果你"升级"了定位，是真的升级还是忘记了根本？

□ CP-11 品类趋势检查
  → 你所在的品类是否正在分化？是否有新品类正在形成？
  → 如果品类在衰退，是坚守还是提前布局新品类？

□ CP-12 梯子位置复核
  → 你在消费者心智梯子上的位置变化了吗？上升了还是下降了？
  → 消费者把你的品牌与什么概念关联？这个关联是否在强化？

未通过 → 触发定位战略复盘，但不要轻易改变核心定位
```
