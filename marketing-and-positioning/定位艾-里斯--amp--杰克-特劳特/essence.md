# 定位：头脑争夺战

> **作者**：阿尔·里斯（Al Ries）、杰克·特劳特（Jack Trout）
> **首版**：1981年（20周年纪念版2001年）
> **一句话总结**：在传播过度的社会中，赢得竞争的关键不是改变产品，而是在预期客户头脑中占据一个独特、有价值的位置。

---

## 一、核心定义

**定位不是你对产品要做的事，定位是你对预期客户要做的事。** 你要在预期客户的头脑里给产品定位。

定位的基本方法不是创造出新的、不同的东西，而是**改变人们头脑里早已存在的东西，把那些早已存在的联系重新连接到一起**。

## 二、为什么要定位：传播过度的社会

- 美国人均广告消费约200美元/年，人脑早已超负荷
- 人脑同时只能处理**7个以内**的信息单位（哈佛心理学家乔治·米勒）
- 每个产品类别中，消费者通常只能记住1-2个品牌
- **大脑是阻隔过度传播的屏障**——只接受与先有知识或经验相适应的信息

> 极其简化的信息是唯一的出路。传播和建筑一样，越简洁越好。

## 三、三大时代演变

| 时代 | 核心策略 | 代表人物 |
|------|---------|---------|
| 产品至上（50年代） | USP——独特的销售主张 | 瑞夫斯 |
| 形象至上（60年代） | 品牌形象长期投资 | 大卫·奥格威 |
| 定位至上（70年代以来） | 创造心理位置，强调第一 | 里斯 & 特劳特 |

## 四、脑中小阶梯：心智模型

人脑中有一组**产品梯子**，每个梯子代表一类产品，每层上有个品牌。最多**七层**。

### 进军大脑的捷径：争当第一

- 第一个独自飞越北大西洋的人？林德伯格。第二个？没人记得。
- 第一品牌占据的市场份额通常是第二品牌的**两倍**、第三品牌的**三倍**
- 柯达（照相）、IBM（计算机）、施乐（复印机）、赫茨（租车）——"第一"的力量
- 关键：**第一个打入预期客户的大脑**，而不是发明最好的产品

### 如果你不是第一

- **对比定位**：艾维斯——"我们屈居第二，所以我们更努力"
- **非可乐定位**：七喜——把自己挂到可乐梯子旁边，成为可乐的替代品
- **给竞争对手重新定位**：泰诺——通过指出阿司匹林的缺陷，抢占止痛药第一

## 五、领导者战略

### 保持领先的两条原则

1. **"真东西"战略**（如可口可乐）——强化最初印象，你就是标准
2. **不露声色**——一旦发现有前景的新产品/新概念就采用，别等醒悟过来为时已晚

### 多品牌拦截（宝洁模式）

- 每个品牌有**独特定位**：汰渍（洁白）、奇尔（更白）、Bold（酒后生辉）
- 多品牌战略 = 单一定位战略 = 以不变应万变
- 不要用产品延伸，而是推出新品牌

### 反应要快

- 强生公司的泰诺在Datril发起价格战时**立刻降价拦截**，彻底击退对手

## 六、跟随者战略：找空子

法国人的智慧：**Cherchez le créneau**（找空子）

| 空子类型 | 案例 |
|---------|------|
| 尺寸空子 | 大众甲壳虫——"往小里想" |
| 高价空子 | 米什劳啤酒、芝华士、哈根达斯 |
| 低价空子 | 奎普传真机（月租29美元 vs 施乐45美元） |
| 性别空子 | 万宝路（男性化）、弗吉尼亚细枝（女性化） |
| 年龄空子 | 目标牌牙膏（儿童） |
| 时段空子 | Nyquil（夜间感冒药） |
| 渠道空子 | L'eggs连裤袜（超市渠道） |

**关键**：填补的是**头脑里的空当**，不是工厂里的空当。

## 七、给竞争对手重新定位

> 没有空子可钻时，必须通过给已经在人们头脑里占有一席之地的竞争对手重新定位来创建空当。

- **泰诺**：为阿司匹林重新定位——"会刺激胃黏膜，引起哮喘或过敏反应"
- **皇家道尔顿**：为雷诺克斯重新定位——"英国瓷器 vs 美国瓷器"
- **斯多里奇纳亚伏特加**：为美国伏特加重新定位——"它们都产于美国"
- **智慧牌薯片**：为品客重新定位——"脱水土豆、甘油二酸酯 vs 土豆、植物油和盐"

**重新定位 ≠ 对比性广告**。对比广告说"我们比对手强"，重新定位则**从根本上动摇**竞争对手在人们心中的地位。

## 八、名字的威力

> 名字就是把品牌挂在预期客户头脑中产品梯子上的钩子。

### 好名字的特征
- 能启动定位程序，告诉预期客户产品的主要特点
- 海飞丝、呵护、苗条、DieHard、Edge——**名字即定位**
- 近似通用但又不完全通用

### 坏名字的代价
- 东方航空——地区性名字，暗示只飞东海岸
- 古德里奇——总被误认为固特异
- Metrecal——计算机发明的词，不如"苗条"
- 含义过于宽泛/狭窄/过时的名字

### 缩写名的陷阱
- IBM、GE出名在先，简称在后（因果关系不可颠倒）
- 不知名公司用缩写 = 自杀行为
- 耳朵优先于眼睛：名字首先要**听起来好**

## 九、产品延伸陷阱（全书重点警告）

> 过去十年最重要的营销动向，也是最危险的陷阱。

### 跷跷板原则
**一个名字不能代表两个彼此完全不同的产品。一个起来的时候，另一个就会下去。**

### 产品延伸为什么诱人
- 短期确实有效（零售商会进货，初期销量好看）
- 经济学上合理（节省广告费）
- "道理都在产品延伸那一边，但真理不在那一边"

### 产品延伸为什么有害
- 削弱品牌在头脑中的**清晰印象**
- 使品牌从"通用名称"退化为"品牌名称"
- 长期损害致命：蓓尔美尔从第一品牌跌落、大众从进口车第一跌至第四

### 经典失败案例

| 品牌 | 延伸方式 | 结果 |
|------|---------|------|
| 蓓尔美尔 | 清凉型/超淡/轻型 | 从第一跌至第二，份额逐年下降 |
| 蛋白质21 | 香波→定型水→护发素→香水 | 香波份额从13%跌至2% |
| 斯各特 | 纸巾/卫生纸/纸尿布 | "斯各特"不代表任何东西 |
| 救星 | 糖果→口香糖 | 口香糖市场份额始终一位数 |
| 拜耳 | 阿司匹林→不含阿司匹林的拜耳 | 止痛药市场份额逐年下降 |
| 施乐 | 复印机→计算机 | 损失8440万美元 |

### 购物单检验法

写下：舒洁、佳洁士、李斯特灵、救星、拜耳、日晷。让家人去买——买回来的全是**原始产品**。延伸产品在头脑里没有自己的位置。

### 什么时候产品延伸可以管用

条件极其苛刻：
1. 竞争对手很愚蠢
2. 你的销量很小
3. 你没有竞争对手
4. 你不想建立定位
5. 你不做任何广告

## 十、公司定位

### 孟山都案例：行业领先之路
- 产品领先？杜邦已占此位——不可行
- 业务领先（自由企业）？已被太多公司占了——不可行
- **行业领先**：为化学品辩护，成为行业代言人——"生活中的化学常识"
- 结果：公众对化学工业支持率从36%升至42%

### 公司定位的核心
- 名字代表某个事物（不是"什么都不是"）
- 不要追求"多样化"定位——它几乎没有意义
- 领导地位可以在银行里兑换成现金

## 十一、国家/区域/个人定位

### 比利时案例
- 问题：在游客头脑中几乎没有位置
- 解法：利用《米奇林指南》——"美丽的比利时有五个阿姆斯特丹"
- 教训：混淆是定位的大敌，简单化是定位的救星

### 个人定位六步法
1. **给自己下定义**——选定一个具体概念与自己联系
2. **要敢于犯错**——值得一做的事就值得乱做一气
3. **起个合适的名字**——避免无名陷阱和产品延伸
4. **找匹马骑**——公司、上司、朋友、好点子、信心、自己
5. **坚持你的定位**——年复一年钉在那儿
6. **保持客观**——从外部角度看自己

## 十二、通往成功的六个步骤

1. **你处在什么位置上？**——从预期客户头脑里的观念开始
2. **你想拥有什么样的位置？**——关键词是"拥有"，别做不可能的事
3. **谁是你必须要超过的？**——绕开障碍胜过克服它
4. **你有足够的资金吗？**——在有限资金下，宁可在一个城市花过头
5. **你能坚持下去吗？**——定位是累计概念，基本战略几乎从不改变
6. **你与你的地位相称吗？**——创新只有在为定位目标服务时才有价值

## 十三、定位思维的精髓

### 由外而内（Outside-In）

> 你必须从预期客户的角度，而不是从你自己的角度看问题。

- 不要在产品里寻找答案，要在预期客户头脑里寻找
- 把观念当作现实来接受，然后重构这些观念
- "顾客永远是对的"——商家/传播者永远是错的

### FWMTS陷阱（忘记使你成功的根本）

- 艾维斯："我们要当第一"——失败
- 七喜："美国处处有七喜"——失败
- 一旦建立了定位，**永远不要离开它**

### 定位游戏的玩法

| 要素 | 要求 |
|------|------|
| 眼光 | 确定基础业务，坚持五年十年 |
| 勇气 | 在形势未定时投入营销资金 |
| 客观 | 排除虚荣心，从外部看自己 |
| 简单 | 只有一目了然的点子才行得通 |
| 精明 | 在两极中间找到空当 |
| 耐心 | 地域/阶层/年龄逐步推进 |
| 全球观 | 利用一国定位打入另一国 |
| 为他人想 | "为他"型营销人看得更清楚 |
