# SKILL.md — 定位与心智营销技能库
**来源：《新定位》杰克·特劳特**

## 技能概要

本技能库基于特劳特定位理论，提供一套系统化的品牌定位、重新定位和心智营销方法论。核心能力是在顾客心智中针对竞争对手确立最具优势的位置。

## 核心技能

### 1. 心智诊断（Mind Diagnosis）
**用途**：了解目标顾客心智中对你和竞争对手的已有认知

**操作步骤**：
1. 列出产品品类的6个基本特性
2. 让顾客对所有竞争者在每个特性上打1-10分
3. 绘制心智地图，看清各品牌占据什么概念
4. 找出未被占据的特性或对手的薄弱环节
5. 确定你要占据的心智位置

**关键原则**：
- 不问消费者"你想要什么"，只问他们对现有品牌的认知
- 调研竞争对手比调研消费者更重要
- 获取的是"心智快照"，不需要深度思考

**检查点**：
- [ ] 是否调研了至少3个主要竞争对手？
- [ ] 心智地图是否清晰显示各品牌占据的概念？
- [ ] 是否找到了未被占据的差异化位置？
- [ ] 调研对象是否为目标顾客群（非内部员工）？

---

### 2. 定位战略制定（Positioning Strategy）
**用途**：为品牌确立一个简洁、差异化的概念

**操作步骤**：
1. **找到显而易见的概念**：内部显而易见的通常对顾客也显而易见
2. **聚焦一个词/概念**：佳洁士=防蛀，沃尔沃=安全
3. **关联已有认知**：用顾客心智中已有的观念作为起点
4. **确认竞争框架**：每个定位都需要一个对手
5. **极度简化**：删掉所有其他品牌也能说的、需要复杂分析的、不匹配已有认知的内容

**输出模板**：
```
品牌名 + [品类] + [差异化概念]
示例：
- 莲花发展公司，先是电子制表软件，如今是群组软件
- 毕马威，全球领导者
- 卡维尔，冰淇淋蛋糕店
```

**检查点**：
- [ ] 定位能否用一句话说清？
- [ ] 是否聚焦于一个词/概念？
- [ ] 是否有明确的竞争对手作为参照？
- [ ] 顾客能否在5秒内理解你的定位？
- [ ] 定位是否与顾客已有认知关联？

**禁忌**：
- 不要矫揉造作（让创意人士改得面目全非）
- 不要品牌延伸（一个品牌只代表一个概念）
- 不要盲目改进（内部"改进"往往造成消费者认知混乱）
- 不要执着未来（先解决当前，再投资未来）

---

### 3. 重新定位（Repositioning）
**用途**：当市场变化、品牌失去焦点或竞争加剧时，重新确立品牌位置

**触发条件**：
- 消费者态度发生转变
- 技术超越现有产品
- 品牌偏离了消费者心智中的已有认知
- 竞争者大量涌入，品牌失去焦点

**操作步骤**：
1. **诊断当前认知**：品牌在顾客心智中代表什么？
2. **识别变化**：市场、技术、消费者态度发生了什么变化？
3. **转移战场**：在新标准上确立优势（毕马威案例）
4. **关联旧认知**：用旧概念作为新定位的起点（莲花案例）
5. **CEO全身心投入**：内部营销比外部更难
6. **循序渐进**：给组织时间适应变化

**经典模式**：
- **回归根本**：西尔斯出售金融服务回归零售，施乐回归复印机
- **品类转移**：莲花从电子制表转为群组软件
- **焦点转移**：卡维尔从软冰淇淋转为冰淇淋蛋糕
- **战场转移**：毕马威从"六大之一"转为"全球领导者"

**检查点**：
- [ ] 是否确认市场发生了真实变化（非内部焦虑）？
- [ ] 新定位是否关联了顾客已有认知？
- [ ] CEO是否愿意全身心投入并承担短期阵痛？
- [ ] 是否有分阶段执行计划（非一步到位）？
- [ ] 内部是否做好了"有人会离开"的准备？

### 4. 品牌命名（Brand Naming）
**用途**：创建能启动定位的品牌名称

**命名原则**：
1. **关联产品利益**：名字直接挂钩产品价值（永久电池、海飞丝）
2. **确保好听**：心智靠耳朵运转，名字被说出来比被看到多
3. **易记**：利用声音重复和押韵
4. **慎用缩写**：缩写不是名字，易记度低40%
5. **慎用生造词**：只在万众期待的真正新产品时使用
6. **考虑多语言**：避免在目标市场语言中有负面含义
7. **可获得性**：10个名字中9个已被注册

**子品牌策略**：
- 当母品牌有固定含义可能引起混淆时，使用子品牌
- 子品牌大写，公司名小写（如"灰净 by 百得"）
- 子品牌可以新方式进入心智，避免老品牌的包袱

**品类命名原则**：
- 品类由用户创造，不由公司创造
- 分析产品功能，用消费者能理解的语言描述
- 对既有品类进行细分优于创造全新品类
- 避免有负面含义的品类名

**检查点**：
- [ ] 名字是否关联产品利益/定位？
- [ ] 大声说出来是否好听？
- [ ] 在目标市场语言中是否有负面含义？
- [ ] 商标是否可获得？
- [ ] 是否避免了缩写和生造词（除非万众期待的新品）？

### 5. 广告与传播策略（Advertising & Communication）
**用途**：基于心智运作规律制定有效传播

**核心洞察**：
- 心智靠耳朵运转（回声记忆4-5秒 > 影像记忆1-2秒）
- 听觉信息比视觉信息记更多
- 单纯语言信息比语言+图像更能说服购买

**操作原则**：
1. **语言为王**：书面文字承载主要销售信息，视觉为辅
2. **标题要读起来好听**：押韵和节奏感增强记忆
3. **电视广告中声音为主**：绝不能让视觉画面压过声音
4. **生活片段广告有效**：提供"过往经验"便于关联
5. **利用情感**：情感只能用来传递销售信息，不能单独使用
6. **新闻性**：有新闻性的标题注目率更高
7. **类比**：把新信息同已有观念关联（"直播电视就是我的音像店"）

**检查点**：
- [ ] 广告的核心信息是否用语言（非纯视觉）传递？
- [ ] 标题是否押韵或有节奏感？
- [ ] 是否关联了顾客已有认知？
- [ ] 情感是否服务于销售信息（非独立存在）？
- [ ] 电视广告中声音是否为主导？

---

### 6. 公关先行策略（PR-First Strategy）
**用途**：通过第三方背书建立品牌合法性

**操作原则**：
1. **公关先行，广告第二**：公关播种，广告收获
2. **时间性项目**：各组成部分在一段时间内慢慢展开，互相强化
3. **一致性**：年复一年坚持同一概念
4. **利用争议**："你反对什么"比"你是什么"更有新闻价值
5. **保护光环**：不要滥用声誉和公信力

**执行顺序**：
```
行业媒体 → 科学/专业媒体 → 商业/金融媒体 → 主流消费者媒体
```

**检查点**：
- [ ] 是否在广告之前建立了公关基础？
- [ ] 媒体推进是否按行业→专业→商业→消费者顺序？
- [ ] 是否年复一年坚持同一概念？
- [ ] 是否保护了品牌公信力（未滥用）？

---

### 7. 多品牌战略（Multi-Brand Strategy）
**用途**：用多个互补品牌覆盖不同细分市场

**操作原则**：
- 不同品牌名、不同定位、不同目标客群
- 品牌之间互补而非竞争
- 需要高管委员会监督差异化和资源分配
- 每个品牌有自己的销售和市场部门

**案例**：
- 吉列四品牌（TracⅡ、Atra、Sensor、Good News）= 65%市场份额
- 睿烁三品牌（睿烁=汽车、坎普萨=服务、波特诺=价格）

**检查点**：
- [ ] 各品牌是否有明确不同的定位？
- [ ] 品牌之间是否互补而非自相残杀？
- [ ] 是否有高管委员会监督差异化？
- [ ] 每个品牌是否有独立的销售和市场团队？

## 失败模式（Failure Modes）

| 失败模式 | 根因 | 早期信号 | 补救措施 |
|----------|------|----------|----------|
| **定位模糊** | 试图代表太多概念 | 顾客无法用一句话描述品牌 | 砍掉次要概念，聚焦一个词 |
| **内部共识失败** | CEO未全力投入，高管各执一词 | 内部会议反复讨论定位但无结论 | CEO亲自站台，用"世界变了"框架统一认知 |
| **品牌延伸失控** | 短期营收压力驱动 | 同一品牌出现在不相关品类 | 启用子品牌或新品牌，止损回归核心 |
| **调研方法错误** | 问消费者"你想要什么" | 调研结果与市场表现矛盾 | 只问消费者对现有品牌的认知，不问需求 |
| **创意凌驾策略** | 广告公司追求获奖 | 广告好看但顾客记不住品牌 | 坚持"语言为王"，视觉为辅 |
| **公关缺位** | 直接砸广告，无第三方背书 | 广告可信度低，转化差 | 先做公关播种，再做广告收获 |
| **品类命名错误** | 用公司视角而非用户视角命名 | 顾客不理解品牌属于什么品类 | 用消费者语言描述品类，细分优于创造 |
| **过早放弃** | 定位需要时间但内部失去耐心 | 6个月内无明显效果就改方向 | 定位是长期投资，至少坚持18个月 |

## 反模式（Anti-Patterns）

### ❌ 反模式1：问卷问需求
**错误做法**：问消费者"你希望产品有什么功能？"
**正确做法**：只问消费者对现有品牌的认知和评价
**原因**：消费者不知道自己想要什么，但清楚知道现有品牌好在哪、差在哪

### ❌ 反模式2：定位PPT化
**错误做法**：定位描述长达3页PPT，包含使命、愿景、价值观
**正确做法**：定位=一个词/概念+竞争对手
**原因**：心智厌恶复杂，复杂=不存在

### ❌ 反模式3：视觉主导广告
**错误做法**：广告以震撼画面为核心，语言为注脚
**正确做法**：语言承载销售信息，视觉辅助强化
**原因**：回声记忆4-5秒 > 影像记忆1-2秒，耳朵主导心智

### ❌ 反模式4：品牌延伸贪多
**错误做法**：用一个强势品牌覆盖所有新品类
**正确做法**：每个新品类用新品牌或子品牌
**原因**：品牌在心智中只能代表一个概念，延伸即稀释

### ❌ 反模式5：英雄式推翻
**错误做法**：新CEO上任否定前任所有定位决策
**正确做法**：用"世界变了"框架重新评估，保留有效部分
**原因**：推翻前任=推翻顾客已有认知，代价巨大

### ❌ 反模式6：广告直接上
**错误做法**：新品牌一出来就砸大广告
**正确做法**：公关先行播种，建立第三方背书后再广告收割
**原因**：没有公关基础的广告缺乏可信度

### ❌ 反模式7：跟随领导者
**错误做法**：模仿行业领导者的定位和策略
**正确做法**：找到领导者未占据的差异化位置
**原因**：心智只记得第一，第二用同样策略永远是第二

## 决策检查清单

### 启动新定位前
- [ ] 是否进行了心智诊断（竞争对手在顾客心智中的认知）？
- [ ] 定位概念是否显而易见？
- [ ] 是否聚焦于一个词/概念？
- [ ] 是否有明确的竞争对手？
- [ ] CEO是否全力支持？

### 品牌延伸前
- [ ] 是否会让品牌在顾客心智中丧失焦点？
- [ ] 是否有高度聚焦的专家品牌作为竞争对手？
- [ ] 延伸后负面公关是否会波及全品牌？
- [ ] 是否考虑过使用新品牌或子品牌？

### 重新定位前
- [ ] 市场是否发生了真正变化？
- [ ] 当前定位是否真的不再有效？
- [ ] 新定位是否关联了顾客已有认知？
- [ ] 内部是否做好了准备（可能有人离开）？

### 命名检查
- [ ] 名字是否关联产品利益/定位？
- [ ] 大声说出来是否好听？
- [ ] 在目标市场语言中是否有负面含义？
- [ ] 商标是否可获得？

## 常见陷阱速查

| 陷阱 | 表现 | 解法 |
|------|------|------|
| 品牌延伸 | 一个品牌覆盖多个品类 | 启用新品牌或子品牌 |
| 矫揉造作 | 定位描述被创意人士改得面目全非 | 保持简单直白 |
| 盲目改进 | 不断推出新品种、新口味 | 让品牌"稳步向前" |
| 执着未来 | 为了容纳未来计划而模糊当前定位 | 先解决当前成功之道 |
| 纠结数字 | 为短期营收目标牺牲长期定位 | CEO介入，保持耐心 |
| 英雄情结 | 新官上任否定前任所有决策 | 用"世界变了"框架沟通 |
| 改变心智 | 企图让消费者改变已有认知 | 利用已有认知，不要对抗 |
