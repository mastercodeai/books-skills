# SKILL.md — 改变消费者行为的十大策略

## Skill 定义

基于亚当·费里尔的行为改变框架，运用10个"行动刺针"和行为经济学原理来设计营销策略、改变消费者行为、提升传播效果。

---

## 何时触发此 Skill

- 设计营销/广告/传播活动时
- 需要说服他人改变行为时（不限于商业场景）
- 产品上市、品牌重塑、定价策略制定时
- 需要提高用户参与度/转化率时
- 策划众筹、募款、社会运动时
- 任何需要"让人行动起来"的场景

## 场景路由（Scene Routing）

根据用户意图快速路由到最优刺针组合和行动路径：

### 🔀 路由表

```
IF 场景 = "销售/成交" →
  主路径: 化繁为简 → 敢承诺 → 重塑
  核心目标: 降低购买摩擦 + 建立承诺惯性 + 重塑价值认知
  关键指标: 转化率、客单价、复购率
  行动顺序: ①消除购买障碍 ②设计小承诺入口 ③重塑产品价值框架

IF 场景 = "营销活动/campaign" →
  主路径: 动之以情 → 集体主义 → 归属感
  核心目标: 制造情感引爆点 + 引发群体参与 + 深化品牌归属
  关键指标: 传播量、参与率、品牌提及率
  行动顺序: ①设计情感冲击内容 ②创造"人人都在做"氛围 ③让参与者产生拥有感

IF 场景 = "行为改变/习惯养成" →
  主路径: 化繁为简 → 赋能技能 → 敢承诺 → 玩乐
  核心目标: 移除障碍 + 教会方法 + 建立承诺 + 增加趣味
  关键指标: 行为发生率、持续率、完成率
  行动顺序: ①诊断并消除环境障碍 ②提供技能培训 ③设计阶梯式承诺 ④游戏化激励

IF 场景 = "说服/影响决策" →
  主路径: 重塑 → 样板 → 动之以情
  核心目标: 改变认知框架 + 提供社会证据 + 触发情感共鸣
  关键指标: 态度转变率、说服成功率
  行动顺序: ①重新框定问题 ②展示可信榜样 ③激发情感认同

IF 场景 = "品牌建设/定位" →
  主路径: 重塑 → 归属感 → 集体主义
  核心目标: 建立差异化认知 + 创造用户共创 + 形成品牌社群
  关键指标: 品牌认知度、NPS、社群活跃度
  行动顺序: ①重塑品牌价值框架 ②设计用户参与共创 ③构建品牌共同体
```

```

关键词 → 路由映射:
  "卖货/成交/转化/购买" → 销售路由

```

## 核心操作框架

### 第一步：明确目标
```
目标 = 可量化的结果（不是行为本身）
示例：3个月内会员增长30%
反例：让更多人了解我们的品牌（模糊）
```

### 第二步：选择要改变的行为
用四问诊断法评估候选行为：
1. **个体激励**：做这件事对他们有什么好处？奖励够大吗？
2. **社会常规**：做这件事别人会怎么看？有没有面子问题？
3. **能力**：他们具备做这件事的技能和资源吗？
4. **机会**：所处环境允许这个行为发生吗？

选择标准：
- 高动机 + 高容易度 → 最优先
- 规模足够大（能影响多少人？频率多高？普及率多大？）

### 第三步：选择行动刺针

根据诊断结果匹配刺针：

**如果问题是"动机不足"：**
| 障碍所在 | 推荐刺针 | 适用场景 |
|----------|---------|---------|
| 个体激励弱 | 重塑、实用性 | 需要重新包装好处时 |
| 社会常规负面 | 集体主义、样板 | 需要改变"别人怎么看"时 |
| 需要情感驱动 | 动之以情 | 行为已根深蒂固需"一巴掌打醒" |
| 需要参与感 | 归属感 | 可让消费者参与创造时 |
| 可以游戏化 | 玩乐 | 能控制环境时 |

**如果问题是"容易度不足"：**
| 障碍所在 | 推荐刺针 | 适用场景 |
|----------|---------|---------|
| 缺技能 | 赋能技能 | "我不知道怎么做" |
| 有障碍 | 化繁为简 | 环境/流程阻塞 |
| 改变太大 | 敢承诺 | 需要从小步开始 |

### 第四步：发展创意
刺针是策略方向，创意是执行灵魂。要求：
- 刺针应用需有"使命"包装（让行为有意义）
- 创意要让人"想参与"而非"被告知"
- 优先设计可互动、可分享的执行形式

### 第五步：执行 + 评估
- 写出完整执行计划
- 设定行为指标和时间节点
- 持续监测→调整→迭代

## 10 个刺针速查卡片

### 1️⃣ 重塑 Reframing
**公式**：同一内容 × 不同包装 = 不同认知
**操作要点**：
- 利用系统1的自动判断（命名、包装、价格、渠道全部一致地支撑新形象）
- 损失框架 > 收益框架（"不做会失去什么"比"做了会得到什么"更有力）
- 参照物决定认知——选择有利的对比对象
- 提价比降价更容易建立高端感知
**检查清单**：品牌名是否支撑定位？包装是否一致？价格信号是否匹配？渠道是否强化形象？

### 2️⃣ 动之以情 Evocation
**公式**：情感冲击 → 注意力捕捉 → 行动力 → 行动
**操作要点**：
- 8种核心情感：喜悦、忧伤、信赖、讨厌、恐惧、愤怒、惊讶、期待（5个负面>1个正面）
- 负面偏见：负面信息天然更能抓注意力
- 病毒传播三要素：惊奇 + 张力 + 欢乐
- 情感只是"开门"——必须接上行动闭环（ZMOT零点关键时刻）
**检查清单**：是否足够震撼？是否避免了理性信息干扰情绪？制作质量是否达标？行动路径是否清晰？

### 3️⃣ 集体主义 Collectivism
**公式**：服从性 + 从众心态 + 行动 + 使命 = 集体行动
**操作要点**：
- 创造"好像每个人都在做"的感觉（即使最初人很少）
- 赋予崇高使命（拯救XX/改变XX/支持XX）
- 让参与行为可见（徽章、T恤、面具、标签）
- 利用媒体放大"群体感"
**检查清单**：指令是否明确？参与是否容易可见？是否有感召力的使命？社交媒体传播路径是否打通？

### 4️⃣ 归属感 Ownership
**公式**：参与创造 → 投入精力 → 更高价值感知 → 更强行为倾向
**操作要点**：
- 三个组成：自主权 + 个人相关性 + 认知失调
- 强度梯度：征求意见 < 共同创造 < 占有 < 自主创造
- 即使简单征询意见也会产生归属感（霍索恩效应/焦点座谈会效应）
- 富兰克林效应：让消费者帮品牌一个忙
**检查清单**：消费者是否参与了创造？是否有自主决定的空间？参与后是否感受到拥有？

### 5️⃣ 玩乐 Play
**公式**：枯燥行为 + 游戏机制 = 享受体验
**操作要点**：
- 需要对环境有掌控力
- 积分、挑战、竞赛、排行榜
- 正向激励（奖励好行为）比惩罚坏行为更有效
**检查清单**：行为是否可以游戏化？奖励机制是否吸引人？环境是否可控？

### 6️⃣ 实用性 Utility
**公式**：行为 + 额外实际价值 = 更高动机
**操作要点**：
- 提供与品牌相关的有用工具/服务/信息
- 让消费者在行动中获得即时回报
- 例：APP提供实时数据、工具、便利功能
**检查清单**：附加价值是否与品牌相关？是否真正实用？消费者是否愿意为此行动？

### 7️⃣ 样板 Modelling
**公式**：看到他人做 → 自己也做
**操作要点**：
- 班杜拉社会学习理论：观察→模仿
- 样板人物需具备：高知名度 + 高信赖度 + 与目标行为相关
- 可以是名人、KOL、同行、"隔壁老王"
**检查清单**：样板人物与目标人群是否相关？行为示范是否明确？是否有可信度？

### 8️⃣ 赋能技能 Skill Up
**公式**：教会技能 → 降低能力门槛 → 行为发生
**操作要点**：
- 教学方式：体验式 > 告知式
- 降低学习曲线（简化步骤、提供工具、手把手教）
- 例：品酒之夜、烹饪课、使用教程
**检查清单**：目标人群缺什么技能？教学方式是否体验导向？是否有跟进机制？

### 9️⃣ 化繁为简 Eliminate Complexity
**公式**：移除障碍 → 行为自然发生
**操作要点**：
- 列出所有可能的环境障碍（支付、流程、距离、时间、认知复杂度）
- 逐一消除
- 经典案例：健康食物放前面、小盘子装食物、一站式流程
**检查清单**：行为链条上有几步？每一步的摩擦在哪里？能否减少到最少？

### 🔟 敢承诺 Commitment
**公式**：小承诺 → 认知失调 → 大承诺
**操作要点**：
- "脚在门槛效应"（Foot-in-the-door）：先小后大
- 公开承诺比私下承诺更有力
- 签字、宣言、投票、加入社群都是承诺形式
**检查清单**：最小承诺是什么？承诺是否公开？后续大请求是否自然衔接？

## 失败模式（Failure Modes）

### 🚨 策略失败模式

| 失败模式 | 根因 | 预警信号 | 补救措施 |
|----------|------|---------|---------|
| **情感驱动但无行动闭环** | 只做了"开门"没做"引导" | 传播量高但转化低 | 在情感触点后立即嵌入行动入口（ZMOT设计） |
| **重塑不一致** | 品牌名/包装/价格/渠道信号矛盾 | 消费者认知混乱、定位模糊 | 统一所有触点的信号一致性检查 |
| **单一刺针失效** | 复杂行为改变需要组合拳 | 尝试一次后行为无变化 | 诊断障碍类型，叠加2-3个互补刺针 |
| **动机和容易度只解决一个** | 只关注"想不想"忽略"能不能"（或反之） | 有意愿但不行动 / 行动门槛低但没人做 | 用四问诊断法找出短板并补齐 |
| **行为跨度太大** | 从A直接跳到Z，中间没有过渡 | 用户流失率高、中途放弃 | 设计阶梯式承诺（小步→中步→大步） |
| **样板人物错配** | 代言人/榜样与目标人群脱节 | 关注度高但行为转化低 | 选择与目标人群身份认同匹配的样板 |
| **游戏化过度** | 奖励机制喧宾夺主，忘了本来目的 | 用户只追奖励不关心品牌/行为本身 | 重新对齐游戏机制与核心行为目标 |
| **情感疲劳** | 反复使用同一情感刺激导致钝化 | 初期效果好但衰减快 | 轮换情感类型，升级刺激强度 |

### 🔍 诊断流程

```
策略效果不佳 →
  1. 检查: 行为目标是否明确可量化？ → 否 → 重新定义目标
  2. 检查: 四问诊断是否完整？ → 否 → 补做诊断
  3. 检查: 刺针选择是否匹配障碍类型？ → 否 → 重新选刺针
  4. 检查: 所有触点信号是否一致？ → 否 → 统一触点
  5. 检查: 是否有行动闭环？ → 否 → 补设计行动入口
  6. 检查: 行为跨度是否太大？ → 否 → 拆分阶梯
  7. 检查: 是否只用了单一刺针？ → 是 → 叠加组合刺针
```

## 反模式（Anti-patterns）

❌ **只打情感牌不做行动设计**：情感是开门，不是终点。必须在情感高峰时嵌入明确的行动入口。
❌ **只追求洞察不推动行动**：了解消费者≠改变消费者。洞察只是起点，行动设计才是交付物。
❌ **动机和容易度只解决一个**：两者缺一，行为难发生。四问诊断必须同时覆盖动机和能力。
❌ **行为改变幅度太大**：谷物片当早餐失败案例——跨越太远。用阶梯式承诺逐步推进。
❌ **重塑不一致**：名字说高端，包装像低端，渠道在超市——自相矛盾。所有触点必须统一。
❌ **依赖理性信息说服**：系统2很懒，大部分决策在系统1完成。先打情感/直觉，再提供理性支撑。
❌ **只用单一刺针**：复杂行为改变需要组合拳。至少组合2-3个互补刺针。
❌ **忽略社会常规压力**：即使个体想做，社会眼光也可能阻止。必须解决"别人会怎么看"的问题。
❌ **奖励取代内在动机**：外在奖励消失后行为也消失。设计时要逐步从外在激励过渡到内在认同。
❌ **不做A/B测试就上线**：不同刺针对不同人群效果差异大。小范围测试后再全面推开。

## 检查点（Checkpoints）

### ✅ 项目启动前检查

- [ ] 目标是否可量化、有明确时间节点？
- [ ] 目标行为是否经过四问诊断（个体激励、社会常规、能力、机会）？
- [ ] 是否选择了最优先的高动机+高容易度行为？
- [ ] 场景路由是否已确定（销售/营销/行为改变/说服/品牌）？
- [ ] 刺针选择是否匹配诊断出的障碍类型？

### ✅ 创意发展阶段检查

- [ ] 刺针是否有"使命"包装（行为有意义感）？
- [ ] 创意是否让人"想参与"而非"被告知"？
- [ ] 是否设计了可互动、可分享的执行形式？
- [ ] 所有触点（品牌名、包装、价格、渠道）信号是否一致？
- [ ] 情感触点后是否有行动闭环入口？

### ✅ 执行上线前检查

- [ ] 行为链条是否已简化到最少步骤？
- [ ] 是否设计了阶梯式承诺（小→中→大）？
- [ ] 是否有社会证据/样板支撑？
- [ ] 是否做了小范围A/B测试？
- [ ] 是否设定了行为指标和监测节点？

### ✅ 效果评估检查

- [ ] 行为发生率是否达到预期？
- [ ] 是否出现预警信号（转化低、流失高、衰减快）？
- [ ] 是否需要叠加额外刺针？
- [ ] 是否需要调整情感类型或刺激强度？
- [ ] 下一轮迭代计划是否明确？
