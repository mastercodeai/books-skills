# SKILL.md — 品牌原型营销（Brand Archetype Marketing）

> 基于《如何让品牌直击人心——品牌的12个心理原型》（The Hero and the Outlaw）

---

## Skill: brand-archetype-marketing

### Description
运用荣格原型心理学进行品牌定位、品牌内涵管理和消费者动机分析。通过12种品牌原型（archetype）帮助品牌建立永恒、跨文化的身份认同，连接消费者深层心理需求。

### When to Use
- 新品牌定位或品牌重塑
- 广告创意策略制定
- 消费者洞察与动机分析
- 品牌一致性审计（企业文化 vs 品牌身份 vs 营销传播）
- 竞品原型分析
- 跨文化品牌传播策略
- 非营利组织/政治候选人形象管理

---

## 场景路由（Scene Routing）

根据用户请求类型，选择对应工作流：

### 路由 1：原型选择（Archetype Selection）
**触发词**：新品牌定位、品牌重塑、选择原型、我们是什么品牌
**流程**：
1. 填写「品牌原型识别问卷」（下方 Step 1）
2. 用快速决策树缩小候选范围（2-3个原型）
3. 竞品原型扫描——排除已被强势占据的原型
4. 选择1个主导原型 + 确认无冲突的辅助原型（可选）
5. **检查点**：主导原型是否与核心产品功能/创始人DNA一致？

### 路由 2：品牌身份构建（Brand Identity Construction）
**触发词**：品牌故事、视觉识别、品牌语言、品牌声音
**流程**：
1. 确认已选定主导原型（未选定→先走路由1）
2. 确定含义层次（四层中选哪一层？参考 Step 4）
3. 提取该原型的营销关键词、视觉元素、叙事结构
4. 对齐企业文化 = 品牌身份 = 营销传播三层
5. **检查点**：所有触点（网站/包装/广告/客服话术）是否传达同一原型？

### 路由 3：营销策略制定（Marketing Strategy）
**触发词**：广告策略、传播方案、campaign、内容策略
**流程**：
1. 确认主导原型及含义层次
2. 按原型营销要点表制定创意方向
3. 设计原型意象符号系统（图像、颜色、故事母题）
4. 审查阴暗面风险——是否有触发公关危机的可能？
5. **检查点**：广告能否在3秒内通过原型画面抓住注意力？

### 路由 4：品牌审计（Brand Audit）
**触发词**：品牌审计、一致性检查、品牌健康度、为什么品牌失效
**流程**：
1. 诊断当前品牌实际传达的原型（消费者调研/社媒语义分析）
2. 对比品牌自认原型 vs 消费者感知原型
3. 识别原型漂移/混杂/稀释的信号
4. 输出修复建议
5. **检查点**：消费者给品牌取的昵称是否反映预期原型？

## 使用方法

### Step 1: 原型识别（品牌审计）

**问题清单：**
1. 公司的核心使命/价值观最贴近哪个原型？
2. 创始人/CEO的主导原型是什么？
3. 目标消费者心中活跃的原型是什么？
4. 产品功能和属性天然对应哪个原型？
5. 竞品的原型是什么？如何差异化？

**诊断信号：**
- 消费者给品牌取昵称 → 原型已激活
- 品牌频繁更换广告公司/形象 → 原型不清晰
- 市场份额下降但产品质量没变 → 可能原型被稀释

### Step 2: 原型选择原则

1. **唯一性**：选择一个主导原型，不要混用（李维斯的教训）
2. **一致性**：企业文化 = 品牌身份 = 营销传播
3. **深度**：不要停留在原型的表面层次，挖掘深层含义
   - 例："爱人"不只是性暗示，还包括亲密关系、自我认可、博爱
   - 例："探索者"不只是户外旅行，更包括寻找自我
4. **适配性**：原型必须与产品质量和功能匹配
5. **差异性**：如果竞品已占据某原型，考虑相邻原型或更高层次表达

### Step 3: 原型营销策略

**通用规则：**
- 原型意象通过**符号**（图像、颜色、故事）唤醒消费者潜意识
- 广告应该在**3秒内**通过原型画面抓住注意力
- 原型营销不是嫁接内涵，而是**发现**产品已有的内涵并放大

**按原型分类的营销要点：**

| 原型 | 营销关键词 | 视觉元素 | 注意事项 |
|------|-----------|---------|---------|
| 天真的人 | 纯净、简单、怀旧、重生 | 白色、柔和色彩、婴儿、自然 | 避免幼稚化；关注"天真"的更高层次——精神纯真 |
| 探索者 | 自由、发现、个性、真实 | 广阔天空、道路、大自然 | 注意探索者的孤独面；可安排同伴元素 |
| 智者 | 专业、思考、信息、客观 | 书本、实验室、素雅色调 | 不要强行推销；让消费者觉得自己是行家 |
| 英雄 | 勇气、挑战、胜利、坚持 | 运动场景、力量象征、强烈色彩 | 避免傲慢；关注英雄的更高层次——为和平而战 |
| 规则破坏者 | 叛逆、自由、革命、真实 | 暗色调、强烈对比、反叛形象 | 拿捏好度；大多数受众是好公民 |
| 魔法师 | 转变、奇迹、愿景、同步性 | 彩虹、星空、魔法物品 | 避免操纵感；强调由内而外的转变 |
| 普通人 | 归属、真实、平等、亲切 | 日常生活场景、温馨色调 | 绝不要装腔作势；接地气是核心 |
| 爱人 | 亲密、美丽、感官、承诺 | 浪漫场景、暖色调、精致细节 | 超越表面的性感；关注深层情感连接 |
| 小丑 | 快乐、幽默、当下、轻松 | 鲜艳色彩、卡通、搞笑元素 | 注意"不健康"产品的道德边界 |
| 照料者 | 保护、关爱、牺牲、信任 | 家庭场景、温暖光线、怀抱 | 区分"呵护"与"束缚"；关注照料者自身需求 |
| 创造者 | 创新、设计、表达、独特 | 艺术作品、工具、工作坊 | 让消费者参与创造过程 |
| 统治者 | 权力、地位、品质、责任 | 建筑、皇冠、深色质感 | 现代统治者要服务而非压迫 |

### Step 4: 含义层次管理

每个原型都有**4个含义层次**，营销人员需根据目标消费者的意识水平选择：

| 层次 | 说明 | 示例（以"爱人"为例） |
|------|------|---------------------|
| 呼唤 | 原型被唤醒的触发信号 | 热恋、吸引力 |
| 第一层 | 表面/初级表达 | 寻找性爱和浪漫 |
| 第二层 | 中间层次/成长 | 跟随幸福、许下承诺 |
| 第三层 | 最高/精神层次 | 精神层面的爱、自我认可 |
| 阴暗面 | 过度或扭曲的表现 | 滥交、痴迷、嫉妒 |

**关键洞察：** 后现代消费者意识觉醒，停留在第一层的广告越来越无效。应根据受众成熟度选择合适的层次。

## 失败模式（Failure Modes）

### FM-1: 原型漂移（Archetype Drift）
**症状**：品牌传播中主导原型信号模糊，消费者无法识别品牌"是谁"
**根因**：产品线扩张、换帅、并购后未重新锚定原型
**后果**：品牌资产稀释，营销投入回报递减
**修复**：回归 Step 1 重新审计，选择单一主导原型并在所有触点统一

### FM-2: 原型冲突（Archetype Collision）
**症状**：企业内部文化（如统治者）与外部传播（如普通人）不一致
**根因**：品牌身份由外部广告公司定义，未与企业DNA对齐
**后果**：员工不认同品牌价值观，消费者感受到"不真实"
**修复**：优先对齐内部文化，再向外传播——品牌由内而外建立

### FM-3: 阴暗面失控（Shadow Activation）
**症状**：原型的消极面被消费者或媒体放大（如耐克英雄→血汗工厂）
**根因**：只利用原型的光明面，未预判阴暗面风险
**后果**：公关危机，品牌信任崩塌
**修复**：每个原型策略必须包含阴暗面预案，定期做"最坏情况"推演

### FM-4: 嫁接失败（Grafted Identity）
**症状**：品牌强行采用与产品不匹配的原型，消费者不买账
**根因**：模仿成功品牌的原型，而非从自身产品DNA出发
**后果**：广告投入打水漂，品牌沦为笑柄
**修复**：回到产品本质——原型必须从产品已有内涵中"发现"，而非"嫁接"

### FM-5: 层次错配（Level Mismatch）
**症状**：高意识受众看到肤浅广告觉得被侮辱，低意识受众看到深层广告觉得看不懂
**根因**：未做受众意识水平调研，凭假设选层次
**后果**：目标人群流失
**修复**：做消费者意识水平测试，根据多数人所在层次选策略

## 反模式（Anti-Patterns）

### AP-1: "原型自助餐"
> 同时使用3个以上原型，声称"我们的品牌既英雄又探索者还魔法师"
**正确做法**：1个主导原型，最多1个不冲突的辅助原型

### AP-2: "原型贴标签"
> 把原型当作目标人群画像（"我们的目标是英雄型消费者"）
**正确做法**：原型是品牌身份，不是消费者画像。任何人都可能被任何原型吸引

### AP-3: "原型口号化"
> 只在slogan里体现原型，产品/服务/体验完全不匹配
**正确做法**：原型贯穿产品设计、定价、渠道、传播所有4P

### AP-4: "原型静态化"
> 选定原型后十年不变，不随市场/文化/消费者意识变化调整
**正确做法**：原型本身不变，但含义层次和表达方式需要进化

### AP-5: "原型武器化"
> 用原型操纵消费者无意识决策（如用恐惧驱动照料者原型卖保险）
**正确做法**：遵循伦理原则——原型是连接，不是操控

## 检查点（Checkpoints）

### 选定原型前 ✅
- [ ] 完成品牌原型识别问卷（Step 1全部5个问题）
- [ ] 竞品原型扫描完成，确认无直接冲突
- [ ] 原型与核心产品功能匹配（不是"想成为"而是"本来就是"）
- [ ] 创始人/领导团队认同该原型
- [ ] 已确认含义层次（选第几层？）

### 品牌身份构建中 ✅
- [ ] 企业文化、品牌身份、营销传播三层对齐
- [ ] 视觉系统与原型一致（颜色/字体/图像风格）
- [ ] 品牌声音/话术与原型一致
- [ ] 所有触点（产品/包装/网站/客服/社媒）原型信号统一
- [ ] 阴暗面风险已评估并有预案

### 营销执行后 ✅
- [ ] 广告3秒内能识别出原型
- [ ] 消费者调研确认感知到的原型 = 预期原型
- [ ] 无原型冲突信号（员工反馈/消费者反馈/媒体反馈）
- [ ] 品牌资产指标（认知度/联想度/忠诚度）正向变化
- [ ] 阴暗面未被激活

### 定期审计（每6-12个月）✅
- [ ] 品牌自认原型 vs 消费者感知原型是否一致？
- [ ] 是否出现原型漂移信号？
- [ ] 含义层次是否需要升级（受众意识水平变化）？
- [ ] 竞品原型格局是否变化？

## 常见陷阱

1. **原型混杂**：频繁切换原型 → 品牌失去内涵（李维斯案例）
2. **表面化**：只触及原型第一层 → 肤浅、被消费者识破
3. **嫁接内涵**：将不匹配的原型强加于产品 → 不可信
4. **忽视阴暗面**：不了解原型的消极面 → 陷入公关危机（耐克童工丑闻）
5. **刻板印象**：将原型等同于目标人群画像 → 丢失更大的市场
6. **文化误判**：忽视原型在不同文化中的表达差异 → 跨文化传播失败

---

## 伦理原则

1. **不欺骗**：原型内涵必须与产品质量一致
2. **不操纵**：不用原型意象控制消费者的无意识决策
3. **负责任**：考虑品牌内涵对社会集体意识的影响
4. **不滥用**：不将原型力量用于伤害弱势群体
5. **持续学习**：了解原型才能负责任地使用它们
