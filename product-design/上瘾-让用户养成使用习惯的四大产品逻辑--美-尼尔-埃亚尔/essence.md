# 上瘾：让用户养成使用习惯的四大产品逻辑

**作者：** [美] 尼尔·埃亚尔 (Nir Eyal)、瑞安·胡佛 (Ryan Hoover)
**译者：** 钟莉婷、杨晓红
**出版社：** 中信出版社，2017

---

## 核心模型：上瘾模型 (The Hook Model)

上瘾模型是一个四阶段循环框架，用于打造让用户形成使用习惯的产品：

```
触发 (Trigger) → 行动 (Action) → 多变的酬赏 (Variable Reward) → 投入 (Investment)
    ↑                                                                    |
    └──────────────────── 加载下一个触发 ←────────────────────────────────┘
```

通过反复循环，产品从外部触发逐步过渡到内部触发，用户从有意识使用转变为无意识习惯。

---

## 第一章：习惯的力量

### 核心概念

- **习惯**：在情境暗示下产生的无意识行为，人类近一半日常活动受习惯驱动
- **习惯的神经机制**：基底神经节存储无意识行为模式，大脑走捷径时形成习惯
- **习惯区间**：行为频率 × 可感知用途 → 进入习惯区间 → 变为默认行为
- **维生素 vs 止痛药**：习惯养成产品初期是"维生素"（锦上添花），形成习惯后变成"止痛药"（不可或缺）

### 习惯给企业带来的优势

| 优势 | 说明 |
|------|------|
| 提升用户终身价值 | 用户使用时间更长、频率更高，生命周期价值更高 |
| 提高价格灵活性 | 用户对价格敏感度降低（如免费游戏→付费道具） |
| 加快增长速度 | 忠实用户成为品牌推广者，缩短病毒循环周期 |
| 提高竞争优势 | 用户习惯一旦形成，竞品几乎无法撼动 |

### 关键洞见

- **Evernote微笑曲线**：初期使用量下滑，但一旦形成依赖，使用量大幅攀升。免费用户→付费用户的转化率随时间显著增长（第1个月0.5% → 第33个月11% → 第42个月26%）
- **QWERTY键盘效应**：即便有更好的替代品，改变用户习惯的代价太大
- **产品中的存储价值**（如Gmail邮件、Twitter粉丝、Instagram照片）增强了用户的路径依赖

## 第二章：触发 (Trigger)

### 外部触发

外部触发是促使用户采取行动的外在诱因，分为四类：

| 类型 | 说明 | 示例 |
|------|------|------|
| **付费型触发** | 广告、搜索引擎推广 | 拉新有效，但不可长期依赖 |
| **回馈型触发** | 媒体报道、应用商店推荐 | 昙花一现，需持续投入 |
| **人际型触发** | 口碑推荐、社交分享 | 最有效的病毒式增长引擎 |
| **自主型触发** | 应用图标、通知、邮件订阅 | 用户已注册/安装后出现，是习惯养成的关键 |

### 内部触发

内部触发与用户的思想、情感或常规活动关联，看不见摸不着但自动出现。

- **情绪是威力最强大的内部触发**，尤其是负面情绪：无聊、孤独、沮丧、困惑、焦虑
- 用户几乎在瞬间不自觉地采取行动来消除不适感
- 例：感到孤独→打开Facebook；感到无聊→浏览新闻；感到不确定→查邮件

### 如何发现用户的内部触发

**5问法**（源自丰田大野耐一）：连续问5个"为什么"，直到触及核心情绪。

```
用户为什么用电子邮件？→ 收发信息
为什么收发信息？→ 即时获取信息
为什么获取信息？→ 了解他人生活
为什么了解他人生活？→ 知道是否被需要
为什么在意被需要？→ 害怕被圈子抛弃 ← 核心情感触发
```

## 第三章：行动 (Action)

### 福格行为模型：B = MAT

```
行为(B) = 动机(M) × 能力(A) × 触发(T)
```

三者缺一不可，否则行为不会发生。

### 三大核心动机

| 动机 | 说明 |
|------|------|
| 追求快乐/逃避痛苦 | 最基础的生存驱动力 |
| 追求希望/逃避恐惧 | 对未来的期待与担忧 |
| 追求认同/逃避排斥 | 社交归属需求 |

### 简洁性六要素（影响能力的因素）

1. **时间** — 完成活动所需时间
2. **金钱** — 经济投入
3. **体力** — 消耗的体力
4. **脑力** — 消耗的认知资源
5. **社会偏差** — 他人对该活动的接受度
6. **非常规性** — 与常规活动的匹配程度

### 关键原则

> **先解决能力问题，再解决动机问题。** 简化操作比强化动机更有效、更经济。

- Twitter主页演变：2009年复杂文本 → 2012年极简界面+行为召唤，战略重心从"激发动机"转向"推动行动"
- Google搜索：干净界面、自动纠错、预测结果 → 不断降低使用门槛
- iPhone锁屏拍照：一键打开相机 → 简化到极致

### 启发法（认知偏差）

| 偏差 | 说明 | 应用 |
|------|------|------|
| **稀缺效应** | 物以稀为贵 | 亚马逊"仅剩14台"、限时抢购 |
| **环境效应** | 环境影响价值判断 | 同一啤酒标价不同→感知不同 |
| **锚定效应** | 第一个信息锚定后续判断 | 打折促销利用原价锚定 |
| **赠券效应** | 目标渐近效应 | LinkedIn进度条、星巴克集星卡 |

## 第四章：多变的酬赏 (Variable Reward)

### 神经科学基础

- **伏隔核**：大脑中与欲望相关的区域
- 关键发现：**期待酬赏时**大脑活动最强烈，而非获得酬赏时
- 多变性使大脑伏隔核更活跃，提升多巴胺分泌 → 催生渴望

### 三种酬赏类型

#### 1. 社交酬赏（部落酬赏）
来自与他人的互动关系：被接纳、被认同、受重视、受喜爱

- **Facebook**：点赞、评论、分享 → 社交认同
- **Stack Overflow**：威望值、勋章 → 同行认可
- **英雄联盟**：荣誉值系统 → 打击"捣蛋鬼"，奖励正向行为

#### 2. 猎物酬赏
对具体资源或信息的追逐

- **老虎机**：追逐金钱，不确定性驱动重复行为
- **Twitter信息流**：不断滚动寻找有趣内容
- **Pinterest**：露出半截的图片 → 激发好奇心继续滚动

#### 3. 自我酬赏
来自个体的操控感、成就感和终结感

- **视频游戏**：升级、获取特权、通关满足感
- **电子邮件**：清空收件箱的终结感（Mailbox"未读为零"）
- **Codecademy**：即时反馈的纠错功能 → 把学习变成挑战

### 关键原则

1. **多变的酬赏必须与用户需求匹配**
   - Mahalo（经济激励）失败 vs Quora（社交认同）成功 → 社交酬赏比经济酬赏更有号召力
   - "游戏化"不是万能药，必须基于用户真实需求

2. **保障用户自主权**
   - Quora强制显示浏览记录 → 用户反弹 → 功能撤回
   - "你有权接受，也有权拒绝" → 说"是"的可能性翻倍
   - 逆反心理：强制行为会引发排斥

3. **有限多变性 vs 无穷多变性**
   - 有限多变性：单人游戏、电视剧 → 使用后产生可预见性 → 热情消退
   - 无穷多变性：多人在线游戏、用户生成内容平台 → 长期吸引力
   - Zynga"农场小镇"的成功与衰落 → 生搬硬套导致多变性丧失

## 第五章：投入 (Investment)

### 投入阶段的特殊性

- 行动阶段与**即时满足**相关，投入阶段与**对未来酬赏的期待**有关
- 投入阶段**增加摩擦**（打破"一切越简单越好"的传统思维）
- **时机至关重要**：在用户享受酬赏之后提出投入要求

### 三大心理趋势

1. **我们总会高估自己的劳动成果**（宜家效应）
   - 自己折的纸鹤价值评估是旁人的5倍
   - 用户组装家具 → 对产品产生非理性喜爱

2. **我们总会尽力和过去的行为保持一致**
   - 先同意贴小标识 → 76%同意立大牌子（vs 17%直接要求）
   - 微小投入可能导致未来行为的巨大变化

3. **我们总会避免认知失调**
   - 酸葡萄心理：改变看法以消除矛盾
   - 游戏中花钱后 → "这样做肯定值得，因为我已经花了时间"

### 储存价值的五种形式

| 形式 | 示例 |
|------|------|
| **内容** | iTunes音乐收藏、Facebook照片/状态 → 数字化生活记录 |
| **数据资料** | LinkedIn简历、Mint财务数据 → 信息越多，服务越个性化 |
| **关注者** | Twitter粉丝数 → 社交资本，不可转换 |
| **信誉** | eBay/Airbnb评分 → 直接影响商业价值 |
| **技能** | Adobe Photoshop → 学习成本高，迁移成本更高 |

### 加载下一个触发

投入阶段的核心任务是**为下一次使用设置触发**：

- **Any.do**：连接日历 → 会议结束后自动推送通知（触发）
- **Tinder**：每次点击增加匹配概率 → 匹配成功即推送通知
- **Snapchat**：发送照片即隐含回应请求 → 双击即可回复
- **Pinterest**：点赞/评论/转贴 → 推送相关通知

## 第六章：道德操控

### 操控模式（四象限）

| | 自己会使用 | 自己不使用 |
|---|---|---|
| **提高生活质量** | ✅ 健康习惯推广者（最可能成功） | 兜售商 |
| **不提高生活质量** | 娱乐用户者 | ❌ 经销商（最不道德） |

### 四类设计者

1. **健康习惯推广者**：自己使用 + 相信提高生活质量 → 成功率最高
   - 必须真心相信产品价值，否则立刻停手
   - 例：杰克·哈里曼与肯尼亚农民同住，发现真实需求

2. **兜售商**：自己不使用 + 相信提高生活质量 → 脱离用户需求
   - 最常见的例子：广告领域

3. **娱乐用户者**：自己使用 + 不认为提高生活质量 → 产品缺乏持久力
   - 例："农场小镇"、"愤怒的小鸟" → 热潮过后被遗忘

4. **经销商**：自己不使用 + 不认为提高生活质量 → 剥削利用
   - 例：赌场、毒品

### 关键原则

- 对产品上瘾的用户仅约1%（病理性上瘾），但公司有道德义务识别并帮助这些用户
- 公司首次被允许使用数据标记过度使用者
- 设计习惯养成产品 ≠ 教人成瘾。习惯可以是积极的（刷牙、运动），成瘾则是自我毁灭

## 第七章：案例研究——《圣经》应用程序

YouVersion《圣经》应用：下载量超1亿次，每1.3秒一次新安装。

### 上瘾模型在《圣经》应用中的应用

| 阶段 | 实现方式 |
|------|----------|
| **触发** | 每日推送通知、400+读经计划、牧师指示、社区邮件、红色徽章提醒 |
| **行动** | 将经文分解为小片段、提供音频朗读（查尔顿·赫斯顿）、将有趣经文前置 |
| **酬赏** | "今日任务完成"画面、未知经文的神秘感、经句与个人奋斗的关联 |
| **投入** | 高亮显示、添加评注、创建书签、分享经文 → 储存个人礼拜史 |

### 关键经验

- 桌面版完全无法吸引用户 → 移动版成功
- 圣诞节推送通知测试：用户非但不反感，反而截图分享
- 每周日新下载量最高 → 口碑传播效应
- 传道者依赖应用 → 会众跟随

## 第八章：习惯测试与寻找机会

### 习惯测试三步法

1. **确定用户**：定义忠实用户的使用频率标准
2. **分析用户行为**：筛选数据，找到"习惯路径"（忠实用户的共同行为模式）
   - 例：Twitter发现新用户关注≥30人 → 留存率大幅提高
3. **改进产品**：推动新用户朝习惯路径前进

> **基准线**：如果不到5%的用户达到习惯标准，说明产品需要重新设计。

### 寻找机会的三个方向

1. **观察自己的行为**：保罗·格雷厄姆建议——"不要问'我应该解决什么问题'，要问'我希望其他人为我解决什么问题'"
   - 例：Buffer创始人从自己发推文的痛点出发

2. **关注新生行为**：利基市场行为可能扩展为主流
   - Facebook从哈佛花名册 → 全球社交网络
   - 布朗尼相机从"儿童玩具" → 普及摄影

3. **利用界面更改**：人机交互方式的改变带来新机会
   - Instagram：智能手机摄像头 + 滤镜 → 手机摄影革命
   - Pinterest：无限滚动界面 → 图片浏览新体验
