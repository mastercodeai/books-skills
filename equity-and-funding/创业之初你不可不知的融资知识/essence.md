> 作者：桂曙光（ReachVC创始人）

## 第一章：认识风险投资（VC）

### 1. VC是什么
- **定义**：VC（Venture Capital）是专业投资机构，将资金投入具有高成长潜力的初创企业，通过IPO或并购实现退出并获取回报。
- **组织形式**：有限合伙制（LP出资、GP管理），基金周期通常10年（投资期3-5年，退出期5-7年）。
- **资金来源**：养老金（~42%）、保险公司（~24%）、捐赠基金（~10%）、个人/家族等。
- **收益模式**：管理费（~2%/年）+ 收益分成（carry，通常20%），LP需先回收本金+门槛收益率后GP才分carry。

### 2. VC的商业模式
```
投资者 → LP出资 → VC基金(GP管理) → 投资创业公司 → 增值培育 → 退出(IPO/并购) → 收益分配
```
- VC每期基金投资10-30个项目，"广撒网"分散风险
- 典型回报：10个项目中4个失败、2个持平、3个获25X回报、1个获80X回报

### 3. VC筛选项目的标准
| 维度 | 核心关注 |
|------|----------|
| **团队** | 创业经验、互补性、执行力、诚信（最重要指标） |
| **商业模式** | 是否已验证、可扩展性、简单清晰 |
| **市场** | 足够大（>10亿）、快速增长、有壁垒 |
| **投资回报** | 预期10X+回报、明确退出路径 |
| **估值/价格** | 合理P/E（通常5-15X Forward P/E） |

### 4. VC的"十大英文关键词"
- Deal、Business Model、Cash Flow、Valuation、Option Pool
- Founder、P/E、Term Sheet、DD（Due Diligence）、IPO

### 5. VC的"十大谎言"
1. "我们保持联系" → 没兴趣
2. "我们内部讨论一下" → 通常不会
3. "我们对市场还不太了解" → 不想投
4. "我们正在寻找类似项目" → 你在帮他学习
5. "我们喜欢你的团队" → 但不会投
6. "我们投过很多成功项目" → 炫耀而非承诺
7. "我们可以提供很多增值服务" → 大部分只是说说
8. "这些条款是标准条款" → 偏向VC的"标准"
9. "估值太高对公司没好处" → 他想压价
10. "投资之后我们就是一家人了" → 等你不行时翻脸

## 第二章：谁需要/适合VC

### 1. VC不是唯一选择
| 融资方式 | 特点 | 适合阶段 |
|----------|------|----------|
| 银行贷款 | 需抵押、利息高、门槛高 | 有稳定现金流的企业 |
| 政府补贴 | 申请难、金额小、用途受限 | 特定行业/政策扶持 |
| **天使投资** | 个人资金、决策快、附加价值有限 | 种子期/早期 |
| **VC** | 专业机构、金额大、增值服务、股权稀释 | 成长期 |
| 可转债 | 先债后股、延迟估值、简化流程 | 早期/过渡期 |

### 2. VC投资的"500万~5000万"甜蜜区
- 太少（<100万）→ VC管理成本不划算
- 太多（>5000万）→ 需PE而非VC
- 项目需匹配VC的基金规模和投资阶段

### 3. VC给公司带来的变化
- ✅ 资金支持、行业资源、品牌背书、公司治理规范化
- ⚠️ 股权稀释、业绩压力、创始人可能被替换、决策受限

### 4. 什么时候该找VC
- 产品已有验证、商业模式清晰
- 需要资金加速规模化（而非维持生存）
- 资金需求量≥1.5~2年的运营预算

## 第三章：商业计划书

### 1. 核心内容（9大要素）
1. **公司概况与远景目标**
2. **团队介绍**（重点！CEO/CTO背景要具体有力）
3. **产品描述**（解决什么痛点）
4. **商业模式**（怎么赚钱，越简单越好）
5. **市场与营销策略**
6. **竞争分析**（不要说"没有竞争对手"）
7. **公司发展规划**
8. **财务预测**（3-5年，收入/成本/利润/现金流）
9. **资金需求与用途**

### 2. 三种文件形态
- **执行摘要**（1-2页Word）→ 5分钟看懂
- **演示文件**（20页PPT）→ 30分钟路演
- **详细计划书**（30-100页Word）→ VC深入研究

### 3. 投资人最想看到什么
- "你在解决什么问题" — 不是"你的产品有多好"
- 团队为什么能做成这件事
- 市场有多大、增长有多快
- 竞争壁垒是什么

### 4. 常见错误
- ❌ 写了100页的计划书 → VC没时间看
- ❌ 夸大市场规模（"万亿市场，我们只需1%"）
- ❌ 忽略竞争分析
- ❌ 财务预测不切实际
- ❌ 要求VC签NDA才给看计划书

## 第四章：与VC沟通的技巧

### 1. 找到合适的VC
- 通过**律师**（最佳渠道）、成功的创业者、行业活动
- 研究VC的投资方向、阶段、基金规模是否匹配
- **不要群发**商业计划书，要针对性沟通

### 2. 演示（Pitch）15条规则
1. 了解每个VC的偏好
2. PPT要**简洁有力**，一图胜千言
3. 展示**产品演示**（如果可能）
4. 关闭电脑/手机
5. 突破前5分钟抓住注意力
6. 提前到VC办公室踩点
7. 着装得体但不必过度
8. 展示团队实力
9. 带备用U盘
10. 财务模型格式要可打印
11. 不要依赖WebEx远程演示
12. 不要主动算IRR给VC看
13. 让VC看到实物产品/原型
14. 大胆但诚实
15. 不要开口就谈钱

### 3. 面对拒绝
- 99%的项目会被VC拒绝 — 这是正常的
- VC说"不"的常见原因：市场太小、团队不行、时机不对、与基金不匹配
- 不要纠缠，寻找下一个合适的VC

### 4. 尽职调查（DD）的双向性
- **VC查你**：团队背景、市场、财务、法律、技术
- **你查VC**：投过什么项目、对创始人态度、是否追加投资、增值服务是否真实

## 第五章：财务预测与估值

### 1. 估值方法
| 方法 | 适用场景 | 说明 |
|------|----------|------|
| **P/E法**（市盈率） | 盈利企业 | 公司价值 = P/E × 预期利润，VC用Forward P/E（通常5-15X） |
| **P/S法**（市销率） | 有收入但未盈利 | 类似P/E，用收入替代利润 |
| **可比交易法** | 有同类并购/融资 | 参考同行业最近的交易价格 |
| **DCF法** | 成熟企业 | 折现未来现金流，初创企业不确定性大 |
| **资产法** | 重资产企业 | 净资产价值，通常给出最低估值 |

### 2. 关键概念
- **Pre-money（投资前估值）**= Post-money - 投资额
- **Post-money（投资后估值）**= Pre-money + 投资额
- **Option Pool（期权池）**：通常15-20%，从Pre-money中扣除
- **Promote**：投资后创始人所持股份的总价值（比pre-money更关键）

### 3. 估值谈判技巧
- 不要只看pre-money，要看promote和退出时的权益价值
- 估值太高→后续融资困难（down round风险）
- 估值太低→创始人股权稀释过多
- **同时拿几个VC的Term Sheet**是最佳谈判策略

### 4. 财务预测的构建
1. 收入预测（产品单价 × 客户数量）
2. 成本预测（COGS + 运营费用）
3. 员工计划
4. 固定资产投资
5. 税务预测
6. 汇总为利润表和现金流预测
7. 关键假设要合理，要有依据

## 第六章：Term Sheet 核心条款解析

### 1. 清算优先权（Liquidation Preference）
- **最核心的条款**：决定公司被卖掉/清算时谁先拿钱
- 三种类型：

| 类型 | 说明 | 对创始人影响 |
|------|------|------------|
| **不参与分配（Non-participating）** | VC选择：拿回本金 或 转为普通股按比例分配 | 最友好 |
| **完全参与（Full-participating）** | 先拿本金 + 再按比例分剩余 | 最不友好（"双重分配"） |
| **有上限参与（Capped-participating）** | 先拿本金 + 按比例分，直到达到X倍上限 | 折中 |

- 谈判要点：争取1X不参与分配（Non-participating），这是最公平的条款

### 2. 反稀释保护（Anti-dilution）
- 防止后续融资价格更低时，早期投资人股权被稀释
- **完全棘轮（Full-ratchet）**：对创始人最不利，新价格直接替代旧价格
- **加权平均（Weighted Average）**：更常见，按新旧价格和数量加权计算
- 谈判要点：争取**广义加权平均**（Broad-based），避免完全棘轮

### 3. 董事会（Board of Directors）
- 控制董事会 = 控制公司
- A轮常见结构：2创始人 + 1投资人 = 3人董事会
- 谈判要点：确保创始人占多数席位，CEO席位归属要明确

### 4. 保护性条款（Protective Provisions）
- VC对公司重大事项的否决权
- 标准条款包括：修改章程、增发股份、并购、分红、清算等
- 谈判要点：减少条款数量、提高生效门槛（如2/3而非50%）

### 5. 股权归属（Vesting）
- **标准结构**：4年期，1年悬崖期（cliff），之后按月归属
- 创始人已工作的时间应获得相应归属
- **加速归属**：单触发（并购时自动加速25-50%）、双触发（被解雇+并购）

### 6. 股权赎回（Redemption）
- VC有权要求公司回购其股份（通常在投资后第5年）
- 实际执行率很低，但对VC是退出保底手段
- 谈判要点：推迟行权时间、延长分期支付

### 7. 强制随售权（Drag-Along）
- 多数股东同意出售公司时，可强制少数股东按同等条件出售
- 防止小股东"钉子户"阻碍交易
- 谈判要点：提高触发门槛（2/3而非50%）、设定最低价格保护

### 8. 竞业禁止（Non-Compete）
- 限制创始人/关键员工离开后从事同类业务
- 谈判要点：限制期限（1-2年）、范围要窄、需有补偿

### 9. 股利（Dividends）
- 通常为投资金额的5-15%/年
- 大多数VC不会实际要求分红，更多是增加退出时的保障
- 谈判要点：争取非累积（non-cumulative）、由董事会决定

### 10. 注册权（Registration Rights）
- VC要求公司在SEC注册其股份以便出售
- 包括：要求注册权、附带注册权、S-3注册权
- 关注IPO门槛条款（Qualified IPO定义）

## 第七章：拿到VC投资后的管理

### 1. VC的钱到账才是开始
- 投资人对你有了期待，业绩压力随之而来
- 定期与VC沟通，保持信息透明

### 2. 投后关系管理
- VC会要求董事会席位和定期报告
- 好的VC提供资源、人才推荐、战略建议
- 不好的VC只在公司出问题时才出现

### 3. 创始人与VC的关系
- 本质是**利益绑定**，不是"一家人"
- VC的目标是**投资回报最大化**，创始人的目标是**事业成功**
- 保持专业关系，重大决策提前沟通

## 跨境投资架构（VIE模式）

### 为什么需要VIE架构
- 中国法律限制外资进入特定行业（互联网、媒体、教育等）
- 通过**可变利益实体（VIE）**实现：境外公司通过协议控制境内运营实体
- 常见架构：开曼公司（上市主体）→ BVI公司 → 香港公司 → 境内WFOE → 协议控制境内运营公司

### VIE架构的好处
1. 灵活的资本进出
2. 法律环境友好
3. 公司治理清晰
4. 税务优化

### 注意事项
- 10号文限制了"红筹模式"
- 实际控制协议的法律风险
- 随时关注政策变化

## 核心公式速查

```
Post-money = Pre-money + 投资额
VC持股比例 = 投资额 / Post-money
创始人持股比例 = (Pre-money - Option Pool) / Post-money
每股价 = Pre-money / 总股数
反稀释新转股价 = (原有股×原价 + 新发股×新价) / (原有股 + 新发股)
```

## 关键数据点
- VC投资的典型退出周期：5-7年
- VC期望的IRR：20%+
- 签了Term Sheet但最终没拿到钱的比例：25-33%
- VC看1000个项目 → 约100个面试 → 约10个DD → 3-5个投资
- 标准Vesting：4年期，1年cliff
- 标准Option Pool：15-20%
- A轮VC通常要求：20-40%股权
