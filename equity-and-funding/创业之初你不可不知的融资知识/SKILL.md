# SKILL: 创业融资与VC知识

## Skill Description
创业融资知识体系，涵盖VC运作机制、Term Sheet谈判、估值方法、股权稀释、尽职调查、商业计划书撰写、路演技巧等。基于《创业之初你不可不知的融资知识》一书提炼。

## When to Use
- 创业者准备融资或正在与投资人谈判
- 需要理解Term Sheet条款含义及谈判策略
- 需要撰写商业计划书或准备路演
- 需要理解估值方法和股权结构设计
- 需要了解VIE架构和跨境融资结构
- 需要评估VC是否适合自己

## Core Knowledge

### VC运作机制
- VC采用有限合伙制：LP出资（养老金/保险/捐赠基金等），GP管理投资
- 基金周期10年：前3-5年投资，后5-7年退出
- 收益模式：2%管理费 + 20%收益分成(carry)
- VC需要10X+回报率来覆盖失败项目，只投有高增长潜力的公司
- 筛选比例：1000个项目 → 100个面谈 → 10个尽调 → 3-5个投资

### 估值与定价
- **Pre-money（投资前估值）** = Post-money - 投资额
- **Post-money（投资后估值）** = Pre-money + 投资额
- VC持股比例 = 投资额 / Post-money
- Option Pool（期权池）从Pre-money中扣除，通常15-20%
- 创始人应关注Promote（投资后创始人权益价值），而非单纯Pre-money
- 估值方法：P/E法（Forward P/E通常5-15X）、P/S法、可比交易法、DCF法
- 估值太高导致后续融资困难(down round)，太低导致股权过度稀释

### Term Sheet核心条款

**1. 清算优先权（Liquidation Preference）**
- 决定公司被卖掉时的分配顺序
- 三种类型：
  - Non-participating：VC选拿本金或转普通股（最友好）
  - Full-participating：先拿本金+再按比例分（"双重分配"，最不友好）
  - Capped-participating：参与分配但有上限
- 标准为1X Non-participating，谈判争取此项

**2. 反稀释保护（Anti-dilution）**
- Full-ratchet：新价格直接替代旧价格（对创始人极不利）
- Weighted Average：加权平均计算新转股价（更常见更公平）
- 争取Broad-based Weighted Average，避免Full-ratchet
- Pay-to-play条款：VC必须参与后续融资才享受反稀释保护

**3. 董事会（Board of Directors）**
- 控制董事会=控制公司
- A轮标准：2创始人+1投资人=3人
- 确保创始人占多数席位
- CEO席位归属要明确约定

**4. 保护性条款（Protective Provisions）**
- VC对重大事项的否决权（修改章程、增发、并购、分红、清算等）
- 谈判减少条款数量，提高生效门槛（2/3优于50%）

**5. 股权归属（Vesting）**
- 标准：4年期，1年悬崖期(cliff)，之后按月归属
- 创始人已工作时间应获得credit
- 加速归属：单触发（并购时25-50%）、双触发（被解雇+并购时50-100%）

**6. 股权赎回（Redemption）**
- 通常投资后第5年可要求公司回购，实际执行率很低
- 谈判推迟行权时间、延长分期支付

**7. 强制随售权（Drag-Along）**
- 多数股东同意出售时强制少数股东同价出售
- 提高触发门槛（2/3优于50%），设定最低价格保护

**8. 竞业禁止（Non-Compete）**
- 期限1-2年，范围要窄，需有补偿
- 注意VC自己的竞业条款（防止VC投竞争对手）

**9. 股利（Dividends）**
- 通常5-15%/年，实际很少支付
- 争取非累积(non-cumulative)、由董事会决定

**10. 注册权（Registration Rights）**
- 要求注册权、附带注册权、S-3注册权
- 关注IPO门槛条款定义

### 商业计划书要素
1. 执行摘要（1-2页）→ 能5分钟看懂
2. 演示文件（20页PPT）→ 30分钟路演
3. 详细计划书（30-100页）→ VC深入研究
4. 财务预测模型（Excel）→ 3-5年预测

**VC最想知道的三件事：**
- 你在解决什么问题？（不是你的产品有多好）
- 为什么你的团队能做成？
- 市场有多大、增长有多快？

### 融资流程
```
准备文件 → 找到合适的VC → 初次见面/演示 → VC有兴趣 → 
签Term Sheet → 尽职调查(DD) → 投资委员会审批 → 签正式协议 → 资金到账
```
- 整个流程通常需要3-6个月
- 签Term Sheet不等于拿到钱（25-33%最终未投资）
- 同时与多个VC沟通，制造竞争态势

### VIE架构（跨境融资）
- 境外上市主体（开曼）→ BVI → 香港 → 境内WFOE → 协议控制境内运营实体
- 用于外资受限行业（互联网、媒体、教育等）
- 通过协议而非股权实现控制
- 关注10号文限制和政策风险

### VC的增值服务（有限）
- 品牌背书、行业资源、人才推荐
- 公司治理规范化
- 但大部分VC的实际增值服务被高估
- 选VC时：品牌>增值服务承诺>价格

## Key Principles
1. **团队第一**：VC投的是人，不是创意
2. **同时拿多个Term Sheet**：制造竞争是最好的谈判策略
3. **关注Promote而非Pre-money**：投资后创始人的权益价值才是关键
4. **条款比价格更重要**：糟糕的条款可能让创始人失去公司控制权
5. **VC的钱是有条件的**：了解所有条款的含义再签字
6. **估值不是越高越好**：过高的估值会给后续融资埋雷
7. **时间是最大的敌人**：公司烧钱时VC会压价，尽早融资
8. **法律文件很重要**：Term Sheet的每个条款都有谈判空间，找好律师

## Failure Modes（失败模式）

### 💰 融资策略失败
| 失败模式 | 症状 | 根因 | 修复 |
|---------|------|------|------|
| 估值虚高陷阱 | A轮估值过高 → B轮无人接盘 → down round或倒闭 | 盲目追求高估值，未考虑后续轮次 | 以合理估值融资，预留上升空间 |
| 单一VC依赖 | 只拿一家TS → 无谈判筹码 → 被迫接受苛刻条款 | 未并行接触多家VC | 至少同时推进3-5家VC |
| 融资时机错误 | 账上只剩2个月才开始融资 → 被迫贱卖 | 未提前规划融资节奏 | 账上≥6个月时启动下一轮 |
| 赛道不匹配 | 找消费赛道VC聊SaaS项目 → 反复被拒 | 未研究VC投资方向 | 精准匹配VC的投资阶段+赛道偏好 |

### 📝 条款与法律失败
| 失败模式 | 症状 | 根因 | 修复 |
|---------|------|------|------|
| 控制权旁落 | VC占董事会多数 → 创始人被踢出 | 未重视董事会构成谈判 | A轮保持创始人2:1董事会多数 |
| 清算黑洞 | 公司被低价收购 → 创始人几乎分文不得 | 接受了Full-participating条款 | 坚持1X Non-participating |
| 反稀释绞杀 | B轮降价融资 → 创始人股份被大幅压缩 | 接受了Full-ratchet反稀释 | 争取Broad-based Weighted Average |
| Vesting陷阱 | 联合创始人1年后离开 → 带走大量股份 | Vesting无cliff或未设已工作credit | 4年Vesting+1年cliff+已有工龄credit |

### 🏗️ 架构与执行失败
| 失败模式 | 症状 | 根因 | 修复 |
|---------|------|------|------|
| VIE结构风险 | 政策突变 → 协议控制失效 → 境外上市受阻 | 过度依赖VIE架构忽视政策趋势 | 评估替代架构（如港股直接上市） |
| DD裸奔 | 尽调阶段发现财务混乱/法律瑕疵 → 交易流产 | 未提前做Pre-DD自查 | 融资前自行完成法律+财务+IP尽调清单 |
| BP失效 | 路演10家VC全部无下文 | BP未回答VC核心三问 | 重写BP：问题→团队→市场，而非产品功能堆砌 |

## Anti-Patterns（反模式）

### 条款谈判类
- ❌ **"估值越高越好"** → 过高估值导致后续融资down round，比低估值更致命
- ❌ **"先签了再说"** → Term Sheet一旦签字，核心条款很难再改
- ❌ **"条款都一样，看价格就行"** → Full-participating vs Non-participating可能差数倍回报
- ❌ **"VC推荐的律师一定中立"** → VC推荐的律师倾向保护VC利益
- ❌ **"Protective Provisions只是走形式"** → 这些否决权可实质性阻止公司运营决策

### 融资流程类
- ❌ **"有了好产品自然有人投"** → 融资是主动销售过程，需系统性BD
- ❌ **"TS等于钱到账"** → 25-33%签了TS但最终未投资
- ❌ **"融资期间不需要继续经营"** → DD期间公司表现直接影响最终估值和成交
- ❌ **"NDA签了再聊"** → 早期阶段要求VC签NDA会被视为不专业/不信任

### 股权结构类
- ❌ **"Option Pool越大越好"** → 大Pool从Pre-money扣，等于创始人单方面让利
- ❌ **"Vesting是VC用来控制我的"** → Vesting保护创始人，防止联合创始人中途离开带走股份
- ❌ **"回购条款不会执行"** → 虽然执行率低，但条款存在本身就是谈判筹码

## Checkpoints（检查清单）

### 📋 融资前检查点（Pre-Fundraise）
- [ ] 公司财务账目清晰，可出具正规报表
- [ ] 知识产权（IP）归属明确，已签署发明转让协议
- [ ] 核心团队已签署Vesting协议（4年+1年cliff）
- [ ] 公司股权结构清晰，无历史遗留代持问题
- [ ] 准备好三版BP：执行摘要、20页PPT、详细计划书
- [ ] 已筛选20-30家目标VC，按赛道/阶段/偏好排序
- [ ] 已选定独立律师（非VC推荐）

### 💰 估值检查点
- [ ] 使用至少2种估值方法交叉验证
- [ ] 计算Option Pool稀释后的实际持股比例
- [ ] 计算Promote（投资后创始人权益的美元价值）
- [ ] 验证估值在后续融资中是否可持续（避免down round）
- [ ] 了解同赛道同阶段公司的估值区间（可比交易）

### 📝 Term Sheet检查点
- [ ] 清算优先权：确认为1X Non-participating
- [ ] 反稀释：确认为Broad-based Weighted Average
- [ ] 董事会：确认创始人占多数席位
- [ ] Protective Provisions：确认数量合理、门槛≥2/3
- [ ] Vesting：确认已有工龄获得credit
- [ ] Drag-Along：确认触发门槛≥2/3、有最低价格保护
- [ ] Redemption：确认行权时间≥5年、分期支付
- [ ] 竞业禁止：确认范围窄、期限≤2年、有补偿

### 🔍 尽调阶段检查点
- [ ] 法律尽调：公司设立文件、股权历史、诉讼记录、合同清单
- [ ] 财务尽调：审计报表、税务记录、现金流预测模型
- [ ] 业务尽调：客户访谈、市场数据、竞品分析
- [ ] IP尽调：专利/商标注册状态、员工IP转让协议
- [ ] 人事尽调：核心团队背景、薪酬结构、期权池分配

### ✅ 交割后检查点（Post-Close）
- [ ] 资金确认到账，银行余额核对
- [ ] 工商变更完成（如有）
- [ ] 董事会首次会议纪要
- [ ] 投资人报告机制确认（月报/季报频率和内容）
- [ ] 下一轮融资时间节点规划（≥6个月提前量）

## Common Pitfalls
- ❌ 只关注估值高低，忽视条款细节
- ❌ 没有多个VC Term Sheet就签第一家的
- ❌ 不理解Option Pool从Pre-money扣除的影响
- ❌ 接受Full-participating Liquidation Preference
- ❌ 接受Full-ratchet Anti-dilution
- ❌ 董事会席位让给VC多数
- ❌ Vesting没有给创始人已有工作年限的credit
- ❌ 找律师时只看VC推荐的律师
- ❌ 把所有商业机密一次性告诉VC（注意NDA）
