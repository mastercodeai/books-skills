# 《给你一个亿你能干什么》第二部 — 精华笔记

## 概述
本书通过真实商业案例（RED摄影机、Apple、Titanic等）讲述创业融资、VC运作、企业估值与投资人关系的核心逻辑。

---

## 一、创业者的"野外生存法则"（SURVIVAL框架）

创业 = Mission Impossible，没有大后方、没有源源不断的产品、没有千万现金、没有安全网。

**五种公司形态：**
1. 没有收入的公司 — 停业只是时间问题
2. 有收入但远不够支出 — 每月都在烧钱
3. 支出远超收入 — 创业大会的谈资，股权价值为零
4. 盈利的公司 — 创业者的梦想，也可能是投资人的目标
5. 有高额盈利的公司 — 这才是真正的"好公司"

**Plan A + Plan B 双轨思维：**
| 维度 | Plan A（融资路线） | Plan B（自力更生） |
|------|-------------------|-------------------|
| 产品 | 先做商业模型，抓营收 | 先做产品，做实事的客户 |
| 团队 | 高薪精英 | 用期权激励 |
| 目标 | 不怕亏钱，要速度 | 先赚钱，现金为王 |
| CEO | 专业经理人 | 自己干 |

**SURVIVAL 七大生存法则：**
- **S**ize up — 评估环境、市场、挑战
- **U**ndue — 不要盲目追求速度
- **R**emember — 记住自己的定位
- **V**anquish — 用坚强意志克服恐惧
- **I**mprovise — 应变创新
- **V**alue — 创造价值
- **A**ct — 行动起来，在实践中学习
- **L**ive — 珍惜每一天

## 二、创业者的十个关键问题

### 1. VC的钱会被"管住"吗？
VC投资后获得股权，公司需要定期汇报、接受监督——这就是"管住"。但对比Apple案例：乔布斯找Mike Markkula投资25万美元获得1/3股权，8万是投资，17万是贷款——VC的钱是有代价的。

### 2. 股权分配
- 创始团队自己掏钱出力——值得大比例股权
- 不要为了吸引VC而过度稀释
- **关键原则：** 早期股权要集中，不要一上来就全分出去

### 3. CEO的薪酬
- 设立薪酬委员会，由董事会确定高管薪酬
- 创业者应学习乔布斯——不以薪酬为目标

### 4. 葡萄酒与第一次约会（估值比喻）
- 估值没有标准答案，是"谈"出来的
- 创业公司：先合理出价，再让投资人还价
- **核心原则：** 公司估值基于未来潜力，不是当前净资产

### 5. 投资人的退出
- 公司做大→卖给更大的公司
- IPO上市
- 团队回购
- 不要想着"投资人的钱怎么退出"，要想"怎么把公司做大"

## 三、VC的运作机制

### VC的本质
VC是**专业投资组织**，用LP（有限合伙人）的钱投资高成长创业公司，目标是获得10倍、50倍甚至100倍回报。

### VC的组织架构
| 职位 | 职责 |
|------|------|
| Managing Partner（管理合伙人） | 最终决策、基金大小 |
| Partner（合伙人） | 寻找项目、投资决策 |
| Venture Partner（风险合伙人） | 成功创业者转型，提供经验 |
| VP / Principle | 合伙人的副手，筛选项目 |
| Investment Manager | 执行尽调、谈判、签约 |
| Analyst | 行业研究、数据支持 |
| Associate（投资经理） | 入门级，项目初筛 |

### VC投资阶段与金额
| 阶段 | 金额（美元） | 说明 |
|------|-------------|------|
| A轮（种子期） | 10万~100万 | 验证想法 |
| B轮（成长期） | 100万~1000万 | 模式验证 |
| C轮（扩张期） | 1000万~5000万 | 规模化 |

**关键：** 不同阶段找不同的VC，不要一上来就找C轮基金要种子钱。

### VC的投资决策流程
1. 看项目→决定是否感兴趣
2. 尽职调查（Due Diligence）
3. 出具Term Sheet（投资条款清单）
4. 投资委员会（IC）审批
5. 签署正式投资协议
6. 资金到账

## 四、商业计划书的"21页"法则

**核心原则：** 商业计划书是CEO的"作战计划"，不是交给外人写的。

### 21页结构
| 页码 | 内容 | 要求 |
|------|------|------|
| 1 | 执行摘要 | 一页纸说清全部，15秒内抓住兴趣 |
| 2 | 产品/服务 | 做什么、怎么赚钱 |
| 3 | 市场 | 宏观+微观数据（iResearch等权威来源） |
| 4 | 竞争 | 竞争对手分析、差异化 |
| 5 | 团队 | 核心成员背景（不必名校，但要实在） |
| 6 | 里程碑 | 关键节点：产品上线、盈利时间 |
| 7 | 财务计划 | 3年预算，重点第一年（假设→利润表→现金流） |
| 8 | 股权结构 | 现有股东、投资后稀释 |
| 9 | 公司组织架构 | 注册地、部门设置 |
| 10 | 已有投资 | 自己投了多少钱（诚意的体现） |
| 11-21 | 补充材料 | 路线图、商业模式详解等 |

### 商业计划书的三大禁忌
1. **虚假包装** — 请设计师做的华丽PPT，VC一眼看穿
2. **过度冗长** — 80-100页没人看
3. **敷衍了事** — 几页纸的"想法"不是计划

### 如何判断VC是否感兴趣
- A步：发Executive Summary
- B步：VC回复要详细BP→发10页版
- C步：VC要求财务模型→发Excel
- D步：VC约面谈→准备PPT演示
- E步：如果VC不回复→放弃，不要反复催

## 五、创业公司估值

### 估值的本质
> "估值是双方博弈的结果，不是数学公式"

### 估值的六条原则
1. 没有标准答案
2. 钱的价值在不同时间点不同（风暴中和阳光下）
3. 估值是一场谈判，一个愿打一个愿挨
4. 公司规模越大，估值越有依据
5. 创业者希望越高越好
6. 投资人希望越低越好

### 早期公司估值方法
- **对标法：** 参考同行业同阶段被投公司的估值
- **反推法：** VC要求的回报倍数（10x-50x）→反推当前估值
- **里程碑法：** 达成什么目标值多少钱

### 融资轮次与估值递增
```
天使轮：50万（估值100万）→ 公司值150万
A轮：150万（估值400万）→ 公司值550万
B轮：300万（估值1000万）→ 公司值1300万
C轮：800万（估值3000万）→ 公司值3800万
D轮：2000万（估值8000万）→ 公司值1亿
```

### 估值陷阱
- A轮估值过高 → B轮无人接盘（"估值死亡"）
- 创始人与A轮投资人"双输"
- **原则：** 合理估值，给后续投资人留出空间

## 六、关键案例精华

### RED摄影机：Jim Jannard的二次创业
- Oakley太阳镜创始人，300美元起家
- 2006年发现数字电影摄影机市场空白
- 创立RED，推出4K数字摄影机，价格仅为SONY的1/10
- **核心启示：** 第二次创业比第一次更有优势——有资金、有经验、有人脉

### Steve Jobs与"海盗精神"
- 1985年被逐出Apple → 创立NeXT（失败）→ 收购Pixar（差点破产）
- Pixar靠《玩具总动员》一战成名
- 1997年回归Apple → iMac → iPod → iPhone
- **核心启示：** 创业需要"海盗精神"——Think Different，不怕失败

### James Cameron与《泰坦尼克号》
- 用650万美元拍《终结者》→ 票房7800万（投资回报10倍+）
- 《泰坦尼克号》预算7500万美元，严重超支
- Cameron放弃导演费+票房分成，与20世纪福克斯共担风险
- 最终全球票房超18亿美元
- **核心启示：** 
  - 花小钱也能做出好东西
  - 创业者要敢于"赌上自己"
  - 投资人看重的是"这个人"而不是"这个项目"

## 七、投资人关系核心要点

### 找VC的正确姿势
1. **研究VC的投资方向** — 不要盲目投递
2. **匹配阶段** — 种子期找天使/早期VC，不要找PE
3. **先建立关系** — 通过熟人推荐最有效
4. **不要群发** — VC圈子很小，同时发多家会"穿帮"

### Term Sheet（投资条款清单）
- 这是VC给出的"求婚戒指"，拿到Term Sheet才算真正开始
- **关键条款：** 估值、股权比例、董事会席位、反稀释、优先清算权
- 建议请律师审阅，但不要在拿到Term Sheet之前请律师

### VC的"拒绝话术"
| VC说的话 | 真实含义 |
|---------|---------|
| "很感兴趣，但我们最近太忙了" | 不感兴趣 |
| "方向偏了一些，我们再观察一段时间" | 不投 |
| "可以介绍给其他VC看看" | 不投 |
| 任何其他话 | 不投 |
