## Scene Routing — 按场景快速导航

### Scene 1: 股权架构设计 (Equity Structure Design)
**触发词**: 持股比例、股权架构、持股平台、创始团队分股、股权分配
**路由**: → Quick Reference 24条线 → 6大架构模型 → 持股架构决策树 → 分股14雷区自测
**关键决策**: 谁持股（自然人/合伙/控股公司）、比例如何定、平台如何搭

### Scene 2: 控制权机制 (Control Mechanisms)
**触发词**: 控制权、一票否决、一致行动人、AB股、章程条款、钱权分离
**路由**: → 7大控制工具 → 公司章程14项自治 → 控制工具组合拳
**关键决策**: 用哪种工具（合伙/金字塔/一致行动人/章程）、如何组合、法律效力边界

### Scene 3: 税务筹划 (Tax Planning)
**触发词**: 税收、税负、退出税、分红税、税收洼地、递延纳税、重组税务
**路由**: → 不同架构退出税负对比 → 税收洼地风险 → 企业重组特殊税务处理
**关键决策**: 持股架构的税务成本、退出路径选择、合规边界

### Scene 4: ESOP/股权激励 (Employee Stock Ownership Plan)
**触发词**: 股权激励、ESOP、员工持股、期权、限制性股票、兑现(vesting)
**路由**: → 股权兑现术 → 有限公司vs股份公司 → 持股平台选择 → 契约型架构
**关键决策**: 激励工具选择、持股平台搭建、退出/回购定价、写入章程

### Scene 5: 资本运作/上市重组 (Capital Operation & IPO)
**触发词**: IPO、上市、红筹、VIE、并购、重组、新三板、科创板
**路由**: → 海外股权架构 → 架构重组 → 子公司模型 → 签名案例索引
**关键决策**: 境内vs境外上市架构、VIE搭建/拆除、重组税务处理

### Scene 6: 家族传承/风险隔离 (Family Succession & Risk Isolation)
**触发词**: 家族传承、离婚、夫妻股权、信托、风险隔离、继承
**路由**: → 家族企业股权传承 → 控股公司架构 → 签名案例(龙湖/富贵鸟/昆仑万维)
**关键决策**: 家族信托架构、夫妻股权钱权分离、控股公司隔离层设计

## Quick Reference: 24 Core Shareholding Ratios

### 有限公司
- **34%** — 捣蛋线：7项重大事项否决权（章程修改、增减资、合并、分立、解散、变更形式）
- **51%** — 绝对控股线：控股股东（但上述7项仍不可独断）
- **67%** — 完美控制线：全部事项控制权
- **25%** — 外资待遇线：合资企业外资比例门槛
- **20%** — 重大影响线：权益法核算投资
- **10%** — 申请解散线/临时股东会召集线

### 股份公司/新三板/上市公司
- **1%** — 股东代表诉讼线（连续180日持股）
- **3%** — 股东提案资格线
- **5%** — 重要股东线（关联方认定、内幕知情人、减持预披露、举牌）
- **10%** — 激励总量/股东大会召集/股份回购上限
- **20%** — 科创板激励上限/详式权益报告
- **25%** — 首发公众股比线
- **30%** — 实际控制认定/要约收购触碰线

## 7 Control Tools (分股不分权)

Priority order by frequency of use:

1. **有限合伙企业** — GP掌控话语权，LP享受财产权。创始人用一人公司做GP实现风险隔离+灵活调整。最常用。
2. **金字塔架构** — 多层控股，以少量现金流权撬动大比例控制权。六大优点：杠杆、税筹、融资、人事、独立上市、市值管理。
3. **一致行动人** — 签协议抱团投票，写入章程增强效力。有期限和第三方约束力弱的局限。
4. **委托投票权** — 完全放弃表决权交受托人。适用于不愿绑定为一致行动人的财务投资人。
5. **公司章程控制** — 有限公司14项自治空间。关键：分红/表决权/转股限制/继承排除。
6. **优先股** — 放弃表决权换优先分配。适用范围窄。
7. **AB股** — 同股不同权。科创板/有限公司/境外上市可用。

## 6 Main Architecture Models

| 模型 | 适用对象 | 核心优势 | 核心劣势 |
|------|----------|----------|----------|
| 有限合伙架构 | 钱权分离高的创始人、短期套现投资人、员工持股 | 税收透明、钱权分离灵活 | 退出税率不确定、分红需纳税 |
| 自然人直接架构 | 套现财务投资人、创业期创始人 | 个税可预期(20%)、免增值税 | 不利控制权集中、无杠杆 |
| 控股公司架构 | 长期持股实业家、多元化集团、家族传承 | 杠杆+税筹+融资+人事 | 退出税负高达40%、缺乏灵活性 |
| 混合股权架构 | 明确境内上市规划公司 | 量体裁衣 | 需精准预判资本战略 |
| 海外股权架构 | 境外上市、海外并购、资产配置 | 融资/退出/税筹灵活 | 10号文审批、外汇登记、反避税 |
| 契约型架构 | 上市公司员工持股 | 杠杆收益、市值管理 | 仅限上市公司、IPO障碍 |

## 3 Subsidiary Models

| 模型 | 定义 | 典型案例 | 核心价值 |
|------|------|----------|----------|

## Tax Planning Essentials

### 不同持股架构退出税负对比
- **自然人直接持股转让上市公司股票**: 个税20%（限售股）、免增值税
- **自然人→合伙企业→上市公司**: 个税20%或5%-35%（各地不一）
- **自然人→控股公司→上市公司**: 企业所得税25% + 个税20% = 综合约40%
- **居民企业间分红**: 免征企业所得税（控股公司作为资金池免税再投资）

### 税收洼地（有限合伙注册地）
- 新疆/西藏/鹰潭等地：核定征收+财政返还
- **风险**: 政策不稳定、可能被中央叫停、仅适合短期套现

### 企业重组特殊税务处理
- 符合条件的分立/合并/股权收购可递延纳税（财税[2009]59号）
- 关键条件：合理商业目的、12个月内不改变实质经营、股权支付≥85%

## Vesting (股权兑现术)

中国本土化操作要点：
1. **写入公司章程**（否则回购缺法律依据）
2. **未兑现股份处理**：全体股东按比例回购 / 最大股东回购 / 公司减资
3. **回购价格**：原始出资额 / 出资额+利息 / 经审计净资产
4. **特殊情形**：有限公司股改为股份公司后可能失效

## Key Decision Frameworks

### 持股架构选择决策树
```
持股目的？
├── 短期套现 → 自然人直接持股（上市公司）/ 有限合伙+税收洼地（非上市）
├── 长期持股 → 控股公司架构
├── 钱权分离 → 有限合伙架构
├── 多元化集团 → 控股公司架构（业务板块管理公司）
└── 员工激励 → 有限合伙企业持股平台
```

### 有限公司 vs 股份公司选择
```
创业期 → 有限公司（闭合型、章程自治高、人合性强）
     ↓ 商业模式成熟 + 上市规划
股份公司 → 开放型、流动性强、章程约束严
```

### 分股14项雷区自测
1. 股东间均分股权？ 2. 创始股东超过3人？ 3. 给兼职大量股权？ 4. 仅提供一次性资源的股东？ 5. 短期资源承诺者大量股权？ 6. 小股东操盘？ 7. 只有大股东没有合伙人？ 8. 无退出机制？ 9. 外部投资人控股？ 10-14. 未预留投资/管理团队股权、未实缴过高、配偶股权未钱权分离、章程用范本？

→ 3项以上为"是"，建议深度测试股权结构。

## Failure Modes — 常见失败模式

### FM1: 均分股权僵局
- **现象**: 50:50或33:33:33均分，决策无法推进
- **后果**: 真功夫式内斗、公司瘫痪、最终双输
- **预防**: 创始人必须>51%，或至少1人>34%+一致行动人

### FM2: 控制权旁落
- **现象**: 多轮融资后创始人持股<34%，失去否决权
- **后果**: 被资本清洗出局（1号店、小马奔腾）
- **预防**: AB股/有限合伙/一致行动人+投票权委托组合锁定控制权

### FM3: 退出机制缺失
- **现象**: 股东离职/离婚/死亡，股权无回购条款
- **后果**: 股权被外部人继承或配偶分割，创始团队失控
- **预防**: 章程中写入强制回购条款+定价机制+vesting兑现

### FM4: 税收洼地政策突变
- **现象**: 依赖某地核定征收/财政返还做税筹
- **后果**: 政策收紧后税负骤增，已退出部分无法追溯调整
- **预防**: 税筹方案按"政策取消"做压力测试，预留缓冲

### FM5: 对赌连带配偶
- **现象**: 创始人签对赌协议未做夫妻隔离
- **后果**: 对赌失败债务成为夫妻共同债务（小马奔腾案）
- **预防**: 配偶签署知情确认书+家族信托隔离+婚前/婚内协议

### FM6: 红筹/VIE架构合规风险
- **现象**: VIE协议控制被认定无效或触发反避税
- **后果**: 境外上市架构崩塌、资金无法回流
- **预防**: 预留境内上市备选路径、合规性持续审计

## Anti-Patterns — 反模式清单

| # | 反模式 | 为什么有害 | 正确做法 |
|---|--------|-----------|----------|
| AP1 | 用工商模板章程 | 缺少自治条款，无法锁定控制权和退出机制 | 定制章程，写入14项自治条款 |
| AP2 | 口头承诺股权比例 | 无法律效力，事后反悔 | 签署书面协议+写入章程+工商登记 |
| AP3 | 给兼职/资源方大量股权 | 资源兑现不可控，股权已给出无法收回 | 用期权/分期兑现/业绩对赌替代 |
| AP4 | 自然人直接持股做激励平台 | 人数多则工商变更频繁，退出操作复杂 | 用有限合伙企业做持股平台 |
| AP5 | 离婚后才做股权隔离 | 已属共同财产，隔离失效 | 婚前/婚内协议+家族信托+控股公司 |
| AP6 | 上市前临时搭建复杂架构 | 时间不够、税务成本高、合规风险大 | 公司成立时就规划3-5年资本路径 |
| AP7 | 一致行动人协议不写入章程 | 仅合同约束，对第三方无对抗效力 | 写入公司章程并工商备案 |
| AP8 | 忽视34%否决权线 | 小股东意外获得否决权，绑架公司决策 | 精确计算每轮融资后各股东持股比例 |

## Checkpoints — 关键检查点

### CP1: 架构搭建期（公司成立/融资前）
- [ ] 创始人持股比例是否锁定在控制权安全线以上（>67%或>51%+控制工具）？
- [ ] 是否选择了合适的持股架构模型（对照6大模型表）？
- [ ] 公司章程是否定制（非模板）并写入关键自治条款？
- [ ] 股东配偶是否签署知情确认/股权隔离协议？
- [ ] 是否预设了股权兑现(vesting)条款并写入章程？

### CP2: 融资/稀释期（每轮融资前后）
- [ ] 本轮融资后创始人控制权是否仍安全（表决权计算，不只是持股比例）？
- [ ] 投资人协议中是否有对赌/回购/一票否决等特殊权利？
- [ ] 一致行动人/投票权委托是否已更新并写入章程？
- [ ] 是否预留了ESOP池（通常10%-20%）？

### CP3: 退出/套现期（股权转让/IPO前）
- [ ] 退出架构的综合税负是否已测算（含企业所得税+个人所得税）？
- [ ] 是否利用了居民企业间分红免税等政策？
- [ ] 税收洼地方案是否做了"政策取消"压力测试？
- [ ] 企业重组是否符合59号文递延纳税条件？

### CP4: 激励落地期（ESOP实施）
- [ ] 激励工具是否匹配（期权/限制性股票/分红权/持股平台）？
- [ ] 持股平台选择是否合理（有限合伙最常用，避免契约型阻碍IPO）？
- [ ] 退出/回购定价机制是否写入章程？
- [ ] 激励总量是否在法定上限内（上市公司≤10%，科创板≤20%）？

### CP5: 风险隔离期（家族传承/离婚/继承）
- [ ] 控股公司是否已设置风险隔离层？
- [ ] 家族信托是否在红筹架构中嵌入（H股无法嵌入需注意）？
- [ ] 夫妻股权是否实现钱权分离（配偶持财产权不持表决权）？
- [ ] 股权继承排除条款是否写入章程？
