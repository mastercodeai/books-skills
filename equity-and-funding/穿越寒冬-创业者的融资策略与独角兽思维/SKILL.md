## 技能一：融资决策判断

### 触发条件
- 创业者考虑是否融资、何时融资、融多少钱

### 决策流程
1. **先验证再融资**：商业模式未验证前不要接触VC，先花时间接触客户
2. **融资时机判断**：
   - ✅ 已有产品-市场匹配的证据
   - ✅ 有明确的指数级增长路径
   - ✅ 第三年预期年营收≥5000万美元
   - ❌ 仅有创意，无数据支撑
   - ❌ 商业模式未经验证
3. **过早融资的危险信号**：从探索模式→扩张模式，用资金掩盖问题

### 资金来源选择矩阵
| 阶段 | 来源 | 金额 | 特点 |
|------|------|------|------|
| 创意期 | 自有资金/兼职工作 | 少量 | 保留100%控制权 |
| 验证期 | 天使投资人 | 几千-数十万 | 经验+指导+人脉 |
| 种子期 | 种子基金 | 数十万-数百万 | 桥梁角色 |
| A轮前 | 超级天使/萌芽VC | 50万-数百万 | 填补种子与A轮空缺 |
| A轮 | 顶级VC | 数百万-数千万 | 需5亿美元+退出潜力 |

### ❌ 失败模式：融资决策
| 失败模式 | 症状 | 后果 | 纠正方案 |
|----------|------|------|----------|
| 为融资而融资 | 没想清楚钱用来干什么就去路演 | 烧完钱后更被动 | 先写清楚18个月资金使用计划 |
| 估值虚荣症 | 追求高估值忽略条款 | 清算优先权吃掉创始人收益 | 关注Post-money和实际稀释比例 |
| 过早稀释 | 天使轮出让30%+股权 | B轮后创始人失去控制权 | 种子轮出让10-15%，A轮15-20% |
| 只找顶级VC | 非独角兽赛道硬找红杉 | 浪费6个月时间 | 匹配阶段找对的基金，而非最大的 |

### 🚫 反模式：融资决策
- **"先融到钱再说"** → 用资金掩盖商业模式缺陷，烧完后问题更大
- **"估值越高越好"** → 高估值=高期望=下轮必须翻倍，否则down round
- **"投资人越多越好"** → 股权分散+决策混乱+后续轮协调困难
- **"不融资就死"** → 说明商业模式本身有问题，融资只是延缓死亡

### ⏸️ 决策检查点
```
□ 商业模式验证：是否有付费客户？复购率？
□ PMF信号：NPS>40？用户主动推荐？
□ 资金需求计算：18个月跑道×月烧钱率
□ 资金来源匹配：阶段、金额、行业是否对口
□ 稀释模拟：出让比例×估值=实际到手
□ 备选方案：不融资能否存活？能否bootstrapping？
```

## 技能二：投资人沟通与成交

### 30秒电梯路演模板
```
你好，我是[姓名]
我是[公司]的创始人
我们生产/制造[产品描述]
我们的客户群体是[目标客户]
我们想要解决的问题是[痛点]
我们的做法[解决方案]
我们的优势是[差异化]
```

### 融资路演11页PPT结构
1. 标题页（公司名+商标+联系方式）
2. 痛点（用图片说明）
3. 大愿景（如何改变行业）
4. 产品（功能+好处+可选30秒视频）
5. 目标市场（数字+客户数量+付费意愿）
6. 竞争（对手+独家秘诀+差异点）
7. 商业模式（赚钱方式+定价+利润率）
8. 收入预测（增长+烧钱率+里程碑）
9. 上升势头（客户+营收+合作伙伴+指标）
10. 团队（3-6人照片+头衔+成就，每人≤10词）
11. 答疑（已融金额+目标+估值+资金用途）

### 成交三振出局法
```
第一次会面 → 路演+展示投资意向书
第二天 → 简短跟进邮件
一周无回复 → 第二封跟进邮件
再一周 → 最后一封跟进邮件
仍无回复 → 出局，转向下一个投资人
```

### 投资人心理操控
- **利用贪婪**：让投资人相信这是独角兽级机会
- **制造恐惧**：暗示还有其他投资人竞争
- **设定截止日期**：人为紧迫感推动决策
- **隔离投资人**：不让投资人之间串通报价
- **永不绝望**：表现得你不需要他们的钱

### 让投资人说"不"的提问术
- "您为什么会对这笔交易产生兴趣？"
- "对于我们的业务您还有哪些顾虑？"
- "是什么让您无法向前推进这笔交易？"
- "您已经放弃这项投资了吗？"
- "如果我能改变公司中的某一样东西，会是什么？"

### ❌ 失败模式：投资人沟通
| 失败模式 | 症状 | 后果 | 纠正方案 |
|----------|------|------|----------|
| 数据造假 | 夸大营收/用户数 | 尽职调查暴露→信誉崩塌 | 只说可验证的数据，承认弱点 |
| 追踪狂魔 | 每天发邮件催投资人 | 引发反感，口碑变差 | 三振出局法，给空间但设底线 |
| 只讲故事不讲数 | PPT很美但没数据 | 专业投资人秒拒 | 每页至少1个可量化指标 |
| 饥不择食 | 接受任何人的钱 | 不合适的投资人+苛刻条款 | 做投资人背景调查，拒绝"有毒"的钱 |

### 🚫 反模式：投资人沟通
- **"先给NDA再讲"** → 大多数VC不签NDA，要求签=暴露不专业
- **"我们没有竞争对手"** → 投资人觉得你在撒谎或不了解市场
- **"保守估计"** → 投资人自动打3折，不如给激进+合理假设
- **"投后什么都可以谈"** → 条款要在term sheet阶段锁定，不要留模糊空间

### ⏸️ 决策检查点
```
□ 投资人清单：≥10个目标，按优先级排序
□ 路演材料：11页PPT+财务模型+产品demo
□ 背景调查：每个投资人的portfolio和决策风格
□ 跟进节奏：三振出局法时间线已设定
□ 竞争态势：是否有多个投资人同时在谈
□ 底线设定：最低估值/最高稀释/不可让步条款
```

## 技能三：估值与条款谈判

### 估值方法
- 没有公式，只有市场愿意支付的价格
- 参考Crunchbase、AngelList等平台的可比交易
- 在估值区间选取中位数
- 制造竞争：多个投资人同时竞标

### 律师选择要点
- 顶级律所：品牌效应+投资人推荐+可延期收费（5000-5万美元）
- 精品律所：更便宜+一揽子固定费率
- **关键品质**：给出清晰明确的商业建议，而非罗列选项
- **反应速度**：律师响应慢 = 交易可能失败
- 可用免费法律模板（Orrick工具包）省钱，但复杂交易必须请律师

### ❌ 失败模式：估值谈判
| 失败模式 | 症状 | 后果 | 纠正方案 |
|----------|------|------|----------|
| 忽视条款细节 | 只看估值不看清算优先权 | 退出时创始人一无所得 | 让律师逐条解读term sheet |
| 没有竞争报价 | 只和一个投资人谈 | 完全丧失议价权 | 同时启动3-5个投资人的流程 |
| 过度让步Anti-dilution | 接受full ratchet条款 | 后续融资创始人被严重稀释 | 坚持weighted average |
| 忽视Vesting条款 | 接受4年cliff+单边触发 | 离开公司股权归零 | 争取双触发/加速vesting |

### 🚫 反模式：估值谈判
- **"估值=我的身价"** → 估值是融资工具，不是个人价值证明
- **"条款以后可以改"** → 一旦签字很难改，必须一次到位
- **"投资人说这是标准条款"** → 没有"标准"，每条都可以谈

### ⏸️ 决策检查点
```
□ 可比交易：至少3个同行业同阶段案例
□ Term sheet比对：估值/清算优先/反稀释/Board seat
□ 律师审阅：已由独立律师review所有条款
□ 竞争报价：≥2个投资人同时在谈
□ 董事会构成：创始人是否保持多数席位
□ 退出场景模拟：IPO/并购/清盘各场景创始人收益
```

## 技能四：团队组建与管理

### 共同创始人筛选
- 问关键问题："你只是感兴趣还是会全身心投入？"
- 观察指标：愿意辞职吗？周末加班有意见吗？今年有长假计划吗？
- 不接受"只在晚上和周末工作"的人
- 共同创始人≥5%股权，平分股权也是好选择

### 人才识别6大品质
1. 自我提升能力
2. 天生好奇心
3. 热爱学习
4. 追求成功的内在驱动力
5. 责任感
6. 享受工作（工作=玩乐）

### 面试方法
- 至少面试10位合格候选人
- 组建招聘委员会独立评分（避免确认偏差）
- 评分标准：学习能力、领导力、谦逊、责任心、专业能力
- 背景调查：找不在推荐名单上的人
- 检查社交媒体帖文（寻找严重问题）

### 解雇12步
1. 记录每一件做错的事
2. 寻找替代者
3. 制订离职计划
4. 准备遣散费+授权协议
5. 上半周安排终止合约
6. 直截了当、诚实告知原因
7. 重点放在具体行为上
8. 言简意赅
9. 回答可能的问题
10. 不要闲谈，不给第二次机会
11. 当天收拾东西离开
12. 当天下午召开全体会议解释

### ❌ 失败模式：团队管理
| 失败模式 | 症状 | 后果 | 纠正方案 |
|----------|------|------|----------|
| 联合创始人不均权 | 50/50无决策者 | 僵局→公司瘫痪 | 设定tie-breaker机制或51/49 |
| 招聘凭感觉 | "聊得来就行" | 低绩效者拖垮团队 | 结构化面试+评分卡 |
| 不舍得开人 | 业绩差但人好 | 劣币驱逐良币 | 设30天PIP，到期不达标即走 |
| 过早招高管 | 天使轮就招VP级别 | 烧钱+文化冲突 | A轮前创始人亲自带核心职能 |

### 🚫 反模式：团队管理
- **"一起创业就是兄弟"** → 股权、职责、vesting必须白纸黑字
- **"期权画饼就行"** → 没有vesting和行权价的期权没有约束力
- **"先招便宜的再换"** → 解雇成本+团队动荡远超省下的薪资

### ⏸️ 决策检查点
```
□ 联合创始人协议：股权/职责/vesting/退出机制已书面化
□ 招聘标准：评分卡+面试流程+背景调查清单
□ Vesting schedule：4年+1年cliff，所有人包括创始人
□ 绩效机制：季度review+30天PIP流程
□ 文化文档：核心价值观+行为准则已写明
□ 关键人依赖：核心职能是否有backup人选
```

## 技能五：增长黑客与游击营销

### 增长黑客漏斗实验
| 环节 | 目标 | 案例 |
|------|------|------|
| 客户获取 | 最优化在线获取 | Airbnb黑客Craigslist引流 |
| 客户激活 | 与访问者互动 | Twitter推荐5个关注对象 |
| 收入增长 | 增加首次/重复销售 | Ticketmaster倒计时功能 |
| 产品推介 | 让客户分享 | Dropbox推荐送存储空间 |
| 客户留存 | 建立忠诚度 | YouTube连续播放功能 |

### 游击营销12条规则
1. 活动形式有创意
2. 问自己为什么这样做很重要
3. 挑战受众
4. 有趣和诙谐
5. 简化信息
6. 寻找合适人群
7. 尽早让有影响力的人参与
8. 方式离谱也无妨
9. 充分利用能走红的视频和照片
10. 允许参与并鼓励分享
11. 利用社交平台
12. 避免复制，创意最重要

### 集客营销操作清单
- [ ] 建立邮件列表（用白皮书/电子书/折扣券吸引注册）
- [ ] 创建用户画像
- [ ] 参与目标客户的活动
- [ ] 找到客户聚集的社区/平台
- [ ] 分析数据（用Hubspot等工具）
- [ ] 搜索引擎优化（每个关键词争取搜索顶部位置）
- [ ] 网站上设计小测验互动
- [ ] 鼓励员工/朋友/合作方分享内容
- [ ] 动态表单收集用户信息
- [ ] 实时沟通（聊天机器人/网络研讨会）
- [ ] 创建为主业务导流的站点
- [ ] 跨媒体转化（文章→播客→视频→信息图）
- [ ] FAQ页面结构优化
- [ ] 探索语音商务平台（Alexa/Siri/Assistant）

### ❌ 失败模式：增长
| 失败模式 | 症状 | 后果 | 纠正方案 |
|----------|------|------|----------|
| 虚荣指标迷恋 | 看注册数不看留存 | 以为增长实际流失 | 北极星指标=留存/活跃/付费 |
| 渠道单一化 | 只靠一个渠道获客 | 渠道衰减即死亡 | 同时测试3+渠道，分散风险 |
| 过早规模化 | PMF未确认就烧钱买量 | CAC远高于LTV | 先做到有机增长>50%再规模化 |
| 忽视激活 | 只关注注册不关注首次体验 | 漏斗底部漏水 | 优化onboarding，7日留存是关键 |

### 🚫 反模式：增长
- **"烧钱买增长"** → 没有PMF的烧钱=用钱买假增长
- **"复制竞品策略"** → 你的用户画像和渠道不同，照搬必败
- **"数据越多越好"** → 没有假设的A/B测试=随机试错

### ⏸️ 决策检查点
```
□ 北极星指标：是否定义了1个核心增长指标
□ 漏斗数据：获客→激活→留存→变现各环节转化率
□ 渠道测试：≥3个渠道在跑，ROI已计算
□ A/B实验：每周至少1个增长实验在运行
□ LTV/CAC比：是否>3x
□ 留存曲线：30日/90日留存是否趋于平稳
```

## 技能六：销售22条军规

核心原则精炼：
1. **建立360度客户模型**
2. **全面了解产品技术细节**
3. **理解对面的人想要什么**
4. **尽早发现真正的决策者**
5. **找到销售行家当合作伙伴**
6. **提出正确问题 > 滔滔不绝推销**
7. **帮助客户做正确决定，而非榨取价值**
8. **销售是创造性过程，不断迭代**
9. **不要咄咄逼人，让他们来找你**
10. **语速不要太慢，确保客户明白**
11. **用数据支撑说法**
12. **不确定的功能不承诺**
13. **拥抱拒绝，要求解释原因**
14. **不试图改变客户核心理念**
15. **让销售可感知（触摸/视频/演示）**
16. **销售未来的前景**
17. **精选潜在客户**
18. **建立过滤和排序流程**
19. **弄清楚底线**
20. **果断放手无法达成的交易**
21. **建立销售渠道跟踪系统**
22. **90%精力集中在少数决定性交易**

### ❌ 失败模式：销售
| 失败模式 | 症状 | 后果 | 纠正方案 |
|----------|------|------|----------|
| 创始人不卖 | 招了销售就不管了 | 失去客户洞察+销售周期拉长 | 创始人亲自卖前100单 |
| 承诺过度 | 答应不存在的功能 | 交付失望→流失 | 只承诺已有的+roadmap |
| 不敢要钱 | 免费试用/折扣太多 | 客户不珍惜+现金流断裂 | 免费≤30天，折扣需审批 |

### ⏸️ 决策检查点
```
□ 销售流程：从线索到成交的每一步是否标准化
□ CRM系统：是否在跟踪所有潜在客户
□ 成交率：当前pipeline成交率是多少
□ 创始人参与度：创始人是否仍在亲自见客户
□ 客户反馈闭环：销售反馈是否传递给产品团队
```

## 技能七：竞争壁垒构建

### 不公平优势清单（优先争取）
1. **网络效应**：用户越多价值越大
2. **数据获取**：专有数据创造持续优势
3. **技术领先**：好2-4倍才有效
4. **锁定客户**：高转换成本（如甲骨文）
5. **品牌**：品类代名词（如Google=搜索）
6. **生态系统/平台**：第三方+应用商店
7. **专利**：半导体/生物科技等核心技术
8. **行业标准**：成为标准后无人能敌
9. **规模经济**：规模越大成本越低
10. **进入市场速度**：第一个进入新市场

### ❌ 失败模式：竞争壁垒
| 失败模式 | 症状 | 后果 | 纠正方案 |
|----------|------|------|----------|
| 技术自嗨 | "我们技术最强"但没人用 | 产品≠壁垒 | 壁垒必须体现在用户价值上 |
| 忽视竞品 | "我们没有竞争对手" | 被偷袭/替代 | 每季度更新竞品分析 |
| 壁垒过窄 | 只靠专利/单一优势 | 对手绕过即崩塌 | 构建2-3层复合壁垒 |

### ⏸️ 决策检查点
```
□ 壁垒清单：当前有几层不公平优势
□ 网络效应：用户增长是否带来价值增长
□ 数据壁垒：是否有竞品无法获取的数据
□ 转换成本：客户迁移成本有多高
□ 竞品监控：最近一次竞品分析是什么时候
```

## 技能八：领导力提升

### 超级老板法则
- **提问而非告知**：用问题激活团队思考
- 克制告诉员工该怎么做的冲动
- 每次想命令时，改为提问
- 当团队快犯错时，问更多问题而非直接干预
- 最终目标：建立能独立运转的团队

### 董事会管理15原则
1. 坦诚（永远不要说谎）
2. 展示弱点（暴露错误建立信任）
3. 谨慎选择（不合适的人不要让其进董事会）
4. 定期沟通（不要等出问题才联系）
5. 共担责任（董事会是团队不是敌人）
6. 避免炒作陷阱（不夸大业绩）
7. 避免沙袋掩护（不故意设定低目标）
8. 避免隐瞒陷阱（不藏问题）
9. 避免数据过滤（不选择性呈现数据）
10. 避免防御心态（不把董事会拒之门外）
11. 避免阳奉阴违（说到做到）
12. 避免骡子陷阱（不无理由拒绝）
13. 避免霸凌（不胁迫董事会）
14. 避免政治（不玩弄权术）
15. 保持透明和一致

### ❌ 失败模式：领导力
| 失败模式 | 症状 | 后果 | 纠正方案 |
|----------|------|------|----------|
| 微管理 | 事事插手不让团队决策 | 人才流失+创始人成瓶颈 | 每周delegate 1件关键决策 |
| 报喜不报忧 | 董事会只听好消息 | 暴雷时失去信任 | 月度报告必须包含"3个最大风险" |
| 创始人独裁 | 不听团队意见一意孤行 | 团队士气崩塌 | 建立定期1:1和skip-level会议 |
| 不敢换人 | 明知高管不行但不动 | 整个部门绩效下降 | 用PIP + 明确时间线 |

### ⏸️ 决策检查点
```
□ 董事会沟通：最近一次董事会会议是什么时候
□ 信息透明度：团队是否知道公司真实状况
□ 决策委托：哪些决策已下放给团队
□ 1:1频率：和直属下属的1:1是否每两周一次
□ 反馈文化：团队是否敢对创始人说"不"
```

## 技能九：生存策略（现金快烧完时）

> **适用场景：跑道<6个月，收入不足以覆盖burn rate，融资窗口未到。**

### 紧急生存清单
```
1. 立即盘点：精确计算剩余跑道（现金÷月burn rate=剩余月数）
2. 砍成本：砍掉所有非核心支出（营销、外包、办公室降级）
3. 裁员决策：保留核心团队，裁掉非关键岗位（先裁后招）
4. 加速变现：推出付费计划/涨价/缩短账期
5. Bridge融资：找现有投资人或战略方做可转债bridge
6. 盈亏平衡：重新设计商业模式，目标=现金流为正
7. 并购选项：如果无法独立存活，主动寻找收购方
```

### ❌ 失败模式：生存策略
| 失败模式 | 症状 | 后果 | 纠正方案 |
|----------|------|------|----------|
| 鸵鸟心态 | 不看财务报表 | 突然发现没钱了 | 每周review现金流+runway |
| 赌博心态 | "再融一轮就好了" | 融不到→猝死 | 同时准备plan B（盈利路径） |
| 最后才裁员 | 拖到没钱才裁 | 赔偿金都付不起 | 跑道<6个月就必须行动 |
| 降价求生 | 疯狂打折拉收入 | 利润率崩塌+品牌贬值 | 砍产品线而非砍价格 |

### 🚫 反模式：生存策略
- **"再坚持一下就好了"** → 没有数据支撑的"坚持"=慢性自杀
- **"投资人会救我们的"** → 投资人倾向止损而非追加
- **"裁员会影响士气"** → 不裁员公司倒闭士气更差

### ⏸️ 生存检查点（每周review）
```
□ 现金余额：精确到元
□ 月burn rate：包含所有固定+变动成本
□ 剩余跑道：精确到月
□ 月收入趋势：是否在增长
□ 成本削减进度：已砍多少/计划砍多少
□ Bridge融资进展：是否有在谈的投资人
□ 盈亏平衡距离：还需多少收入才能打平
```
