# 十亿美金的教训 — 林军

## 核心主题
中国互联网史上10家投资超5000万美元却最终失败的公司，总结出价值十亿美金的三大教训。

---

## 三大核心教训

### 教训一：创始人未能完成从个人英雄到商业领袖的跨越

**三个关键词：**
1. **道义** — 创始人的道德瑕疵摧毁公司信誉（PPG李亮让父亲公司代理2亿广告预算；港湾李一男用父亲监工；ITAT编造品牌欺骗投资人）
2. **团队** — "梦之队"容易陷入群龙无首。51内部十几高管分三派每天吵架；8848两任CEO纠缠；分众管理团队全是被收购公司的原CEO
3. **决断** — 半数公司死于贸然挑战强敌（51挑战腾讯、港湾挑战华为、分众挑战央视），另一半死于IPO对赌压力

**关键判断信号：** 当只有创始人一人认为某决定正确、全公司都反对时，是公司大逆转的显著征兆。

### 教训二：热钱惹的祸

- **方兴东反思博客网**："钱太多了，让自己乱了分寸"
- **刘中青反思ITAT**："如果当初只给ITAT 500万美金，也许就成了"
- **热钱四大症状**：①丧失欲望 ②战线过长 ③人浮于事 ④迷途不知返
- **董事会失控**：要么CEO凌驾董事会之上（PPG/ITAT），要么CEO无法控制董事会（联众/8848），最终公司失控

### 教训三：没有坚守用户价值第一

- 10个案例商业模式都没问题，但在细节上做错，对用户价值尊重不够
- **盛大盒子**：三网融合是趋势，但用户不需要——"激发需求"本末倒置
- **ITAT**：好模式执行走样，选址不考究，没产品管控部门
- **8848**：只想着迎合资本市场，没有花工夫服务用户
- **抄袭悖论**：抄袭者无法超越只能跟随，真正重要的是用户价值挖掘

## 十大案例摘要

### 1. 51.com — 内讧的代价
- 庞升东卖13套房创业，定位三四线城市网吧用户
- 核心矛盾：庞升东（大老板）vs 张剑福（产品负责人）认知错位
- 巨人网络5100万美元入股后，庞升东松懈，忙于打高尔夫
- 腾讯全力反击：20人反51办，Qzone重构后超越51
- **教训**：创始团队角色定位不清，被收购对象成为联合创始人是大忌

### 2. 3721 — 借力未遂
- 周鸿祎发明"网络实名"，用插件弹窗+电信合作+自建渠道三板斧崛起
- 与CNNIC/百度三方恶战，在用户电脑里互删软件
- 卖给雅虎1.2亿美金，错失搜索市场（百度今日市值300亿美金）
- **教训**：不自信+急于求成+虚荣心膨胀 → 以1.2亿卖掉本值百亿的公司

### 3. ITAT — 细节的错失
- "三零模式"（零货款、零场租、零库存）看起来很美
- 蓝山中国5000万美元+摩根士丹利等热钱涌入 → 疯狂扩张到780家店
- 选址无标准、无产品管控部门、投行检查时集体作假
- **教训**：好商业模式+执行走样+热钱催熟 = 灾难

### 4. PPG — 断裂的链条
- "服装业戴尔"模式，用DELL直销卖衬衫，99元震撼市场
- 借VC和上下游的钱砸广告（2007年2.3亿广告费），拖款经营
- 竞争对手宝鸟、凡客杀入 → 价格战 → 质量下降 → 库存门 → 广告门
- 创始人李亮携款"去美国"失踪，公司崩溃
- **教训**：把上下游的钱当自己的钱花，供应链一旦断裂全盘崩溃

### 5. 分众 — 热线的烦恼
- 江南春发明楼宇液晶广告，连续收购聚众/框架/好耶等
- 管理团队全是收购对象的原CEO，派系林立
- "3·15"被央视曝光后48小时内无人做危机公关（两个CEO互相推诿）
- 最终将分众卖给新浪，又流产
- **教训**：只收购不整合，没有伟大公司是靠买出来的

### 6. 港湾 — 资本的蜂拥与溃逃
- 李一男离开华为创办港湾，上百名华为骨干追随
- 进入光通信市场彻底激怒华为 → 华为成立"打港办"（年经费4亿）
- 内部李一男vs彭松（销售团队）不和，两次上市均被狙击
- 最终被华为收购，李一男给了一间透明办公室（被围观）
- **教训**：挑战老东家要选好时机和边界，内部不和是致命伤

### 7. 联想FM365 — 一捅即破的饼
- 联想借互联网概念从香港资本市场圈走28亿港币
- "三点一线"战略（ISP+门户+垂直网站）看似完美，执行中三点渐行渐远
- 与AOL合资，AOL要求的许可费高到永远无法盈利
- 最终连FM365域名都丢了
- **教训**：用硬件思维做互联网，始乱必然终弃

### 8. 联众 — 决策无主
- 鲍岳桥三人创办联众，500万卖给海虹（中公网）
- 海虹只把联众当资本市场道具，不投入不决策
- 驱逐了最优秀CEO谢文，错失与腾讯合作和网游市场
- NHN入股后6年换5次策略，中韩股东持续内斗
- **教训**：创始人股份过少+大股东只想利用公司炒股价 = 决策永远无主

### 9. 盛大盒子 — 陈天桥的自吟自唱
- 陈天桥力排众议推"家庭娱乐战略"，上千名工程师做盒子
- 天时（三网融合未到）、地利（上海不是硬件基地）、人和（全员反对）皆失
- 义乌试销一个月卖20台，广电总局一纸叫停
- **教训**：雄心凌驾于团队和市场之上，一意孤行的代价是数十亿美金

### 10. 8848 — 资本的冒进
- 王峻涛创办8848，中国电子商务先驱
- 57个股东、11席董事会、等待上市批文错过时间窗口
- 被迫转型B2B（迎合资本），B2C业务被万泉河收购后烂尾
- **教训**：股权过度分散+各方资本利益不一致 = 没有人真正关心公司死活

## 反复出现的失败模式

| 模式 | 涉及案例 |
|------|----------|
| 创始人失去控制权 | 8848、联众、FM365 |
| 热钱催熟扩张失控 | ITAT、PPG、分众 |
| 挑战强敌过早暴露 | 51(腾讯)、港湾(华为)、分众(央视) |
| 团队内斗/派系林立 | 51、港湾、联众、分众 |
| 忽视用户体验/产品 | ITAT、PPG、8848、盛大盒子 |
| IPO对赌压力下动作变形 | ITAT、PPG、港湾、8848 |
| 只收购不整合 | 分众 |
| 资本道具命运 | 联众、FM365、8848 |
