# 经营者养成笔记 — 精华提炼

> **作者**：柳井正（迅销集团/优衣库创始人、董事长兼CEO）
> **原书**：KEIEISHA NI NARUTAME NO NOTE（2015年日本出版，2017年中文版）
> **核心命题**：经营者 = 取得成果的人。不是天才或超人，而是通过正确行动兑现承诺的人。

---

## 一、经营者的定义

**"所谓经营者，一言以蔽之，就是取得成果的人。"**

- **成果 = 承诺**：对顾客、社会、股市、员工做出承诺并兑现
- 成果不仅是业绩数字，也包括定性承诺（如"培养200名经营人才"、"创造前所未有的服装"）
- **赚钱很重要** ≠ "只要能赚钱就行"。正当赚钱 = 在兑现承诺的基础上赚钱
- 公司的本质是为**顾客**而存在，而非为股东或员工

---

## 二、经营者必须具备的四种能力（管理四象限）

| 能力 | 对应身份 | 核心要求 |
|------|---------|---------|
| 变革的能力 | 创新者 | 不断革新，创造顾客 |
| 赚钱的能力 | 生意人 | 将变革转化为利润 |
| 建设团队的能力 | 领导者 | 通过团队协作取得大成果 |
| 追求理想的能力 | 为使命而生的人 | 实现企业存在的意义 |

---

## 三、第一章 — 变革的能力（创新者）

### 1. 抱持高远的目标
- 目标必须是"用常识无法想象"的目标
- 迅销做法：销售额每到一个台阶就定3-5倍的新目标（100亿→300亿→1000亿→3000亿→1万亿→5万亿）
- **原理**：高远目标迫使放弃"延续现有做法"的思维，催生真正的革新
- 吉宁名言："从终点开始吧。因为只要你设定了终点，'为了获得成功该做哪些事情'就变得一目了然了。"

### 2. 质疑常识，不受常识束缚
- 妨碍公司成长的最大敌人是"常识"
- **"行业是过去，顾客是未来，不要过分关注竞争对手，而要全心全意以顾客为中心展开经营"**
- 案例：7-11的"夏季关东煮"和"冬季冰激凌"——空调改变了饮食需求
- 区分"危机感"与"不安"：不安大多没有确切证据，要勇于尝试

### 3. 树立高标准，不放松，不放弃
- 质量标准 = "是否真正有益于顾客"
- 自己制定的标准没有意义，必须以**全世界最高标准**来衡量
- HEATTECH案例：从150万件（2003）→8000万件（2010），靠的是年复一年追求更高品质
- IBM沃森："追求完美即便遭遇失败也好过因以不完美为目标而收获的成功"
- **高标准的实现 = 绝对优势**（谷歌、苹果、迪士尼的标准已成为顾客常识）

### 4. 不畏风险，勇于尝试，敢于失败
- **"追求稳定的公司不可能获得稳步发展"**
- 铁律："没有风险就没有利润，风险总是伴随着利润"
- 风险估测 ≠ 停止思考。要冷静分析"风险在哪里"、"有多大"
- 案例：优衣库原宿店决定完全销售本公司商品——放弃进口品牌短期利润，换取自主品牌长远利益
- **一旦决定冒风险，就必须全力以赴做到底**
- 为失败负责 = 从失败中总结经验教训并取得成果，而非中途道歉辞职

### 5. 严格要求，询问本质问题
- 不提问题 → 工作变成"机械的操作"
- 学习丰田的"五个为什么"
- 经营者的职责：挖掘员工潜力，拓展可能性
- **不做善解人意的上司**——善解人意剥夺了部下的成长机会
- 严格要求的前提："你一定可以做好！"+"责任全在上司，功劳全归部下"

### 6. 自问自答
- 自问自答 = 围绕过去、市场、未来向自己提问
- 最大忌讳："自己做得很好"
- **危机感 vs 不安**：危机感是客观评价后的持续努力，不安是无确切证据的心理状态
- 诀窍："站在顾客的立场，以最挑剔的眼光审视自己"
- 好创意来自长期自问自答的积累，而非灵光一闪

### 7. 天外有天，不断学习
- 两个心理准备："人外有人"、"世上没有从未发生过的事"
- 经营者的学习：学以致用 → 结合自身情况思考 → 勇于尝试
- **自身能力的提高是获取真正有价值信息的必要条件**
- 阅读经营者写的书优于学者写的书
- 坚持5年每天看书、看业界杂志、与各类人交流

## 四、第二章 — 赚钱的能力（生意人）

### 1. 从心底希望顾客高兴
- **"一切以顾客为中心"不是口号，必须在所有经营环节彻底贯彻**
- 三个要点赢得顾客满意：
  1. 让顾客感到惊喜（超出想象）
  2. 以高于顾客期待的水准满足需求（顾客追求的是尚未见过的东西）
  3. 生产自己真正认为优质的商品
- 热卖商品共同点：前所未有 + 商家充满信心

### 2. 日复一日，完成好必做的工作
- 赚钱能力 = 脚踏实地做好本职工作
- 经营 = 每天认真完成工作 → 检查 → 思考改进 → 修改计划 → 周而复始
- **未来存在于每天充实的工作中**
- 不起眼的工作中蕴藏着赚钱之道
- 能否做到 = 习惯问题，不是能力问题

### 3. 迅速实行
- 速度两层含义：抢占先机 + 快速完成
- 警惕"报告文化"：报告比行动多，计划占九分行动占一分
- **理想时间分配：行动占九分，计划占一分**
- 永守重信："马上做，必须做，直到做好"
- 本田宗一郎："只有善于利用宝贵时间的人才能成为成功的人"

### 4. 现场、现物、现实
- **"在卖场一定能找到答案"**
- 经营不能纸上谈兵，不能仅凭账簿数值下达指示
- 案例：摇粒绒在中国工厂——电话下达指示无人执行，亲临现场才找到真正原因
- 数值异常时 → 亲临现场确认 → 很快能找到问题根源

### 5. 集中解决问题
- 最理想的经营：仅凭某一种商品就能获得极高销售业绩
- 苹果案例：所有拳头产品一张小桌子就够了
- 乔布斯给耐克CEO的建议："舍弃平庸商品，将经营重点集中在最好的商品上"
- 诀窍：自问"如果不做这件事会怎样？"
- **该节约的地方彻底节约，该投入的地方集中资金投入**
- 与合作伙伴也应集中——只与使命感相同的公司结盟

### 6. 与矛盾做斗争
- 经营就是与矛盾做斗争
- 典型矛盾：提升质量 + 降低成本（看似不可能，但专业人士的工作就是解决它）
- 矛盾越大，解决后顾客的笑容越灿烂，利润越高
- **不能用治标不治本的方法**——找到根本原因再解决

### 7. 做好准备，执着于成果而非计划
- 计划的真正意义：深入思考，直至成功的画面浮现于脑海
- **执着的是"成果"而非"计划"**——情况变了就要毫不犹豫放弃原计划
- 优秀的MD：没按原计划行事，但最终成果与目标一致

## 五、第三章 — 建设团队的能力（领导者）

### 1. 建立信赖关系
- 团队 ≠ 一群人的集合，而是朝着共同目标奋斗的状态
- **信赖的基石：言行一致、始终如一**
- 承诺了就要遵守；信念和价值观不动摇，具体方法可与时俱进

### 2. 全心全意、全身心面对部下
- 百分之百投入——不花100%精力绝无可能建立真正的互信
- 站在部下立场感同身受地倾听
- **时而做"魔鬼"，时而做"菩萨"**：
  - 魔鬼：严格要求直至能胜任工作
  - 菩萨：表扬认可 + 关心健康和家庭

### 3. 共享目标，责任到人
- 目标共享 = 反复传达直到成员能用自己的语言充满热情地描述
- 韦尔奇："一天当中，我会一遍又一遍地强调公司的目标"
- 责任到人 = "一人一责"，不能由"团队"来负责
- 让本人思考如何工作 → 产生责任感 → 责任感产生原动力

### 4. 交托工作并予以评价
- 好公司 = 所有员工把工作当成自己的事
- **工作成果 = 能力 × 干劲**
- 松下幸之助："放手，又不能完全放手"
- 放手后必须准确传达评价——做得好要表扬，做错了要指出

### 5. 提出期望，发挥部下长处
- 委以重任 + 真心期待（"你一定行"）
- 关键：让其发挥优势（强项），而非纠正缺点
- 德鲁克："所有人都是通过自己的强项，而非弱项来获得报酬的"
- 团队互补：缺点大家互补，优势最大限度发挥

### 6. 积极肯定多样性
- 人各不相同——即使没有全球化，每个职场也存在多样性
- 好创意无关国籍、人种、性别、职位高低
- 方法和智慧可以多样，但公司使命和原理原则不可动摇
- 全球化经营：不要照本宣科用旧模式判断，遵循现场、现物、现实

### 7. 抱持最强烈的取胜欲望
- 领导者取胜欲望必须比任何人都强烈
- 让成员做主角、做英雄
- 领导者持续挑战的三个条件：
  1. 对自己寄予期望
  2. 自我完善（冷静客观，谦虚学习）
  3. 自我管理（自律 + 健康管理）

## 六、第四章 — 追求理想的能力（为使命而生的人）

### 1. 使命感的核心地位
- 使命感 = 公司存在的理由，一切经营活动的核心
- 迅销使命："真正的优质服装，让世界上所有人享受身着称心服装的喜悦"
- 迅销宣言："改变服装，改变常识，改变世界"
- **公司是为顾客而存在的**——"为股东存在"是本末倒置

### 2. 使命感不可或缺
- 只有为社会做贡献的企业才能生存
- **"如果只追求金钱，金钱反倒会离你而去。如果追求使命的完成，金钱则会自己来找你。"**

### 3. 使命感赋予的八样东西
1. 责任感
2. 职业道德
3. 主观能动性
4. 成为"不气馁的人"
5. 为团队成员指引方向
6. 吸引优秀人才
7. 让人清楚了解公司
8. 提供判断标准

### 4. 与使命感的绊脚石做斗争
五大征兆：
1. 以自我为中心、高高在上的工作态度
2. 模仿性的工作，缺乏独创
3. 机械的、生搬硬套的思考模式
4. 官僚主义泛滥
5. 评价标准宽松，非实力主义的人事安排

### 5. 面对危机时的必备行为
- 经营者站在最前沿
- 正视现实，即使残酷也要接受
- 信息公开——最高层亲自说明
- 加上一句："虽然现在形势严峻，但是将来我们会这样做"

## 七、柳井正的经营哲学精华

### 顶层设计理念
- **目标倒推法**：从终点开始，为实现终点反推该做什么
- **3-5倍增长法**：每次飞跃期设定当时3-5倍的长期目标
- **使命驱动**：使命感是一切决策的判断标准和行动准则

### 激励机制
- **责任全在上司，功劳全归部下**
- 委以重任 + 真心期待 + 放手让做 + 认真评价
- 让成员把工作当成自己的事 → 产生责任感和原动力
- 时而做"魔鬼"（严格要求），时而做"菩萨"（表扬关怀）

### 经营者自我修炼
- 自问自答：永远不认为自己做得很好
- 危机感：时刻意识到走在悬崖边缘
- 学习：每天看书 + 与杰出人才交流 + 学以致用
- 自律：健康管理 + 不走旁门左道
- 摆出自信的样子 → 逐渐习惯自信的自己
