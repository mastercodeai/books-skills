# 一胜九败：柳井正的创业人生与商业哲学

> 柳井正（Yanai Tadashi），优衣库（UNIQLO）创始人，迅销集团（Fast Retailing）董事长兼CEO。连续多年蝉联日本首富，以"一胜九败"的经营哲学带领优衣库从广岛一家小西装店成长为全球第五大服装零售商。

---

## 核心理念：一胜九败

柳井正的"一胜九败"意味着：**胜利的机会只有1%，但不怕失败，持续尝试，就像棒球中的盗垒——只要不怕被触杀，持续地去盗垒，一定会越来越好。**

关键认知：
- **错九次，就有九次经验**——"尝试新事物不可能不犯错，错误是为成功准备的教训"
- **十战十胜是最可怕的**——连胜让人迷失自我，失败才能找到正确方向
- **失败要有失败的价值**——不在同一个地方跌倒，才是最明智的选择
- **极速撤退**——一旦失败，立即转移目标，撤退速度最关键，早一步撤出就少一点损失

止损的两个标准：
1. 投资已经不可能回收，或看不到回收的可能性
2. 事业只是延续生存，但不能再扩大生产

## 一、SPA模式：从设计到零售的一体化经营

柳井正最核心的商业创新是**SPA（Specialty store retailer of Private label Apparel）模式**——从商品策划、设计、生产到零售的一体化运营。

### 模式形成过程
- **起因**：传统"代销"模式下，经销商以高价位购买货物，利润空间被挤压；生产商间接控制零售定价，与优衣库低价亲民的定位冲突
- **突破**：所有服装款式由优衣库自己设计，生产商只负责加工——"定制品"模式
- **关键决策**：1988年在香港成立商品采购办公室，启用当地人做法人代表，实现商品定价自主权
- **深化**：1999年在上海、广州设立生产管理事务所，派专人驻扎中国工厂

### SPA的核心优势
- **定价自主**：完全控制从设计到终端的价格链
- **质量可控**：通过"匠计划"直接管控生产质量
- **快速反应**：设计→生产→上架的全链条信息实时共享
- **利润最大化**：去除了中间环节的利润损耗

## 二、一入一出（One-in-All-out）：聚焦与舍弃的智慧

### 舍弃的艺术
柳井正深谙"取舍之道"，体现在多个层面：

**舍弃旧业务**：
- 1992年关闭最后一家男士西装店铺，全面转型休闲装
- 关闭不盈利的VM儿童服装公司（保留商标权和设计师）
- 运动装"运动潮"和家居装"家居潮"两个新项目果断撤资
- 伦敦扩张失败后，从21家店收缩到5家

**舍弃旧观念**：
- 舍弃家族式独裁经营→走向组织化运营
- 舍弃终身雇用制→保持人员流动性和竞争活力
- 舍弃中央集权→"超级店长"制度让一线独立经营

**舍弃旧成功**：
- "成功一日就可舍弃"——忘记背后，努力面前
- 双面绒大卖2600万件后，主动回归多元产品线
- 业绩下滑时果断"从零开始"重新出发

### 聚焦的纪律
- 所有店铺统一品牌、统一理念、统一标准
- 通过POS系统实时掌握每家店的销售数据
- 按月结算，及时纠正偏差

## 三、经营哲学精华

### 顾客至上
- "公司只有一个上司，即顾客"
- 自助式购物——顾客需要时才求助，营业员不主动推销
- "顾客是在买商品，更是在买企业的形象"
- 三个月内无条件退换货

### 速度与执行力
- "即断、即决、即实行"——决策力、执行力、即战力是优衣库的DNA
- 所有会议控制在10分钟以内
- 晚上7点准时熄灯，原则上禁止加班
- 办公室没有固定办公桌，没有沙发茶几（拒绝闲聊）
- 行程以15分钟为单位安排

### 人人都是经营者
- **店长是最高职位**——用知识和头脑劳动的"独立商人"
- **超级店长制度**——年收入可超3000万日元
- **扁平化管理**——总部改名"支援中心"，服务而非指挥
- **互联网式组织**——每一点都可以成为中心，相互连接实现资源共享
- **人才观**：优秀的管理者必定是一流的销售员；年轻人也能做经营管理

### 实力主义与评价体系
- 每3.5个月一次人事考评，五级排名
- 业绩决定奖金——"即便公司业绩下降，也不能说明员工个人能力下降"
- 不注重潜能，更侧重于发挥出了多大能力
- 赏罚分明，绝不给华而不实的人生存空间

### 变革永不停止
- "组织是为了进攻，不是为了防守"
- 组织不能固定化——随时根据市场变化调整
- "唯新不破，唯变不破"——模仿和依赖他人无法成功
- ABC计划：All（所有）、Better（更好）、Change（改变）

## 四、关键商业决策复盘

### 成功案例
| 事件 | 核心启示 |
|------|----------|
| 1984年广岛一号店开业 | 低价+自助+大规模宣传→开业即爆满 |
| 双面绒营销（1998-2000） | 从200万件→850万件→2600万件；51种颜色创新 |
| 原宿店入市中心 | 借经济危机低价入市，打破"质次价低"形象 |
| 上市（1994广岛→1997东京二部→1999东京一部） | 系统化改革→透明化经营→资本助力扩张 |
| 匠计划 | 日本退休工匠→中国工厂传授技术→质量飞跃 |
| 2008年金融危机中逆势增长 | 高品质低价位=经济下行期的最佳商业模式 |

### 失败案例
| 事件 | 教训 |
|------|------|
| 纽约设计分公司（1995-1998） | 文化交流障碍；纽约设计不适合日本市场 |
| 大阪美国村店 | 市中心开店不能照搬郊区经验；宣传手段失败 |
| 伦敦扩张（2001-2003） | 过快扩张（3年50家目标）；未做市场调查；当地经理文化不合 |
| 运动装/家居装项目 | 与主业不搭调；造成供应链冲突 |
| 关东市场开拓（1993-1998） | 地域文化差异；关东人不认"关西折扣店" |

## 五、人才与管理哲学

### "匠计划"
柳井正网罗日本拥有30年以上资历的成衣技术人才（平均年龄60+），派往中国工厂传授技术。一举三得：
1. 解决日本退休技术人才的再就业
2. 提升中国工厂的生产技术水平
3. 确保优衣库产品的品质标准

### 人才培养
- "带着问题去实践"——先让人坐在想坐的位置上，遇到问题再下基层解决
- "只用头脑思考是危险的"——理论与实践必须结合
- 兴趣是最好的老师——培养人才的关键在于培养兴趣
- 新老结合——老员工经验+年轻人创新=最佳组合

### 沟通文化
- 任何人会议上必须发言，否则"下次不要来参加"
- 不注重形式汇报——"绝对不会给只擅长汇报、华而不实的人以生存空间"
- 随时随地的"头脑风暴"

## 六、国际化教训与智慧

### 三次海外尝试
1. **伦敦（2001）**——惨败。原因：过快扩张、本地经理文化不合、未做市场调查、廉价策略水土不服
2. **中国（2001起）**——渐进成功。吸取伦敦教训，先盈利再扩张；调整定位为月薪5000+中产阶级
3. **纽约/巴黎**——稳步推进中

### 国际化原则
- 先让一家店盈利，再开第二家
- 不照搬国内经验，尊重当地市场
- 用优衣库DNA的人管理海外业务，而非完全依赖当地人
- "要想在全球成功，就要给世界人民的生活增色"

## 七、柳井正给企业家的十条忠告

1. 积极面对时代和社会的变化
2. 唯一的、绝对正确的评价者是市场和顾客
3. 永远不要失去对未来的期许、计划、梦想和理想
4. 最为重视的应该是日常事务
5. 公司职员要有伙伴观念和团队精神
6. 勤奋，将一天的24小时集中于工作
7. 充分了解现状，在这个基础上不要失去理想和目标
8. 自己对于商业要拥有比任何人都高的目标
9. 自己的未来由自己来开创，命运不取决于他人
10. 努力经营，使之成为不倒闭的公司——可以一胜九败，但不能有决定性的失误

## 八、经营理念二十条（精要版）

1. **直面现实**，适应时代，发挥主观能动性进行变革
2. **实行绝妙的主意**，开动世界、改变社会
3. **商品和卖场**是与顾客唯一的直接连接点
4. **顺应顾客的期待**，想办法创造顾客
5. 组织不能固定化，实行**没有壁垒的项目原则**
6. 每个职员都要**独当一面**，善于自我反省
7. 活用各种人才，确立**品牌特色**，走国际化路线
8. 光明正大，赏罚分明，**完全实力主义**
9. **速度、干劲、革新、执行力**
10. 全体职员**同心协力**，所有部门联合运作
11. 从本质上提高管理能力，**彻底排除无用功**
12. 不论公司、团队还是个人，都要有**明确的目标和理念**
13. 对自己公司的事业要**高标准严要求**
14. 不因种族、国籍、年龄和性别而产生差别
15. **彻底分析**失败和成功的信息，作为下一步行动参考
16. 顾客买的是商品，更是**企业形象**——追求事物的本质
17. 总是向好的方面想象，**抢先投资**，对未来持有希望
18. 从**正确的事、小事、基础的事**做起
19. **积极挑战**，不回避困难和竞争
20. 自己要成为自己最大的批判者，要有**自我革新力**
