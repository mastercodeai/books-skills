## 场景路由（Scene Routing）

当用户提出与段永平投资思想相关的问题时，按以下路由快速定位响应模式：

### 场景 A：投资决策（我要不要买/卖这支股票？）
**触发词**: 买不买、要不要卖、现在能买吗、该不该持有、估值
**响应流程**:
1. 先问：你是否理解这家公司的商业模式？（能力圈检查）
2. 再问：商业模式是否符合"长坡厚雪"标准？
3. 再问：企业文化你了解吗？管理层是否本分？
4. 最后问：价格你满意吗？
5. 任何一环答不上来 → "你还不能买"
**输出格式**: 逐项过清单，给出"可以考虑/还需观察/不应碰"三档结论

### 场景 B：商业模式评估（这个行业/公司商业模式好不好？）
**触发词**: 商业模式、护城河、差异化、竞争力、赚钱模式
**响应流程**:
1. 用"商业模式优劣判断清单"逐项检查
2. 对标段永平对类似公司的评价（参考公司看法表）
3. 识别商业模式"基因缺陷"（不可修复的结构性问题）
**输出格式**: 好的信号 vs 差的信号对照表 + 结论

### 场景 C：企业文化判断（这家公司的文化怎么样？）
**触发词**: 企业文化、管理层、价值观、诚信、本分
**响应流程**:
1. 从使命/愿景/价值观三维度评估
2. 观察出问题时的第一反应
3. 检查是否有 stop doing list
4. 检查供应链关系（是否压供应商款）
**输出格式**: 文化评级 + 关键观察证据

### 场景 D：选股/比较（几家公司选哪个？）
**触发词**: 对比、哪个更好、A和B选谁、组合
**响应流程**:
1. 分别用场景B和C的框架评估每家公司
2. 按三要素优先级排序：商业模式 > 企业文化 > 价格
3. 商业模式差距明显时，直接选好的；差距不大时看文化
**输出格式**: 对比矩阵 + 推荐结论 + 依据

### 场景 E：心态/理念修炼（我该怎么看待投资？）
**触发词**: 心态、持有、焦虑、波动、贪婪、恐惧、平常心
**响应流程**:
1. 引导回到"买股票就是买公司"的本源
2. 用10年20年视角重新看问题
3. 引用段永平的原话或比喻
**输出格式**: 理念重述 + 具体行动建议

### 场景 F：经营决策（我该怎么经营公司？）
**触发词**: 经营、管理、战略、不为清单、造钟人、企业文化建设
**响应流程**:
1. 先区分"做对的事情"还是"把事情做对"
2. 检查决策是否符合"本分"原则
3. 是否有消费者导向
4. 是否在追求短期利益而牺牲长期
**输出格式**: 决策框架应用 + 建议

## 核心框架

### 投资决策三要素（优先级排序）
1. **商业模式（Right Business）**：最重要的过滤器
   - 能否长期产生大量净现金流？
   - 产品差异化程度如何？护城河有多宽？
   - 资本支出需求大不大？
   - 是否有定价权？
2. **企业文化（Right People）**：第二重要的过滤器
   - 是否有"利润之上"的追求？
   - 核心价值观是否以消费者为导向？
   - 管理层是否正直诚信？
   - 是否有明确的"不为清单"？
3. **价格（Right Price）**：第三重要
   - 时间可以降低价格的重要性
   - 以10年20年视角思考，合理即可

### 商业模式优劣判断清单
**好的商业模式信号：**
- 产品有强差异化，用户需要但竞争对手满足不了
- 有定价权（能涨价不流失客户）
- 库存越久越值钱（如茅台）或无库存压力
- 低资本支出即可维持高回报
- 先收钱后发货，现金流充裕
- 用户转换成本高，粘性强
- 不依赖单一超级明星

**差的商业模式信号：**
- 产品无差异化，只能打价格战（航空、光伏）
- 高负债、高资本支出
- 需要大量员工才能支撑营收
- 靠低价/性价比抢市场
- 需要不断投入才能维持现有回报

### 企业文化评估框架
**三个维度：**
1. 使命：企业为什么存在
2. 愿景：共同的远景目标
3. 核心价值观：大是大非的标准

**关键观察点：**
- 最高领导是否真正相信并践行企业文化
- 出问题时是先问"有没有钱赚"还是先问"这是对的事吗"
- 是否有明确的stop doing list
- 对不诚信行为是否零容忍
- 供应链关系：是否准时付款、不压供应商

### "做对的事情" vs "把事情做对"
| 维度 | 做对的事情 | 把事情做对 |
|------|-----------|-----------|
| 性质 | 方向问题、原则问题 | 效率问题、能力问题 |
| 是否可错 | 不可错（错了要立刻停） | 可以犯错（学习过程） |
| 判断标准 | 长期是否正确 | 执行效率和质量 |
| 关键词 | Stop Doing List | 学习曲线、一万小时 |

### 选股实操指南
1. **先看商业模式**：是否在"长长的坡、厚厚的雪"上
2. **再看企业文化**：管理层是否本分、正直、消费者导向
3. **看懂后等价格**：好公司便宜时买入，不着急
4. **持有原则**：以10年20年视角持有，享受公司成长
5. **卖出原则**：当发现看错了、或企业文化出了根本问题

### 能力圈原则
- 不懂的不碰
- 需要问别人该不该买 = 还没到能买的时候
- 一生能找到10-8个能想清楚10年20年的公司就非常好了
- "不碰"比"碰"更重要

### 常见投资陷阱
1. **追逐市场占有率/排名**——它是结果不是目标
2. **关注短期业绩波动**——容易只见树木不见森林
3. **用PE预测未来利润**——时点性，容易误导
4. **为多元化而多元化**——分散风险往往变成分散精力
5. **追求弯道超车/捷径**——"捷径是迷路的最快办法"
6. **用margin/高杠杆**——"知道自己会死在哪里就别去那里"
7. **看便宜买烂公司**——好马配好骑师才出好成绩

## 失败模式识别（Failure Modes）

### 投资决策失败模式
| 失败模式 | 表现 | 根因 | 修正方法 |
|---------|------|------|---------|
| 能力圈幻觉 | 以为自己看懂了，实际没看懂 | 把"知道名字"当"理解生意" | 问自己：能否用一句话说清这家公司10年后怎么赚钱？ |
| 估值锚定 | 股价从100跌到50觉得便宜 | 用过去价格而非内在价值判断 | 忘掉过去价格，重新算内在价值 |
| 沉没成本 | 亏损后不愿卖，等回本 | 把买入价当参考点 | 问：如果现在空仓，还会买吗？ |
| 从众买入 | 看到别人赚钱就跟 | 不是自己的能力圈 | 别人赚钱和你该不该买是两件事 |
| 短期归因 | 一次赚钱就觉得自己看对了 | 运气和能力混淆 | 连续3年以上、多个决策验证 |
| 安全边际忽视 | 好公司随便什么价都买 | "好公司"替代了"好价格" | 再好的公司也要有合理价格 |

### 经营决策失败模式
| 失败模式 | 表现 | 根因 | 修正方法 |
|---------|------|------|---------|
| 规模崇拜 | 追求做大而非做强 | 把"大"当目标而非结果 | 回到消费者价值，规模是副产品 |
| 多元化冲动 | 什么都想做 | 不甘心只赚一个领域的钱 | 检查是否在能力圈内 |
| 短期KPI驱动 | 只看季度业绩 | 激励机制设计问题 | 把长期指标纳入考核 |
| 价格战思维 | 靠降价抢市场 | 缺乏差异化能力 | 先建立差异化再谈竞争 |
| 竞争导向 | 天天盯着竞争对手 | 忘了消费者才是服务对象 | 把精力放在消费者需求上 |

## 反模式与纠正（Anti-Patterns）

### 反模式 1：用PE/PB代替商业模式分析
❌ **错误做法**: "PE才15倍，便宜，买！"
✅ **正确做法**: PE是时点性指标，先判断商业模式是否值得拥有，再看价格是否合理。
> 段永平原话：用PE预测未来利润容易误导。

### 反模式 2：追求"完美时机"
❌ **错误做法**: "等跌到XX再买"，反复等待错过机会。
✅ **正确做法**: 好公司合理价格就可以买，时间会降低买入成本的重要性。
> 10年20年视角看，买入时便宜5%还是贵5%几乎不影响最终回报。

### 反模式 3：分散持仓降低风险
❌ **错误做法**: 持有20+只股票，认为这样更安全。
✅ **正确做法**: 集中持有自己真正懂的3-5家公司。分散是不懂的借口。
> 一生能找到8-10个能想清楚的公司就非常好了。

### 反模式 4：把"长期持有"当教条
❌ **错误做法**: 发现看错了还死拿，"我是长期投资"。
✅ **正确做法**: 长期持有基于"没有看错"。发现错立刻停，这才是"做对的事情"。

### 反模式 5：用公司过去的成功预测未来
❌ **错误做法**: "这家公司过去10年涨了10倍，未来也会"。
✅ **正确做法**: 看商业模式的结构性优势是否还在，文化是否还健康。

### 反模式 6：把"性价比"当竞争优势
❌ **错误做法**: "这家公司产品便宜好用，肯定能赢"。
✅ **正确做法**: 性价比是战术不是战略。靠低价得来的份额，别人也能用低价抢走。

### 反模式 7：看K线/技术指标做买卖决策
❌ **错误做法**: 根据均线、MACD等技术信号操作。
✅ **正确做法**: 段永平从不看图。买股票就是买公司，买的是企业未来现金流的折现值。

## 检查点（Checkpoints）

### 购入前检查清单
- [ ] 我是否真正理解这家公司的商业模式？（能一句话说清怎么赚钱）
- [ ] 商业模式是否在"长坡厚雪"上？（长期可持续）
- [ ] 我是否了解这家公司的企业文化？（管理层是否本分）
- [ ] 我是否在自己的能力圈内？（不需要问别人该不该买）
- [ ] 当前价格是否有足够的安全边际？
- [ ] 如果股市关门5年，我是否还愿意持有？
- [ ] 我的买入理由是什么？写下来。

### 持有期间检查清单（每季度/半年）
- [ ] 公司的商业模式是否发生了根本变化？
- [ ] 企业文化是否出现退化迹象？（管理层言行不一致、开始做不对的事）
- [ ] 管理层是否还在"做对的事情"？
- [ ] 我当初的买入逻辑是否还成立？
- [ ] 我是否因为价格波动而焦虑？（如果是，说明可能买贵了或不够懂）

### 卖出触发条件
- [ ] **发现看错了**: 商业模式的理解有根本性错误
- [ ] **企业文化出了根本问题**: 管理层开始不诚信、追求短期利益
- [ ] **有更好的机会**: 另一家公司明显更好且价格更合理（谨慎使用）
- [ ] **不再在能力圈内**: 行业变化导致自己看不懂了
> 注意：价格下跌、短期业绩不达预期，不是卖出理由。

### 企业经营检查清单（如果你是经营者）
- [ ] 决策依据是"消费者需要"还是"竞争对手在做"？
- [ ] 是否在追求"利润之上"的价值观？
- [ ] 是否有明确的 stop doing list 并在执行？
- [ ] 对不诚信行为是否零容忍？
- [ ] 是否在靠低价抢市场？（危险信号）
- [ ] 供应链伙伴是否被公平对待？

## 学习路径建议
1. 反复理解"买股票就是买公司"
2. 学习识别商业模式的好坏（看苹果、茅台、航空公司的对比）
3. 理解企业文化的重要性（看苹果发布会、读《基业长青》）
4. 建立自己的"不为清单"
5. 培养平常心——回到事物本源思考
6. 积累能力圈，只投自己能想清楚10年20年的公司
7. 实践"做对的事情"，发现错立刻停
