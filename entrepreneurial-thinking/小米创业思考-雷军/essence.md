# 小米创业思考 — 精华提炼

> 雷军 口述 / 徐洁云 整理  
> 中信出版集团，2022年8月

---

## 一、核心命题

**小米自创立至今只干了一件事：用互联网的思维和方法，改造传统制造业，推动商业社会的效率革命，以实现最大化的用户利益和社会经济运转效率。**

商业的目的是什么？如何让商业实现最大化的现实意义？雷军的答案是：**效率**。它能给最多的人带来最大化的美好幸福感。

## 二、小米方法论：互联网七字诀

### 1. 专注
- **清晰的使命、愿景**：使命是企业存在的目的和意义，愿景是企业最终希望成为的样子。小米使命——"始终坚持做'感动人心、价格厚道'的好产品，让全球每个人都能享受科技带来的美好生活"。
- **克制贪婪，少就是多**：决定不做什么跟决定做什么一样重要。苹果1997年回归时砍掉70%产品线，只做四个伟大产品。
- **单点切入，逐步放大**：一个明确且用户迫切需要的产品，更容易找到明确的用户群。小米充电宝只解决"大容量、质量可靠又便宜"这一个问题。
- **全局对表，实时校验**：核心业务和核心方向必须是一元的。小米从"手机+AIoT"改为"手机×AIoT"，强调乘法效应而非简单加法。

### 2. 极致
- **做到自己能力的极限**：哪怕只好1%，也愿意多投入100%。小米手机4的不锈钢中框单台加工成本400元，是铝材的数十倍。
- **追求最优解**：每个品类、每种需求都存在当下的最优解。MIUI的百变主题至今仍是安卓系统个性化的最优解。但最优解有时代性，没有永恒的最优解。
- **不是自嗨和自我感动**：花同行5倍的钱在螺丝钉上铣小米标识，对用户没有价值——这不是极致，是浪费。追求极致要问三个问题：①是不是用户真正需要的？②能不能成为核心竞争力？③能不能形成长期竞争壁垒？
- **投资未来**：陶瓷机身从"烧一炉碎一炉"到成为MIX系列标志，这条路的真正价值不在今天，而在明天的竞争力。

### 3. 口碑
- **超过预期才有口碑**：好产品不一定带来口碑，便宜的不一定带来口碑，又好又便宜的也不一定。只有超过预期的产品才能带来口碑。海底捞用一个西瓜超越预期，帆船酒店期望过高反而失望。
- **口碑第一，销量第二**：口碑总是领先销量或利润一步。MIX一代从项目看亏损，但带来的口碑价值无法估量。
- **做全栈口碑**：产品、服务和沟通——与用户所有触点的表现总和才是口碑。
- **口碑阈值不断提升**：刻舟求剑是大忌。用户预期随着产品升级、客群拓展而不断变化，必须持续追踪。
- **品牌建设是口碑的沉淀固化器**：品牌战略、技术产品战略、用户经营战略"三位一体"时，公司才能持续高速发展。

### 4. 快
- **快是成长效率**：对企业而言是更低的机会成本、更快的改善成长速度。
- **四种能力**：洞察快、响应快、决策快、改善快。MIUI每周更新、"100分钟必有回应"。
- **局部的慢才是全局的快**：战略积累快不得，战术演进慢不得。红米第一代推倒重来，4000万研发费打水漂，但换来了4460万台的销量。
- **不要用战术上的勤奋掩盖战略上的懒惰**：小米前期超高速成长掩盖了管理问题，2015年一下子全部爆发。

## 三、小米模式的实践：四大关键词

### 1. 技术为本
- 小米是一家科技公司，技术是安身立命之本和长期发展之源。
- 五大技术板块：硬件及驱动、系统及基础软件、通信技术、人工智能、设计。
- 技术创新三类：集成式创新 → 自主创新 → 颠覆式创新。
- 2021年研发投入132亿元，近6年复合增长率超40%。未来5年投入超1000亿。
- **工程师文化就是鼓励创新**：包容失败、包容错误，不以成败论英雄。MIX全面屏来自工程师闲聊"未来手机是什么样"——雷军的回应是"不要考虑时间、投入、量产，做到出来为止"。
- **百万美金技术大奖**：小米内部最高技术奖项，每年重奖有突破性贡献的技术团队。

### 2. 和用户交朋友
- **不把用户当上帝，当朋友**：上帝太虚了，朋友不会坑你。
- **性价比是最大的诚意**：不追求"把稻草卖成金条"。向开市客学习克制贪婪——任何商品毛利率不超过14%。
- **性价比不是低价，是比较优势**：同等价格性能最好，同等性能价格最低。98英寸电视19999元仍是极致性价比（同类15万元）。
- **新媒体不是营销，是沟通**：雷军亲自发微博13000多条，坚持十几年。员工发微博犯错不批评，不发微博才批评。
- **让用户参与公司治理**：米粉代表质询管理层，书面回复并向所有用户公开。

### 3. 爆品模式
- **爆品的定义**：产品力超群，具有一流口碑，最终实现海量长销。四个特征：单款、精品、海量、长周期。
- **爆品三条件**：高效的工业化大生产 + 迎合大众消费能力的定价 + 产品体验导向的创新。
- **四项关键能力**：
  - 洞察未来：产品要有"明天属性"（用过就不想放手）
  - 洞察用户：精准取舍功能，满足80%用户的80%需求
  - 创新实现：重组技术和供应链（充电宝用笔记本18650电芯、扫地机用工业级照相机扫描技术）
  - 精准触达：直达目标用户
- **小白模式**：代入普通用户视野，不要用"专家模式"思考。小米电视遥控器从几十个按钮简化到11个。

### 4. 高效率模型
- **小米模式要解决两个矛盾**：
  - "好"与"便宜"的矛盾——通过提升商业效率
  - "合理的低毛利定价"与"企业盈利"的矛盾——通过商业模式创新
- **"不可能三角"**：产品感动人心 + 价格极其厚道 + 公司不错盈利。小米模式让三者兼得。
- **商品定价五要素**：制造分摊成本 + 研发分摊成本 + 市场推广分摊成本 + 销售渠道分摊成本 + 利润。小米通过爆品模式降低全部分摊成本。
- **铁人三项**：硬件（爆品） + 新零售（高效交付） + 互联网服务（"小费模式"赚利润）。
- **投资回报率公式**：ROI =（毛利率 – 费用率）× 年周转次数。小米之家年周转17次，ROI达30%。
- **综合运营成本（OPEX）长期保持在10%左右**，世界级效率。

## 四、新零售：小米之家模式

- **本质**：用极致的电商效率做线下渠道，提供更极致的服务体验。
- **与传统模式的区别**：小米和用户之间只有一层零售商，中间平均费用率仅10%（传统模式30%~33%）。货物所有权在小米手中，实现"交易即数据"的实时管理。
- **不是博弈，而是合作**：小米和渠道商是合伙人，共同追求高效运转。渠道商获得低风险和高效运营带来的收益。
- **破除单品毛利率迷信**：投资回报率 =（毛利率 – 费用率）× 年周转次数。小米之家ROI达30%，证明薄利可以实现可观回报。
- **零售的本质**：用极致的效率实现极致的用户体验。

## 五、生态链模式

- **目标**：改变100个行业的面貌，支撑AIoT先遣队，引流+树品牌+顺便赚钱。
- **原则**：投资不控股，帮忙不添乱。生态链公司是独立的兄弟公司。
- **爆品方法论案例**：
  - **小米移动电源**：目标最优解。只做一条产品线，进口电芯+铝合金一体成型，69元定价。第一年出货1000万只。
  - **小米手环**：功能做减法（砍掉屏幕），续航做到远超30天，79元定价。两年卖出2000万只。
  - **小米空气净化器**：只做PM2.5净化，不做花哨功能。塔式结构+双风机，899元定价。颠覆行业。
  - **小米插线板**：美轮美奂，每个毫米的细节都抠。三个强电+三个USB均匀排布，49元定价。
  - **米家扫地机器人**：只求扫得干净、扫得快，砍掉拖地等功能。自研LDS传感器，1699元定价。

## 六、三大铁律与未来策略

### 三大铁律（永不更改）
1. **技术为本**：科技公司的根本和基石
2. **性价比为纲**：和用户交朋友最大的诚意
3. **做最酷的产品**：小米永恒的追求

### 三大策略
1. **重新创业**：拿出创业热情，豁出去干
2. **互联网+制造**：深度参与制造业，做"制造的制造"
3. **行稳致远**：用十年的长度来看，做长期有价值的事情

## 七、关键管理思想

### 使命、愿景、价值观
- 使命：做"感动人心、价格厚道"的好产品
- 愿景：做用户心中最酷的公司
- 价值观：真诚与热爱

### 5% 红线
2018年IPO前夕，董事会决议：小米硬件综合净利润率永远不超过5%。"优秀的公司赚取利润，伟大的公司赢得人心。"

### 人才哲学
- "四有新人"：有共同愿景、有能力、有责任心、自驱动
- 组建团队时"三十顾茅庐"——脸皮厚，搞得定人才的就是好办法
- 面试极其严格：实习生面试9次，面试官包括联合创始人

### 工程思维
- 找到一个使命
- 从第一性原理出发
- 理解并重构系统
- 找到第一把"扳手"
- 反复验证、快速迭代
