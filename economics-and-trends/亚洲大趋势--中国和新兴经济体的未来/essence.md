# 亚洲大趋势——中国和新兴经济体的未来

**原书名**: How Asia Works: Success and Failure in the World's Most Dynamic Region
**作者**: [英] 乔·史塔威尔 (Joe Studwell)
**译者**: 蒋宗强 | **出版社**: 中信出版社, 2014
**核心主题**: 亚洲经济发展成败的关键在于政府在农业、制造业和金融三大领域的政策选择

---

## 一、核心论点：三个关键政府干预

史塔威尔认为，东亚最成功的经济体（日本、韩国、中国大陆、中国台湾）之所以能从贫穷迅速过渡到富裕，关键在于政府实施了三个相互关联的干预措施：

### 1. 农业产出最大化——土地改革
- **核心逻辑**: 在贫穷国家，75%人口从事农业。通过均田制土地改革，建立劳动密集型的小农经济（"园艺式精耕细作"），充分利用过剩劳动力实现产出最大化
- **成功范例**: 日本（明治维新及战后改革）、韩国、中国大陆（1949-1956年）、中国台湾
- **失败案例**: 菲律宾、印度尼西亚、泰国、马来西亚——土地改革失败或从未真正推行
- **关键洞见**: 小规模家庭农场的产出远高于大规模种植园，因为劳动力投入与产出成正比
- **连锁效应**: 农业产出增加 → 农村购买力提升 → 创造消费品市场 → 为工业化奠基 → 外汇节省

### 2. 制造业发展——幼稚工业保护 + 出口纪律
- **核心逻辑**: 制造业让不熟练劳动力利用机械创造价值，政府通过补贴和保护培育本国企业，同时通过"出口纪律"逼迫企业在国际市场竞争中提升技术水平
- **出口纪律（核心概念）**: 政府根据企业的出口业绩决定是否给予保护、补贴和信贷。达不到出口标准的企业会被兼并或破产，留下的是世界级企业
- **成功范例**: 日本（通产省模式）、韩国（朴正熙的重化工业运动）、中国台湾、中国大陆
- **失败案例**: 马来西亚（马哈蒂尔的宝腾汽车等项目——缺乏出口纪律、国企无竞争）、东南亚各国
- **关键对比**: 
  - 韩国现代集团：从零起步，28年销量突破1000万辆，成为全球第四大汽车厂商
  - 马来西亚宝腾汽车：投资巨大但从未达到全球竞争力，国内市场依赖保护

### 3. 金融管制——引导信贷服务实体经济
- **核心逻辑**: 金融体系必须服务于农业和制造业发展目标，而非追求短期收益或消费信贷
- **核心工具**: 
  - 资本管制（控制国际资本流动）
  - 银行体系国有化或严格监管
  - 再贴现机制（央行引导商业银行信贷方向）
  - 低储蓄利率（相当于对国民征收隐性税，资金用于工业化）
  - 出口业绩作为获得信贷的前提条件
- **成功范例**: 日本（1950-1980年主银行制度）、韩国（朴正熙时期极其严格的金融管制）、中国
- **失败案例**: 泰国、印度尼西亚、马来西亚——过早放松金融管制，银行被家族企业控制，信贷流向房地产投机
- **华盛顿共识的教训**: 国际货币基金组织和世界银行推动的金融自由化在发展中国家屡屡失败

## 二、关键案例深度分析

### 日本：开创者的成功与局限
- **明治维新**: 东亚最早的现代化改革，土地改革消灭封建地主，农业产出大增
- **战后奇迹**: 解散财阀→国家控制银行→通产省主导工业政策→主银行制度→30年高速增长
- **局限**: 土地改革不彻底导致二战前农村重新贫困化；泡沫经济破灭后陷入"失落的二十年"
- **教训**: 即使最好的政策也不能永远有效，需要适时调整

### 韩国：朴正熙的激进工业化
- **起点**: 二战结束时文盲率比当代埃塞俄比亚还高，人均GNI仅110美元（1962年）
- **策略**: 严厉的出口纪律 + 重化工业投资 + 极端的金融管制（实际负利率）
- **结果**: 现代、三星、LG等企业成为世界级企业，人均GNI达2万美元以上
- **代价**: 独裁统治、压制劳工权益、高通胀、高外债

### 马来西亚：马哈蒂尔的"向东看"为何失败
- **五大失误**:
  1. 缺乏出口纪律——企业不必出口也能获得补贴
  2. 偏好国企而非私营企业——缺乏竞争淘汰机制
  3. 将工业政策与种族平权政策捆绑——排斥有经验的华人企业家
  4. 削弱政府机构作用——个人独断专行
  5. 读错了书——沉迷于大前研一《无国界的世界》而非研究日本实际政策
- **结果**: 宝腾汽车、柏华惹钢铁厂等项目均未达到全球竞争力；亿万富翁们靠垄断国内市场和特许经营权致富，却不发展制造业

### 中国：独特的混合模式
- **农业**: 1949-1956年土地改革成功 → 农业集体化倒退 → 1978年家庭联产承包责任制恢复小农经济 → 产出再次大增
- **制造业**: 
  - 上游：大型国企（中石油、中石化等）充当"减震器"，受政府价格管制
  - 中游：国企在建筑设备、造船、发电设备、电信设备等领域获得全球竞争力（国家发改委 + 国家开发银行推动）
  - 下游：私营企业（华为、比亚迪等）充满活力但缺乏政策支持和资金
- **金融**: 国家开发银行是核心工具，累计贷出1000多亿美元支持企业开拓国际市场
- **挑战**: 国企垄断扩张挤压民企空间（"国进民退"）、民营企业被限制在低附加值环节（"产业微笑曲线"）、人口红利开始消退

## 三、被证伪的流行观点

### 1. "民主促进/阻碍发展"——伪命题
- 日本在民主体制下成功，韩国在专制下也成功
- 民主和法治是发展的**结果**而非前提（阿马蒂亚·森的观点）

### 2. "教育决定发展"——过度简化
- 韩国和台湾起步时文盲率极高，但经济增长后教育自然改善
- 关键不是正规教育，而是**企业内学习**（intra-firm learning）
- 工程教育比人文教育更有利于早期工业化

### 3. "自由市场政策适用于所有发展阶段"——华盛顿共识的根本错误
- 日本、韩国、中国台湾均拒绝了华盛顿共识，选择长期管制金融和保护幼稚工业
- 东南亚国家接受了华盛顿共识的建议，结果在1997年亚洲金融危机中崩溃

### 4. "香港和新加坡是发展样板"——误导性对比
- 离岸金融中心面积小、人口密集、无农业部门拖累，不适用于正常国家
- 世界银行将它们与马来西亚、印尼混为一谈，是分析上的严重错误

### 5. "地理/气候决定论"——缺乏论据
- 台湾属亚热带但经济表现与东北亚一致
- 同样炎热的地区可以有截然不同的发展结果

### 6. "IT服务业可以替代制造业"——印度的教训
- 印度12亿人口中只有300万人从事IT服务业（不到劳动力总量1%）
- 制造业才能大规模吸纳劳动力并推动技术学习

## 四、对投资者的核心启示

1. **区分"真正的发展"和"虚假的增长"**: 储蓄率高不等于发展好（泰国、马来西亚储蓄率与韩国相当但结果迥异）。关注是否有全球竞争力的制造业企业
2. **警惕过早金融自由化**: 银行被家族企业控制、信贷流向房地产投机是危险信号
3. **关注出口纪律**: 一个国家的制造业企业如果不需要出口就能生存，说明缺乏真正的竞争力
4. **中国的投资逻辑**: 上游国企受政府保护利润稳定但增长有限；中游制造企业（发电设备、建筑设备、造船、电信设备）正在快速获得全球竞争力；民营企业虽有活力但资源不足
5. **东南亚的结构性风险**: 缺乏全球竞争力的制造业企业意味着经济增长不可持续，容易在外部冲击下倒退
6. **日本的前车之鉴**: 即使制造业强大，如果服务业和农业落后，加上人口老龄化，也会陷入长期停滞

---

## 五、全书结论

> **一个国家的发展前途掌握在其政府手中。**

成功的东亚经济体证明：通过彻底的土地改革实现农业产出最大化 → 通过幼稚工业保护和出口纪律培育具有全球竞争力的制造业 → 通过严格的金融管制引导资本服务实体经济，这三步是贫穷国家走向富裕的必由之路。没有任何捷径，也没有任何替代方案。IT服务业不能替代制造业，自由市场不能替代产业政策，民主和法治是发展的结果而非前提。
