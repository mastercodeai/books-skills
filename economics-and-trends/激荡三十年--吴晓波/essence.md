# 《激荡三十年：中国企业1978—2008》精华

> 吴晓波 著 | 记录1978-2008年中国企业变革史

---

## 核心主题

本书以编年体方式记录了改革开放三十年间中国企业的崛起与沉浮，围绕**国有资本、民营资本、外资**三股力量的博弈展开，揭示了中国从计划经济向市场经济转轨的复杂进程。

---

## 五大历史阶段

### 第一部 1978-1983：没有规则的骚动
- **1978年**：十一届三中全会确立"以经济建设为中心"，改革开放启动
- 小岗村包产到户（1978）→ 农村改革解放生产力 → 乡镇企业"意外崛起"
- "真理标准大讨论"打破思想禁锢，奠定改革三大基石：实践主义、以经济建设为中心、稳定压倒一切
- 年广久"傻子瓜子"雇工事件引发"姓资姓社"争论，邓小平批示"看一看"
- 邓小平访日访新加坡，汲取经济改革经验；深圳特区设立
- 温州"八大王事件"（1982）：体制外资本力量第一次遭到打击
- 步鑫生成为改革典型（1983），后因盲目扩张而陨落

### 第二部 1984-1992：被释放的精灵
- **1984年**："中国现代公司元年"——联想、海尔、万科、科龙、健力宝等均诞生于此年
- 邓小平第一次南巡（1984），确立特区开放格局；14个沿海城市开放
- 张瑞敏砸冰箱（1985）→ 质量意识觉醒
- 柳传志与倪光南联手，联想汉卡问世
- 健力宝借洛杉矶奥运会一炮而红
- "笼子与鸟"理论主导国企改革：放权让利，但不触动所有制
- 马胜利"承包第一人"（1987）→ 承包制热潮 → 快速陨落
- 海南汽车倒卖事件（1985）：制度灰色地带的寓言
- 1989年"倒春寒"：年广久二次入狱，私营经济遭遇寒流
- **1992年**：邓小平第二次南巡 → "社会主义市场经济"确立 → 第二次下海潮（"92派"）
- 牟其中"罐头换飞机"；黄鸿年"中策现象"引入资本经营概念
- 上海证券交易所（1990.12）和深圳证券交易所（1990.12）相继开业

### 第三部 1993-1997：民族品牌进行曲
- 朱镕基主政经济：清理三角债、分税制改革、汇率并轨（人民币大幅贬值33%）
- 沈太福长城集资案（1993）→ 金融秩序整顿
- 禹作敏大邱庄事件（1993）→ 企业家法治意识缺失的教训
- **民族品牌振兴运动**：联想以价格战击败跨国品牌、长虹倪润峰"价格屠夫"、TCL李东生"敢死队长"
- 保健品行业疯狂崛起：三株口服液（80亿销售额）、飞龙延生护宝液、太阳神
- 秦池"标王"现象（1995-1997）：央视广告的造神与毁灭
- **1997年**："崩塌之年"——亚洲金融风暴 + 秦池勾兑丑闻 + 巨人崩塌 + 三株危机 + 太阳神衰落
- 邓小平逝世（1997.2.19）

### 第四部 1998-2002：在暴风雨中转折
- 朱镕基"闯地雷阵"：人民币不贬值、启动内需（催热房地产）、国企三年脱困
- **"国退民进"战略**：诸城经验 → "抓大放小" → 中小国企产权改革
- 柯达"98协议"：全行业收购中国胶卷业，唯一一例外资整体收购案例
- **500强梦想**：国家扶持宝钢、海尔、长虹等6家"种子选手"
- **股市乱象**：吕梁中科创业坐庄、唐万新德隆系"天下第一庄"
- 互联网元年（1999）：马云创办阿里巴巴、新浪/网易/搜狐赴美上市
- **2001年**：加入WTO → 中国经济全面融入全球化；"中国制造"崛起
- 任正非《华为的冬天》（2001）：企业危机意识的典范
- 联想收购IBM PC业务（2004）：中国企业国际化的里程碑

### 第五部 2003-2008：大国梦想成真
- 2003年"重型化运动"：民营资本冲击重化工业 → 2004年宏观调控 → 铁本事件（戴国芳悲剧）
- 德隆崩塌（2004）：1200亿帝国灰飞烟灭，唐万新入狱
- 郎咸平风暴（2004）：质疑国企MBO → 引发产权改革大辩论
- 科龙悲剧：潘宁出走 → 顾雏军入主 → 格力柯尔崩塌 → 产权不清的牺牲品
- **2005年**：股权分置改革启动 → 中国股市"第二次革命"
- TCL跨国并购（汤姆逊彩电、阿尔卡特手机）→ 国际化阵痛
- 联想收购IBM PC → 中国企业最大规模跨国并购
- **2007年**：股市"全民狂欢"（上证突破6000点）；房价暴涨；杨惠妍成中国首富
- 太湖蓝藻事件 → 高污染高能耗发展模式的警钟
- "中国制造"信任危机：玩具召回、食品安全问题
- 2008年：北京奥运会；全球金融危机前夕

## 核心洞察

### 1. 三重博弈格局
中国商业史的本质是**国有资本、民营资本、外资**三种力量此消彼长的过程：
- 国有资本：垄断优势 → 上市圈钱 → 利润独享
- 民营资本：草莽崛起 → 被打压或边缘化 → 在夹缝中求生
- 外资：技术换市场 → 资本渗透 → 独资化趋势

### 2. 改革的"不争论"特征
每当重大改革被激烈争辩时往往成效甚微（如物价闯关、产权改革辩论），而当争论者精疲力竭时，改革却在最不经意的时刻取得突破。

### 3. "中国特色"的企业家群像
- 出身草莽，不无野蛮，性情漂移，坚忍而勇于搏取
- 浅薄使他们能直指核心，冷酷使他们回到利益基本面
- 不畏天命使他们百无禁忌，贪婪使他们敢于编造最美丽的谎言

### 4. 宏观调控的周期律
经济过热 → 能源紧缺 → 中央行政调控 → 不同所有制企业获不同待遇 → 民营企业往往成为"祭旗者"。几乎每隔三到五年循环一次。

### 5. 产权问题贯穿始终
- "红帽子"企业：民营资本自我保护的权宜之计
- MBO（管理层收购）：从郑俊怀（入狱）到牛根生（成功），命运迥异
- 产权不清是科龙、健力宝等明星企业悲剧的根源
- "小企业易改，大企业难改；亏损企业易改，盈利企业难改"

### 6. 中国企业的致命弱点
- 核心技术缺失：价格战只能在20%的空间内进行
- "贸工技"还是"技工贸"之争（柳倪之争的实质）
- 规模优势无法转化为品牌溢价

## 对当下的启示

1. **产权清晰是企业长治久安的基础**——科龙、健力宝的悲剧反复证明了这一点
2. **核心技术是不可替代的竞争力**——价格战终有尽头，品牌溢价来自技术壁垒
3. **政策周期是最大的系统性风险**——企业家必须理解并适应宏观调控的节奏
4. **繁荣本身蕴含毁灭的种子**——秦池、三株、德隆的崩塌无不印证此理
5. **垄断产生效益，但不产生伟大企业**——世界级企业只能诞生于公平竞争的制度环境
6. **所有成功的国际化都建立在本土深耕之上**——TCL的教训、华为的坚持
