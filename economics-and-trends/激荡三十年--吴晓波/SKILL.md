# SKILL.md — 《激荡三十年》知识应用指南

## 书籍定位
**类型**：中国企业编年史（1978-2008）
**作者**：吴晓波（财经作家，哈佛访问学者）
**核心价值**：记录改革开放三十年间中国企业从无到有、从弱到强的全过程，是一部关于政策周期、企业家命运和制度变迁的全景式叙事。

## 场景详解

### 场景一：政策分析
**触发**：用户询问经济周期、宏观调控、政策演变

**核心知识**：
- **调控周期模型**：经济过热 → 通货膨胀/能源紧缺 → 中央行政调控 → 民营企业受损 → 经济冷却 → 刺激政策 → 新一轮过热
- **典型周期**：1981、1985、1989、1993、1997、2004
- **产权改革路径**：放权让利 → 承包制 → 股份制 → MBO → 股权分置改革
- **国企改革策略**："笼子与鸟"→"抓大放小"→"国退民进"
- **对外开放节奏**：特区 → 14个沿海城市 → WTO → 全面开放

**调控周期详解**:

| 周期 | 触发因素 | 调控手段 | 受损群体 | 反弹逻辑 |
|------|----------|----------|----------|----------|
| 1981 | 洋跃进+基建过热 | 压缩基建+紧缩信贷 | 乡镇企业 | 农村改革释放活力 |
| 1985 | 价格双轨制腐败 | 行政限价+信贷紧缩 | 个体户/倒爷 | 价格闯关失败后转向承包制 |
| 1989 | 通胀+政治风波 | 全面紧缩 | 民营经济集体萎缩 | 92南巡重启改革 |
| 1993 | 投资过热+金融混乱 | 朱镕基铁腕调控 | 海南地产商+民企 | 分税制+国企改革+WTO |
| 1997 | 亚洲金融危机 | 积极财政+国企三年脱困 | 外贸企业 | 扩大内需+住房改革 |
| 2004 | 钢铁/水泥/电解铝过热 | 行政手段选择性调控 | 铁本等民企重工业 | 大型国企反而壮大 |

**回答要点**：每次调控的触发条件、政策工具、对民营经济的冲击、后续反弹逻辑

---

### 场景二：创业与企业家精神
**触发**：用户询问创业路径、企业家命运、草根崛起

**核心知识**：
- **草根创业**：鲁冠球（修自行车→万向集团）、南存辉（补鞋匠→正泰集团）、李书福（照相→造车）
- **体制内创业**：柳传志（计算所→联想）、张瑞敏（家电公司→海尔）、任正非（军队→华为）
- **"92派"现象**：陈东升、毛振华、田源、郭凡生、冯仑等政府官员下海
- **成功要素公式**：政策敏感度 × 行业选择 × 管理能力 × 产权安排

**三类企业家的制度套利空间**:
1. **草根型**: 利用体制不屑于做的"低端"市场（如温州纽扣、义乌小商品），等做大后才进入体制关注领域
2. **体制内型**: 利用体制资源（技术、人才、渠道）+ 市场化激励机制，在体制边缘创造增量
3. **92派型**: 利用体制内积累的人脉和信息优势，抓住市场化转轨的制度红利

**回答要点**：创业时机选择、资源获取方式、制度套利空间、个人能力与时代机遇的结合

---

### 场景三：资本运作与资本市场
**触发**：用户询问股市、并购、上市、资本经营

**核心知识**：
- **股市制度缺陷**：指标配额制、股权分置、庄家经济
- **资本经营案例**：黄鸿年中策模式、牟其中空手道、唐万新德隆系
- **上市策略演进**：借壳上市 → 海外上市 → A+H模式
- **并购案例**：联想收购IBM PC、TCL收购汤姆逊、柯达"98协议"

**中国股市制度缺陷的三个层次**:
1. **出生缺陷**: 为国企脱困服务，非为优化资源配置——先天定位偏差
2. **结构缺陷**: 股权分置（流通股vs非流通股）导致大股东与小股东利益不一致
3. **监管缺陷**: 发审制+退市不力→壳资源价值扭曲→投机文化盛行

**回答要点**：资本运作的制度背景、操作模式、风险边界、成功/失败关键因素

---

### 场景四：民营经济发展
**触发**：用户询问民企困境、产权问题、红帽子、国退民进

**核心知识**：
- **"红帽子"现象**：民营资本自我保护的制度性策略
- **产权清晰化路径**："假集体" → 股份合作制 → 量化改革
- **民营资本边缘化**：2006年后在垄断行业重组中被排斥
- **典型案例**：科龙产权悲剧、健力宝产权纠纷、铁本事件

**红帽子的制度逻辑**:
- 1980年代：个体经济被歧视，挂靠集体企业获得合法身份和政策保护
- 代价：产权归属模糊，做大后必然面临"是谁的企业"之争
- 典型命运：做大→政府介入→创始人出局或被迫低价转让
- 科龙案例：潘宁创办→容桂镇政府持股→潘宁被迫出走→顾雏军收购→顾雏军入狱→海信收购

**回答要点**：制度环境约束、产权安排策略、政商关系处理、政策窗口期把握

## 失败模式速查

| 失败模式 | 典型案例 | 核心教训 | 预警信号 |
|---------|---------|---------|---------|
| 盲目多元化 | 太阳神、巨人、三九 | 主业未稳即扩张，资金链断裂 | 主业利润下滑时仍大举投资新领域 |
| 产权不清 | 科龙、健力宝 | 创始人与股东利益冲突，企业内耗 | 创始人控制权与所有权分离 |
| 对抗政府 | 禹作敏、沈太福 | 企业家政治野心膨胀，触碰红线 | 公开挑战政策、自建"独立王国" |
| 过度资本运作 | 德隆、托普 | 产融结合失控，资金链崩断 | 高杠杆、关联交易、庞氏融资特征 |
| 忽视核心技术 | 国产手机、彩电业 | 组装模式无壁垒，价格战内卷 | 核心部件完全依赖进口 |
| 过度营销 | 三株、秦池 | 营销投入远超产品力，崩塌迅速 | 广告费占比畸高、产品质量投诉激增 |
| 冒进国际化 | TCL收购汤姆逊 | 缺乏技术整合能力，消化不良 | 并购前未做充分尽调、文化整合缺位 |
| 政商关系失控 | 赖昌星、周正毅 | 与官员深度捆绑，一荣俱荣一损俱损 | 企业营收高度依赖政府关系而非市场能力 |
| 家族治理僵化 | 众多温州/潮汕企业 | 任人唯亲导致管理能力天花板 | 关键岗位全由家族成员把持，职业经理人无法进入 |

## 反模式（Anti-Patterns）

**在分析中国商业问题时应避免的思维陷阱：**

| 反模式 | 描述 | 正确做法 |
|-------|------|---------|
| 事后归因谬误 | 用今天的标准评判当年的企业家决策 | 置身当时制度环境和信息约束下理解选择 |
| 英雄史观 | 将企业成败归因于单一个人 | 分析制度环境、行业周期、团队能力等系统因素 |
| 线性外推 | 假设政策环境持续不变 | 关注政策周期，理解"一放就乱、一收就死"的循环 |
| 忽视制度约束 | 假设企业家有完全自主权 | 理解产权制度、金融管制、行业准入等硬约束 |
| 简单道德判断 | 用"好人/坏人"框架评价企业家 | 区分制度套利、灰色地带与违法行为 |
| 脱离时代语境 | 用今天的开放程度评价当年的开放步伐 | 理解每一步开放的政治风险和制度成本 |
| 唯GDP叙事 | 仅用规模增长衡量企业成功 | 关注利润率、治理质量、可持续性等多维指标 |

## 检查点（Checkpoints）

**回答问题前的自检清单：**

- [ ] **时间锚定**：是否明确了讨论的具体年代？（1978年的政策环境与2008年完全不同）
- [ ] **制度背景**：是否说明了当时的产权制度、金融制度、行业准入规则？
- [ ] **案例具体化**：是否引用了书中具体案例而非空泛概括？
- [ ] **多方视角**：是否考虑了政府、企业家、外资、消费者等多方立场？
- [ ] **周期意识**：是否将事件置于政策周期的哪个阶段来分析？
- [ ] **因果区分**：是否区分了相关性与因果性？（如"调控导致民企受损"需具体分析传导机制）
- [ ] **教训可迁移**：总结的教训是否对今天仍有参考价值？

**深入分析时的补充检查**:
- [ ] **产业背景**：是否说明了当时该产业的技术水平、竞争格局、政策环境？
- [ ] **区域差异**：是否区分了沿海/内地、特区/非特区的不同条件？
- [ ] **外资角色**：外资在该事件/行业中扮演了什么角色？
- [ ] **舆论环境**：当时的社会舆论和公众情绪如何影响了事件走向？
- [ ] **代际差异**：第一代企业家与第二代（92派）的思维模式有何不同？

## 三种资本力量博弈

| 维度 | 国有资本 | 民营资本 | 外资 |
|------|----------|----------|------|
| 资源优势 | 政策、土地、信贷 | 市场敏感、成本低 | 技术、品牌、管理 |
| 主要领域 | 垄断行业 | 竞争性行业 | 早期消费品→后转入垄断 |
| 核心矛盾 | 效率低下 | 融资难、政策歧视 | 文化冲突、独资化 |
| 1990s趋势 | 抓大放小→收缩 | 快速扩张 | 大举进入 |
| 2000s趋势 | 国进民退→壮大 | 边缘化 | 独资化+本土化 |
| 典型代表 | 中石油、中国电信 | 万向、华为、联想 | 宝洁、大众、摩托罗拉 |

## 延伸阅读建议

1. **制度经济学视角**：结合科斯、诺斯的产权理论理解中国产权改革
2. **比较研究**：与日本战后经济复兴、韩国财阀模式对比
3. **产业经济学**：从家电、汽车、互联网等行业竞争格局理解中国企业成长
4. **延伸读物**：
   - 吴晓波《大败局》（企业失败案例深度分析）
   - 吴晓波《大败局II》（2000年后企业失败案例）
   - 凌志军《交锋》（中国三次思想解放实录）
   - 张维迎相关经济学论文（产权改革理论）
   - 吴晓波《激荡十年》（2008-2018续篇）
   - 许小年《回荡的钟摆》（中国经济改革反思）
