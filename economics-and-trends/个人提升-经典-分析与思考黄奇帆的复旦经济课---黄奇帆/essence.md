## 二、宏观调控：去杠杆与风险化解

### 四大高杠杆指标
1. **M2/GDP = 2.1:1**（美国仅0.9:1）
2. **金融增加值占GDP 7.9%**（全球均值4%，美国危机前8%）
3. **宏观杠杆率250%**（企业160%、政府50%、居民50%）
4. **每年新增融资90%为债权**（美国30%债权、70%股权）

### 去杠杆的五种方法（五管齐下）
1. **核销破产**：无前途企业壮士断腕，约去债6万亿元
2. **收购重组**：债转股、债务豁免，约去债6万亿元
3. **提高股权融资比重**：从10%→50%，减债30+万亿元
4. **良性通胀稀释**：2-3%/年，五年约10个百分点
5. **稳定M2增长率**：≈GDP增速 + 物价指数 + 逆周期系数

### 金融产品创新的六大杠杆工具（风险之源）
高息揽储 → 刚性兑付 → 资金池 → 错配 → 通道 → 嵌套

**核心案例**：宝能收购万科——万能险高息揽储 + 私募通道 + 银行存一贷二 + 融资融券，70亿撬动450亿。

### 金融的本质（三句话）
1. 为有钱人理财，为缺钱人融资
2. 信用、杠杆、风险——把握三者的"度"
3. 一切金融活动为实体经济服务

## 三、供给侧结构性改革

### 与西方供给学派的三大不同
| 维度 | 西方供给学派 | 中国供给侧结构性改革 |
|------|-------------|---------------------|
| 政策组合 | 仅供给侧 | 供给侧 + 需求侧管理 |
| 政府角色 | 主张私有化、放松管制 | "两只手"共同发力 |
| 所有制 | 私有化 | 两个"毫不动摇" |

### 改革的灵魂地位
供给侧结构性改革 = 改革开放的魂（贯穿40年主线）

历史经典案例：
- 家庭联产承包（释放农民积极性）
- 价格双轨制（培育市场机制）
- 1998年住房商品化（GDP重回10%+）
- WTO倒逼营商环境改革

### "三去一降一补"成果
- 去产能：压减粗钢1.5亿吨+，煤炭8.1亿吨+
- 去库存：商品房待售面积减少27%
- 去杠杆：市场化债转股签约2.3万亿元
- 降成本：减税降费累计近3万亿元

### 十大聚财型、生财型改革红利
1. PPP模式改革（五种平衡、防范五种问题）
2. 财政资金 → 产业引导基金（1:3-4杠杆）
3. 外资金融机构开放（64条措施）
4. 服务贸易开放（7500亿→1.5万亿美元）
5. 降低个人所得税（45%→25%）
6. 农民工落户城市（释放人口红利）
7. 延长女性退休年龄（年增1-2万亿社保红利）
8. 降低运输成本（15%GDP→7%，省3-4万亿）
9. 改革住房公积金 → 年金制度（14.6万亿存量）
10. 城乡土地增减挂钩（7万亿元级原始资本）

## 四、基础货币：货币的"锚"与人民币发行制度

### 三种货币制度
| 制度 | 锚定物 | 代表 | 特点 |
|------|--------|------|------|
| 金本位制 | 黄金白银 | 历史 | 币值稳定但限制经济增长 |
| 主权信用货币制 | 国家信用/GDP | 美日欧 | 自主性强但需纪律约束 |
| 联系汇率制 | 强势货币 | 香港 | 稳定但放弃货币政策独立性 |

### 中国货币发行制度的变迁
- **1949-1995**：物资本位制（以国家掌握物资为基础）
- **1995至今**：汇兑本位制（外汇占款发行，实质锚定美元）
  - 2013年峰值：外汇占款占基础货币83%
  - 2019年：降至约60%

### 美元霸权的五大支柱
1. 充分的物资贸易承载（GDP全球24%）
2. 良好的信用基础（石油-美元体系）
3. SWIFT全球结算体系（80%+交易通过美元）
4. 军事武力保障（航母数量超他国总和）
5. 长期在位优势（70年世界货币惯性）

### 人民币发行制度改革方向："三锚"体系

**宏观锚（四条刚性原则）**：
1. 货币发行增速 ≤ GDP增速 + 通胀率 + 平减系数
2. 禁止财政赤字货币化（财政透支国债只能在二级市场发行）
3. 国债余额/GDP ≤ 70%
4. 绝不实施零利率或负利率

**微观锚**：以30-100种物品的市场成交指数为货币锚（张五常方案）

**数字锚**：以数字化生产要素（计算能力、存储能力、通讯能力）形成的数字化指数为锚

### 转轨路径
- 修改《中国人民银行法》允许央行购买国债
- 财政部发行特别国债，5-8年回购3万亿美元外汇储备
- 国债管理从赤字管理 → 余额管理
- 央行定位从"商业银行的央行" → "整个金融系统的央行"

## 五、资本市场

### 中国资本市场的三大功能弱化
1. **晴雨表功能弱化**：GDP翻三番，上证指数20年不动
2. **投入产出功能弱化**：工业PE约10倍，股市PE约50倍（去银行后）
3. **资源优化配置功能弱化**：独角兽成功案例少

### 九大基础性制度改革建议
1. **科创板**：注册制+退市制试验田，VIE架构+降低门槛
2. **注册制+退市制**：纽约交易所30年保持3000家，进出1:1
3. **企业年金+商业保险**：中国版"401K"，长期资金来源
4. **机构投资者**：解决散杂小问题（25万亿基金仅4万亿在股市）
5. **证券公司资本中介功能**：返还客户保证金管理权，壮大投行
6. **上市公司规范**：鼓励回购注销+并购重组+分红；约束大股东减持+股权质押+随意停牌
7. **取消印花税**：全球主要市场仅剩中国和印度征收
8. **设立国家平准基金**：替代证金公司，央行/特别国债资金来源
9. **建立"中国成分股指数"**：替代上证综合指数，类似标普500

### 长期资本的十大来源
企业年金、社保基金、商业保险、企业闲置资金、家庭投资资金、政府产业基金、LP投资公司、融资融券、银行合规股权投资、境外合规投资

## 六、房地产

### 十大失衡
1. 土地供求失衡（人均建设用地少）
2. 土地价格失衡（地价占房价60-70%）
3. 房地产投资失衡（占固定资产投资25%+）
4. 房地产融资失衡（占社融30%+）
5. 房地产税收失衡（占地方财政40%+）
6. 房地产价格收入比失衡
7. 房地产租售结构失衡
8. 房地产市场秩序失衡
9. 房地产调控方式失衡
10. 城乡建设用地失衡（农民两头占地）

### 长效机制建议
- 土地供应与人口挂钩
- 住房供给"保障+市场"双轨制
- 租购并举
- 房地产税立法
- 控制房地产金融化程度
- 房地产总市值/GDP ≤ 3:1（合理2:1）

## 七、对外开放

### "三零"原则
零关税、零壁垒、零补贴——FTA发展趋势

### 服务贸易四大短板
1. 逆差巨大（2922亿美元，占全球逆差40%）
2. 结构低附加值（知识密集型弱）
3. 货物贸易伴随服务贸易潜力未挖掘
4. 跨国公司"三链"服务基地在海外

### 自贸试验区八大政策
1. 境内关外，五大自由（贸易/投资/资金/运输/从业）
2. 税收优惠（15%企业所得税）
3. 负面清单+竞争中性
4. 离岸贸易、转口贸易、跨境电商
5. 跨境金融服务（64条金融开放措施）
6. 数字经济开放
7. 服务业全方位开放
8. 营商环境国际化法治化

## 八、中美经贸

### 莱特希泽五大谬论的反驳
1. 中国入世美国吃亏论 → 美国企业60%在华生产返销美国
2. 中国未兑现入世承诺 → 关税从15.3%降至7.5%，主动单边降税
3. 强制技术转让 → 企业间自愿商业合同行为
4. 中国顺差导致美国次贷危机 → 美国自身房地产泡沫+金融杠杆
5. 中国买国债操纵汇率 → 维系美元信用的关键支撑

### 美国金融战十条款（2015年法案）
贸易脱钩、金融脱钩、资本市场脱钩、信用降级、谈判冻结、资产冻结、巨额罚款、比特币转移资产、SWIFT切断、推动资本项下自由兑换

### 中国四大应对措施
1. **人民币国际化**：跨境贸易清算、本币互换、离岸人民币、大宗商品人民币计价
2. **补齐金融短板**：降低企业负债率160%→120%、金融企业去杠杆
3. **资本市场改革**：九大基础性制度
4. **金融战攻防**：CIPS备胎、资本项下管制、主权数字货币、扩大金融开放

### 美国债务风险预判
- 2019年联邦债务23.5万亿美元（>GDP的110%）
- 第三上限（崩盘线）：年债务付息+到期债务=全年财政收入
- 预计2023年左右可能触及崩盘线
- 超50%概率引发大级别全球金融危机

## 九、互联网金融

### P2P本质：封建余孽的互联网变种
五大致命机制：网络众筹资本金 + 网民高息揽储 + 无场景放高利贷 + 借新还旧资金池 + 无监管

### 网贷公司五大原则
1. 资本信用（较高资本金门槛）
2. 信用规范（银行贷款+ABS融资）
3. 信用杠杆（总杠杆≤1:10，ABS循环≤5-6次）
4. 放贷征信（有产业链场景）
5. 大数据处理（降低不良率）

### 互联网金融的大格局
互联网/物联网数字平台 + 产业链金融 = 数据平台+金融机构+产业链上下游的资源优化配置

### "大智移云"的五全基因
全时空、全流程、全场景、全解析、全价值

## 十、核心金句与思维方法

### 研究范式
**"问题—结构—对策"**：找到结构性、体制性、机制性、制度性短板，改变问题的联系方式和边界条件。

### 金融三句话
1. 为有钱人理财，为缺钱人融资
2. 信用、杠杆、风险——把握三者的"度"
3. 一切金融活动为实体经济服务

### 关键判断
- "市场是王牌，产业链是王中王，金融是盾牌"
- "进口大国是世界强国，供应链掌控是王中王"
- "用开放化解封锁，用更大的开放瓦解对抗"
- "退市制度没搞好，只进不出、只吃不拉，是股市发展不顺利的基本原因"
- "货币的命运最终也将成为国家的命运"
- "供给侧改革是结构性的改革，是长周期经济发展的动力"
