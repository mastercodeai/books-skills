# 21世纪货币政策 - 精华提炼

**作者**: 本·伯南克 (Ben Bernanke) - 2022年诺贝尔经济学奖得主，前美联储主席 (2006-2014)
**译者**: 冯毅 | **出版社**: 中信出版社 | **出版时间**: 2022年11月

---

## 核心主题

本书从历史视角审视美联储货币政策的演变，聚焦三大长期经济发展因素如何重塑了现代央行的工具、策略和沟通方式：

1. **通胀行为的持续变化** - 通胀与就业关系的根本性改变
2. **中性利率(R*)的长期下行** - 限制了传统降息空间
3. **系统性金融不稳定风险增加** - 金融危机不再是罕见事件

---

## 第一部分：通货膨胀的起落 (1960s-1980s)

### 第一章：大通胀

**菲利普斯曲线的演变**
- 原始菲利普斯曲线(1958)：低失业率 ↔ 高通胀，反映需求驱动的经济波动
- 20世纪70年代修正：纳入供给侧冲击(如油价)和通胀预期的作用
- 现代版本三个结论：
  1. 需求增长超越供应 → 更高通胀
  2. 供给侧冲击 → 滞胀(高通胀+低产出)
  3. 通胀预期上升 → 实际通胀按相同比例上升(恶性循环)

**大通胀的根源(1965-1982)**
- 财政政策过度扩张：越战+伟大社会计划
- 自然失业率(u*)估计错误：当时认为4%，实际约5.5-6%
- 尼克松的工资-物价管制失败
- 石油危机(1973、1979)推高成本
- 通胀预期失控："工资-物价螺旋形上升"

**关键教训**
- 政治压力会扭曲货币政策
- 对自然失业率的过度自信会导致灾难性后果
- 央行独立性的重要性

### 第二章：伯恩斯与沃尔克

**阿瑟·伯恩斯(1970-1978)：失败的教训**
- 认为通胀是"成本推动"而非"需求拉动"
- 支持工资-物价管制而非紧缩货币政策
- "时断时续"政策：既未实现低通胀，也未实现持续低失业
- 屈服于尼克松的政治压力，丧失独立性
- 核心错误：认为通胀与货币无关

**保罗·沃尔克(1979-1987)：坚定的胜利**
- 1979年10月关键会议：转向货币主义框架(货币供应量目标)
- 实际目的：转移对20%高利率的政治批评
- 通胀从1979-80年的13%降至1982年的4%
- 代价：1982年失业率达10.8%峰值
- 1982年放弃货币主义，回归联邦基金利率目标

**沃尔克的信誉遗产**
- 恢复了美联储作为通胀斗士的信誉
- 为随后20年的"大稳健"奠定基础
- "信誉是通过行动建立的，不是口头喊话"
- 核心教训：央行独立性+坚定行动=长期经济稳定

### 第三章：格林斯潘和繁荣的20世纪90年代

**格林斯潘的货币政策哲学**
- "先发制人"打击通胀：在通胀上升前加息
- 风险管理方法：考虑多种可能情境，而非单一预测
- 数据驱动决策：依赖市场信息和"小道消息"
- "含糊其词和模棱两可"的沟通风格

**1994-1996年软着陆**
- 在通胀仍低时开始加息(先发制人)
- 联邦基金利率从3%升至6%，然后暂停
- 经济继续增长，失业率下降，通胀稳定
- 格林斯潘最自豪的成就

**1996年推迟加息的关键决策**
- 传统模型显示应加息，但格林斯潘判断生产率增长将抑制通胀
- "员工不安全感假说"：技术变革使工人不敢要求加薪
- 事后证明正确：生产率数据被大幅上修

**"非理性繁荣"与股市**
- 1996年12月著名演讲：警告股市泡沫
- 尝试用"强烈呼吁"而非加息来降温
- 互联网泡沫最终在2000年破裂
- 教训：货币政策难以精确控制资产价格

## 第二部分：全球金融危机和大衰退 (2007-2009)

### 第四章：新世纪与新挑战

**中性利率(R*)的长期下降**
- 10年期国债收益率从1980年代的15%+降至2020年前的<2%
- 两大解释：
  1. **长期停滞假说**(萨默斯)：人口老龄化、生产率放缓、投资需求疲软
  2. **全球储蓄过剩**(伯南克2005)：东亚高储蓄、产油国盈余、全球收入增长
- 后果：限制了美联储降息空间，有效下限(ELB)问题凸显

**2003年通缩恐慌与前瞻性指引的诞生**
- 核心PCE通胀降至1%，25%概率出现通缩
- 联邦基金利率降至1%(1958年以来最低)
- 关键创新："相当长的一段时期"保持低利率
- 前瞻性指引首次成为重要政策工具

**房地产泡沫的形成**
- 三大因素：从众心理、金融创新(证券化)、监管不足
- 宽松货币政策并非主因(2004-05年抵押贷款利率仍在6%左右)
- 次级贷款泛滥："忍者贷款"(无收入、无工作、无资产)
- 监管碎片化：空白+重叠，无人对系统性风险负责

### 第五章：全球金融危机

**影子银行与批发融资**
- 影子银行：非银行金融机构提供类似银行服务
- 批发融资：商业票据、回购协议(隔夜抵押贷款)
- 到2006年底，无保险批发融资(5.6万亿)超过有保险存款(4.1万亿)
- 关键脆弱性：短期融资支持长期资产，易受挤兑

**证券化的风险**
- 将抵押贷款打包成复杂证券，风险被隐藏
- 评级机构利益冲突，过于轻信金融工程
- 次级贷款变成"全球资产"：德国储蓄银行、日本养老基金都持有

**危机的阶段**
1. 2007年8月：法国巴黎银行停止赎回次贷基金 → 恐慌开始
2. 2008年3月：贝尔斯登被摩根大通收购
3. 2008年9月：雷曼兄弟破产 → 全面恐慌
4. 货币市场基金"跌破1美元" → 挤兑蔓延
5. 信贷市场冻结 → 经济急剧下滑

**美联储的应对：最后贷款人**
- 白芝浩原则：自由地向有偿付能力的机构放贷
- 援引《联邦储备法》第13条第3款：向非银行机构放贷
- 与14家外国央行建立货币互换额度：充当全球最后美元贷款人
- 创新工具：定期拍卖工具(TAF)、商业票据融资工具(CPFF)、TALF

**雷曼兄弟为何无法拯救**
- 已严重资不抵债(不仅是流动性问题)
- 没有买家愿意收购(即使有政府担保)
- 但拯救雷曼也无法阻止更广泛的危机

### 第六章：从QE1到QE2

**量化宽松(QE)的诞生**
- 2008年11月25日：宣布购买5000亿美元抵押贷款支持证券
- 2008年12月：联邦基金利率降至0-0.25%
- 2009年3月：QE1扩大至1.75万亿美元(国债+MBS)

**QE如何运作**
- **不是**通过增加银行准备金(货币主义观点)
- **而是**通过两个渠道：
  1. **投资组合平衡渠道**：减少长期证券供应 → 推高价格 → 压低收益率
  2. **信号传导渠道**：承诺长期保持低利率 → 影响市场预期
- 10年期国债收益率下降1个百分点 ≈ 联邦基金利率下降3个百分点

**QE1的效果**
- 市场立即反应：抵押贷款利差收窄0.65个百分点
- 30年期抵押贷款利率下降约1个百分点
- 帮助稳定金融市场，支持经济复苏

**QE2(2010年11月)**
- 购买6000亿美元国债
- 市场已充分预期，反应温和
- 政治反弹：共和党领袖警告"难以控制的通胀"
- 新兴市场批评："货币战争"(巴西财长语)

### 第七章：QE3和缩减恐慌

**增强前瞻性指引**
- 2011年8月："至少持续到2013年中"保持低利率(时间依赖性)
- 2012年1月：引入2%正式通胀目标
- 2012年9月：延长至"2015年中"
- 2012年12月：状态依赖性指引 - 失业率6.5%为加息门槛

**期限延长计划/扭曲操作(2011年9月)**
- 购买4000亿美元长期国债，出售等量短期国债
- 资产负债表不变，但延长所持证券平均期限
- 目的：降低长期利率，同时不增加银行准备金

**QE3(2012年9月)：开放式购买**
- 每月购买850亿美元(450亿国债+400亿MBS)
- 关键创新：开放式，无总额或结束日期
- 条件："就业市场前景没有大幅改善"则继续
- 包含免责条款：考虑"可能的效力和成本"

**2013年缩减恐慌**
- 伯南克5月国会证词暗示可能放缓购买
- 市场反应剧烈：10年期国债收益率上升1个百分点
- 根源：市场预期与美联储内部分歧脱节
- 教训：沟通精确性至关重要，市场情报需要改进

## 第三部分：从加息周期到新冠肺炎大流行

### 第八章：进入加息周期

**耶伦时代的政策特点**
- 与格林斯潘相似：巩固前任成果，实现软着陆
- 关注劳动力市场多维度指标("仪表盘")
- 风险管理方法：不对称风险(接近有效下限)

**加息周期的延迟**
- 2015年12月首次加息(距QE3结束14个月)
- 原因：国际不确定性(人民币贬值、英国脱欧)
- 对"新常态"的重新评估：
  - 中性利率R*持续下降(估计值从4.25%降至2.5%)
  - 自然失业率u*也在下降(从5.5%降至4.1%)
  - 菲利普斯曲线"扁平化"：通胀对失业反应减弱

**菲利普斯曲线的现代演变**
- 1990年后：通胀对失业短期反应变弱
- 通胀更加稳定：冲击后迅速回归目标
- 原因：沃尔克恢复的美联储信誉锚定了通胀预期
- 后果：政策制定者有更多空间支持就业，但通胀指标变得不可靠

### 第九章：鲍威尔和特朗普

**鲍威尔的货币政策哲学**
- 强调"星光引路"的不确定性：R*和u*的估计一直在变
- 数据驱动的风险管理方法
- 与国会建立良好关系以制衡总统攻击
- "用简明的语言"解释政策

**2018-2019年政策转向**
- 2018年加息4次至2.25-2.5%
- 2018年12月市场暴跌 → 政策过于紧缩
- 2019年1月：转向"耐心"，不再加息
- 2019年7-10月：三次"保险性降息"(应对贸易战不确定性)

**特朗普的攻击**
- 打破总统不公开批评美联储的惯例
- 鲍威尔应对策略：强调独立性、不回应批评、与国会发展关系
- 最终成功捍卫了美联储独立性

### 第十章：新冠肺炎大流行

**2020年3月的金融恐慌**
- 美国国债市场近乎崩溃："现金冲刺"
- 与2008年类似但更快更猛
- 美联储迅速行动：
  - 3月3日：紧急降息50基点
  - 3月15日：降至0-0.25%，宣布大规模购买
  - 3月23日：购买量"无限制"
  - 与9家外国央行重建货币互换额度

**《新冠病毒援助、救济和经济安全法案》(CARES Act)**
- 2.2万亿美元财政计划
- 拨款4540亿美元支持美联储紧急贷款
- 美联储可提供至少2.3万亿美元贷款
- 新工具：企业债购买、市政债支持、主街贷款计划

**2020年8月新框架：灵活平均通胀目标(FAIT)**
- 两大改革：
  1. 弥补过去通胀不足：允许通胀"适度超过2%"
  2. 放弃先发制人打击通胀：不再仅因失业率下降而加息
- 反映：中性利率下降、菲利普斯曲线扁平化、劳动力市场包容性

## 第四部分：前路漫漫

### 第十一章：QE和前瞻性的证据

**QE的有效性**
- 事件研究：QE1公告导致10年期国债收益率下降1个百分点
- 投资组合平衡效应：即使在正常市场也有效
- 信号传导效应："缩减恐慌"本身就是证据
- 总体估计：QE+前瞻性指引 ≈ 联邦基金利率额外下降3个百分点

**QE的成本和风险**
- 未导致持续过度通胀(实际上通胀长期过低)
- 退出技术上可行(准备金利息工具)
- 未损害证券市场运行
- 金融稳定风险：可能增加冒险行为，但主要在危机时使用
- 经济不平等：净效应可能降低不平等(热门劳动力市场惠及弱势群体)
- 市场扭曲：不存在"纯"市场结果，所有政策都影响利率
- 僵尸企业：根源在银行监管不足，非低利率本身

### 第十二章：美联储工具箱是否足够

**历史证据**
- 2008年后：QE+前瞻性指引确实产生了比仅降息更好的结果
- 但不太可能完全补偿有效下限的限制
- 2020年：吸取教训后更快更有力部署，经济复苏超预期

**中性利率的关键作用**
- R*=2.5%时：传统降息空间仅2.5个百分点
- 加上QE+前瞻性指引的3个百分点：总火力约5.5个百分点
- 接近但可能不足以应对深度衰退

### 第十三章：新的工具和框架

**国外经验**
1. **购买更广泛资产**：公司债、商业票据(欧洲央行、日本银行)
2. **融资换贷款计划**：补贴银行贷款(英格兰银行、欧洲央行TLTRO)
3. **负利率**：欧洲央行-0.5%、日本银行-0.1%(争议较大)
4. **收益率曲线控制**：日本银行2016年引入(固定10年期国债收益率)

**替代性政策框架**
- **标准通胀目标制**：2%目标，"中期"实现(当前主流)
- **价格水平目标制**：弥补过去偏差(从未被采用)
- **临时价格水平目标制**：仅在有效下限时弥补(伯南克2017年提出)
- **名义GDP目标制**：同时考虑增长和通胀(2011年被美联储否决)
- **提高通胀目标**：如4%(被排除，政治和经济成本太高)

**货币政策-财政政策协调**
- 财政政策优势：不受有效下限限制、可精准定向
- 财政政策劣势：政治程序繁琐、不够灵活
- "直升机撒钱"：效果与传统债务融资退税相似(独立央行条件下)
- 现代货币理论(MMT)：将财政主导制度化，风险极高

### 第十四章：货币政策和金融稳定

**股市泡沫 vs 信贷繁荣**
- 股市崩盘(如1987年)：若无信贷危机，经济影响有限
- 信贷+房地产泡沫破裂：历史证明最危险(1929、2008)

**货币政策的风险承担渠道**
- 宽松政策 → 改善资产负债表 → 增加冒险意愿
- 但也可能变得过度：心理因素(短视、外推)、激励缺陷

**宏观审慎政策**
- 应对金融不稳定的第一道防线
- 美国进步：金融稳定监督委员会(FSOC)、压力测试、有序清算权
- 仍然不足：影子银行监管薄弱、房地产信贷工具缺乏

**"逆风"货币政策**
- "永远在线"：始终谨慎使用宽松(代价太大)
- 情境式：仅对可观察风险反应(更可行但证据有限)
- 当前共识：宏观审慎为主，货币政策为辅

### 第十五章：美联储的独立性及其社会角色

**独立性的理由**
- 技术性：货币政策需要专业知识和快速反应
- 政治性：避免短期选举周期干扰长期经济利益
- 当前意义：两党极化下，独立央行更加重要

**新技术**
- 大数据用于经济监测
- FedNow即时支付系统
- 数字美元：潜力与风险并存
- 加密货币：不会取代法定货币

**社会角色**
- 货币政策的最大贡献：促进充分就业(尤其惠及弱势群体)
- 气候变化：将气候风险纳入银行监管
- 多样性：改善经济学专业和美联储自身的代表性

## 核心教训总结

1. **信誉是最宝贵的资产**：沃尔克恢复信誉 → 30年低通胀+稳定增长
2. **独立性是成功的前提**：伯恩斯屈服政治压力 → 大通胀
3. **沟通是98%的空谈**：前瞻性指引可以显著影响市场
4. **灵活性优于教条**：从货币主义到风险管理的演变
5. **工具创新是必要的**：QE、前瞻性指引成为永久工具箱
6. **财政-货币协调在危机中至关重要**：2020年经验
7. **金融稳定是隐性第三使命**：没有它，前两个目标无法实现
8. **中性利率决定政策空间**：R*越低，创新工具越重要
9. **通胀预期管理是核心**：锚定预期 = 更多政策灵活性
10. **历史是最好的老师**：大萧条、大通胀、大衰退的教训塑造了现代央行

## 关键概念速查

| 概念 | 定义 |
|------|------|
| R* (中性利率) | 经济充分就业且通胀稳定时的利率 |
| u* (自然失业率) | 稳定通胀下的最低失业率 |
| ELB (有效下限) | 政策利率可设定的最低水平(接近零) |
| QE (量化宽松) | 大规模购买长期证券以压低长期利率 |
| 前瞻性指引 | 央行对未来政策路径的沟通 |
| 菲利普斯曲线 | 通胀与失业率之间的关系 |
| FAIT (灵活平均通胀目标) | 允许通胀暂时超标以弥补过去的不足 |
| 宏观审慎政策 | 针对系统性金融风险的监管政策 |
| 白芝浩原则 | 央行在恐慌时应自由放贷给有偿付能力的机构 |
| 影子银行 | 提供类似银行服务的非银行金融机构 |
