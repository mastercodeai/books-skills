# 这就是OKR：让谷歌、亚马逊实现爆炸性增长的工作法

> 作者：约翰·杜尔（John Doerr）| 译者：曹仰锋、王永贵

---

## 核心观点

OKR（Objectives and Key Results，目标与关键结果）是一套由公司、团队和个人协同制定目标的管理方法，旨在将整个组织的力量聚焦于完成对所有人都同样重要的事项。它由英特尔公司安迪·格鲁夫于1971年创立，经约翰·杜尔传播至谷歌等数百家企业，成为硅谷最具影响力的目标管理框架。

**OKR不是万能的**——它不能代替正确的判断、强有力的领导和创造性的文化。但当这些要素到位时，OKR能引导个人和团队走向顶峰。

---

## 第一部分：OKR的基本框架

### 一、什么是OKR

**目标（Objective）** = 你想要实现的东西
- 重要的、具体的、行动导向的、鼓舞人心的
- 防止思维和执行过程中的模糊不清

**关键结果（Key Results）** = 检查和监控如何达到目标的标准
- 具体的、有时限的、具有挑战性但可实现的
- 必须可衡量、可验证
- 要么达到，要么达不到，没有灰色地带
- 如玛丽莎·梅耶尔所说："没有具体数字可以衡量的结果，不算关键结果"

**基本公式**：关键结果全部完成 → 目标自然实现

### 二、OKR vs MBO（目标管理）

| 维度 | MBO | OKR |
|------|-----|-----|
| 周期 | 年度 | 季度（更灵活） |
| 设定方式 | 自上而下 | 自上而下 + 自下而上 |
| 与薪酬关系 | 直接挂钩 | 分离 |
| 透明度 | 通常保密 | 全员公开 |
| 调整频率 | 很少调整 | 随时可修改 |
| 目标性质 | 承诺型 | 承诺型 + 挑战型 |
| 评估方式 | 年终考核 | 持续追踪、季度评分 |

### 三、OKR的四大利器

#### 利器1：聚焦与承诺
- 每个周期最多3~5个OKR
- 领导者必须亲自对OKR过程做出承诺
- 选择比能力更重要——对1000件事说"不"
- 时间框架：季度OKR最适合快速变化的市场

#### 利器2：协同与联系
- OKR对所有人公开透明
- 自上而下的协同 + 自下而上的涌现
- 纵向对齐 + 横向跨职能连接
- 理想比例：50%自上而下，50%自下而上

#### 利器3：责任追踪
- 数据驱动，定期检查（至少每月一次）
- 评分标准（谷歌）：0.7~1.0=绿色，0.4~0.6=黄色，0~0.3=红色
- 四种应对：继续、更新、开始、停止
- 季度总结三步：客观评估 → 自我评估 → 反思

#### 利器4：挑战不可能
- 区分"承诺型目标"（必须100%完成）和"挑战型目标"（完成60~70%即为成功）
- 谷歌的"10倍速"原则：10%改进意味着你和别人一样，10倍改进才能改变游戏规则
- 挑战性目标平均失败率40%，但仍构成OKR的重要部分
- 拉里·佩奇："设定疯狂的目标，即使没实现，也会取得不小成就"

## 第二部分：OKR的实施要点

### 一、如何设定好的OKR

**目标设定原则：**
1. 少即是多（3~5个目标，每个目标≤5个关键结果）
2. 自下而上设定（鼓励员工参与，约50%的OKR来自基层）
3. 共同参与（集体达成一致）
4. 保持灵活（可在执行期间修改）
5. 敢于失败（设定"挑战性目标"）

**关键结果匹配（防止单一维度偏差）：**
- 数量型KR + 质量型KR 配对使用
- 例：处理凭证数量 ↔ 审计发现的错误数量
- 福特平托教训：只追求成本和重量目标，忽视安全

**OKR与KPI的区别：**
- KPI是"维持性"指标，OKR是"突破性"目标
- OKR不能沦为没有灵魂的数字

### 二、OKR的节奏

- **季度OKR**：适合大多数企业和快速变化的市场
- **年度OKR**：长期战略规划，配合季度OKR双轨并行
- **月度OKR**：适合新创企业寻找市场定位
- **6周OKR**：适合工程团队与产品开发同步

### 三、OKR评分与总结

**评分流程：**
1. 客观评估：计算关键结果的百分比完成率
2. 自我评估：结合情境的主观判断
3. 反思：学到了什么？下次如何改进？

**核心反思问题：**
- 我是否完成了所有目标？成功/失败的原因是什么？
- 如果重写目标，需要做什么改变？
- 学到了哪些经验可以指导下一周期？

## 第三部分：OKR与持续性绩效管理（CFR）

### 一、为什么需要CFR

年度绩效评估的问题：
- 平均每位管理者耗费7.5小时/人
- 仅12%的HR主管认为"非常有效"
- 受近因效应、正态分布曲线约束
- 无法公平衡量绩效

### 二、CFR三要素

**对话（Conversation）**：经理与员工之间真实的、高质量的交流
- 目标设定与反思
- 持续进度更新
- 双向辅导
- 职业发展
- 轻量级绩效评估

**反馈（Feedback）**：双向沟通评估工作进展
- 具体、及时、建设性
- 360度全方位反馈
- 匿名或公开均可

**认可（Recognition）**：对等的表彰
- 鼓励同事间认可
- 建立明确标准
- 分享增加认同感的故事
- 提高频率和可获得性

### 三、OKR与薪酬的关系

**核心原则：将OKR与薪酬分开**

理由：
- 员工会因规避风险而设定保守目标
- 年度评估是回顾性的，OKR对话是前瞻性的
- 谷歌：OKR仅占绩效考核的1/3甚至更低，评分周期结束后清零

### 四、OKR + CFR = 完整系统

> "两个工具'联姻'才有真正的威力。如果没有目标，就没什么好谈的；如果只有目标而没有持续对话，目标就会流于形式。" ——道格·邓纳莱恩

## 第四部分：文化与OKR

### 一、OKR需要健康的文化土壤

安迪·格鲁夫的英特尔文化七要素：
1. 以人为本
2. 公开透明
3. 简洁解决方案
4. 结果导向
5. 坚守原则
6. 勇于承担风险
7. 信任与诚实

### 二、谷歌"亚里士多德"项目发现的团队五要素

1. 结构与清晰度（目标、角色、计划是否明确）
2. 心理安全感（能否安心承担风险）
3. 工作意义（是否做重要的事）
4. 可靠性（能否彼此信赖）
5. 工作影响（是否认为工作有意义）

### 三、OKR文化的核心特征

- **透明**：每个人的目标公开可见
- **责任**：不只是因为老板命令，而是不能让团队失望
- **信任**：用"分享原则"替代"规则"
- **持续改进**：催化剂（支持工作）+ 营养液（支持个人）

## 关键案例总结

| 公司/组织 | 核心OKR实践 | 关键成果 |
|-----------|------------|---------|
| 英特尔 | "粉碎行动"——重新夺回微处理器市场 | 赢得2300个设计合约，超过目标2000个 |
| 谷歌 | 全公司推行OKR，75个季度持续使用 | 7款10亿+用户产品，市值超7000亿 |
| YouTube | "日均观看时长10亿小时"挑战性目标 | 提前实现10倍增长 |
| Chrome | 从0到周活跃1.11亿用户 | 三年三次设定挑战目标，最终达成 |
| 盖茨基金会 | OKR用于全球健康项目追踪 | 几内亚蠕虫病例从7.5万降至22例 |
| Adobe | 废除年度评估，推行"签到"模式 | 自愿离职率大幅下降 |
| Remind | 聚焦教师用户，OKR驱动增长 | 从5人到60人，获4000万美元C轮 |
| 减肥宝 | 跨团队协同OKR | 被安德玛4.75亿美元收购 |
| Nuna医疗 | CEO以身作则推行OKR | 为7400万美国人建立医疗数据平台 |
| Zume比萨 | OKR + CFR驱动全流程 | 3个月占领当地10%市场份额 |

## 实践智慧

1. **"想法很容易，执行最重要"** ——约翰·杜尔的口头禅
2. **耐心与坚定**：组织可能需要4~5个季度才能完全适应OKR
3. **从管理层开始**：先在领导团队试行1~2个季度，再向全员推广
4. **OKR是工具，不是武器**：不与惩罚挂钩，鼓励承担风险
5. **完成胜于完美**：不要让完美成为优秀的敌人
6. **"巨石理论"**：先放大石头（最重要的事），再放鹅卵石，最后放沙子

---

## 一句话总结

> OKR是一种已被证实的、能够实现卓越的强大工具——它帮助组织聚焦最重要的事、实现团队协同、基于数据追踪进展、并不断挑战极限。与CFR（对话、反馈、认可）结合使用时，OKR不仅是目标管理系统，更是文化变革的引擎。
