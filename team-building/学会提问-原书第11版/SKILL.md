# SKILL.md — 学会提问：批判性思维实操技能

## 技能概述

**学会提问**是一套系统化的批判性思维技能，核心是通过提出关键问题来评估论证、识别谬误、做出理性判断。适用于阅读、写作、决策、沟通等一切需要理性思考的场景。

**一句话定义：** 用11个关键问题层层盘问任何一个论证，直到分清真金与废铁。

---

## 核心流程：11个关键问题

面对任何论证，按顺序问以下问题：

```
┌─────────────────────────────────────────────────┐
│  第一步：理解论证（我在听什么？）                    │
│  ┌─────────────────────────────────────────────┐│
│  │ Q1 论题和结论是什么？                         ││
│  │ Q2 理由是什么？                               ││
│  │ Q3 哪些词语意思不明确？                        ││
│  │ Q4 价值观假设和描述性假设是什么？               ││
│  └─────────────────────────────────────────────┘│
│                                                   │
│  第二步：评估论证（我该信吗？）                     │
│  ┌─────────────────────────────────────────────┐│
│  │ Q5 论证中有没有谬误？                         ││
│  │ Q6 直觉/个人经历/典型案例/证言/专家意见效力？   ││
│  │ Q7 个人观察/研究报告/类比的效力？              ││
│  │ Q8 有没有替代原因？                           ││
│  │ Q9 数据有没有欺骗性？                         ││
│  │ Q10 有什么重要信息被省略了？                   ││
│  │ Q11 能得出什么合理的结论？                     ││
│  └─────────────────────────────────────────────┘│
└─────────────────────────────────────────────────┘
```

## 技能详解与操作清单

### 技能1：定位论题和结论

**何时用：** 每次面对一篇论证性文章或演说时。

**操作步骤：**
1. 找到论题：看标题、第一段，或问"作者在对什么进行评价/争论？"
2. 区分论题类型：
   - 描述性（世界是什么样的）→ 寻找证据
   - 规定性（世界应该是什么样的）→ 寻找价值判断
3. 找到结论：问"作者想要证明什么？"
4. 查找指示词：因此、由此可知、表明、显示、问题的实质是
5. 排除非结论：例证、数据、定义、背景材料、证据都不是结论

**快速检验：** 能否用一句话概括"作者希望我相信什么？"

---

### 技能2：挖掘理由

**何时用：** 找到结论后立即进行。

**操作步骤：**
1. 对结论问"为什么？"——回答就是理由
2. 查找指示词：因为、由于、鉴于、研究显示、第一/第二/第三
3. 区分理由类型：事实、研究成果、统计数据、专家意见、类比等
4. 画出论证结构：结论 ← 理由1 + 理由2 + …
5. 警惕"逆向逻辑"：先有理由后有结论，不是先定结论再编理由

**快速检验：** 如果理由成立，结论就一定成立吗？有没有隐藏的假设？

---

### 技能3：识别歧义

**何时用：** 在决定是否接受结论之前。

**操作步骤：**
1. 找出论题、结论、理由中的关键词和短语
2. 对每个关键词问："它还有没有其他意思？"
3. 注意抽象词：自由、公平、责任、质量——越抽象越容易有歧义
4. 用反串法：如果采取对立立场，是否会用不同方式定义这些词？
5. 在脑中想象画面：想象不出 → 可能有歧义
6. 检查感情色彩：这个词是否在激发情感而非传递信息？

**判定标准：** 如果一个词有两种以上含义，每种含义放到语境中都说得通，且采用不同含义会显著影响论证的效度 → 你找到了重要歧义。

---

### 技能4：发掘隐藏假设

**何时用：** 理由看起来有道理但总觉得"哪里不对"的时候。

**操作步骤：**

**价值观假设（规定性论证）：**
1. 识别对立的价值观（如自由 vs 安全、效率 vs 公平）
2. 判断作者偏向哪一方
3. 检查作者背景：身份决定倾向
4. 反串：如果采取相反立场会关心什么？
5. 分析后果：哪些后果对作者来说最重要？为什么？

**描述性假设：**
1. 思考理由和结论之间的鸿沟
2. 问："如果理由成立，要得出这个结论还需要什么成立才行？"
3. 检查常见假设：世界是公正的？过去能预测未来？人是理性的？
4. 区分"定义性假设"：对多义词想当然地认定某一个意思

---

### 技能5：揪出谬误

**何时用：** 论证看起来有道理但结论"太方便"的时候。

**操作步骤：**
1. 检查假设是否荒谬——如果需要错误假设才能成立 → 谬误
2. 检查是否转移注意力——作者在谈论真正的问题吗？
3. 检查是否循环论证——理由只是换了说法重述结论吗？
4. 对照常见谬误清单逐项检查

**速查表：**
```
看到人身攻击 → 攻击送信人，不是讨论信息
看到连锁灾难假设 → 滑坡谬误
看到"大家都说" → 诉诸公众
看到"不能完全解决" → 追求完美方案
看到"不是A就是B" → 虚假两难
看到强烈情感词 → 诉诸感情/光环效应
看到话题偏移 → 转移话题
看到结论自己证明自己 → 循环论证
看到贴标签代替解释 → 乱扣帽子
看到歪曲对方观点 → 稻草人谬误
```

---

### 技能6：评估证据（上）

**何时用：** 论证声称基于"事实"或"证据"时。

**证据可靠性阶梯（低→高）：**

```
直觉 → "我就是知道" → 私密、无法验证
个人经历 → "我朋友…" → 样本太小，以偏概全
典型案例 → 生动故事 → 诉诸情感，不具代表性
当事人证言 → 名人推荐 → 选择性、有利益
专家意见 → "研究表明…" → 需查专业度和利益
```

**对每种证据的追问：**
- 直觉：有没有其他类型的证据支撑？
- 个人经历：这个经历有代表性吗？
- 典型案例：是否用情感掩盖了证据不足？
- 当事人证言：作证者有没有利益关系？有没有被省略的信息？
- 专家意见：专家对这个主题有多少专业知识？有没有利益冲突？其他专家同意吗？

---

### 技能7：评估证据（下）

**个人观察：** 观察受价值观、偏见、期待、环境过滤。多个观察者、最佳环境、无偏见 = 更可信。

**研究报告：**
1. 质量如何？是否经过同行评审？
2. 能否被复制？
3. 结论是否被过度概括？
4. 样本：数量够大？覆盖够广？随机性够强？
5. 测量方法：如何定义和测量关键概念？满意吗？
6. 调查问卷：问题有没有措辞偏见？语境有没有影响？

**类比：**
1. 找出两事物的相同点和不同点
2. 判断相同点和不同点的相关性
3. 相关共同点多 + 相关不同点少 = 强势类比

---

### 技能8：寻找替代原因

**何时用：** 看到因果关系断言时。

**操作步骤：**
1. 看到"导致""引起""因为""原因" → 立刻警觉
2. 问自己：
   - 还有没有其他方法可以解读这个证据？
   - 还有什么别的东西可能引发这个结果？
   - 从另一个角度看会怎样？
3. 对相关性证据，列出4种解释：A引起B、B引起A、C引起A和B、A和B相互影响
4. 对组间差异，问：两组还有什么其他不同？
5. 记住：大部分事件不是单一原因，而是多种原因共同作用

**因果谬误速查：**
- 相关性 ≠ 因果关系
- 先后发生 ≠ 因果关系（事后归因）
- 基本归因错误：高估个人因素，低估环境因素

---

### 技能9：识破数据骗局

**操作步骤：**
1. 问数据怎么得来的？来源可靠吗？
2. 平均值是平均数、中位数还是众数？差距可能很大！
3. 知道全距和分布吗？
4. 数据证明的和结论说的是同一件事吗？
5. 绝对数 vs 百分比：两者都需要
6. 自己从数据推导结论，和作者的对比

**速查：**
- "平均收入180万" → 中位数可能才77万
- "增长75%" → 从4家到7家？
- "70%的失窃是电子产品" ≠ "坐地铁70%概率被偷"

---

### 技能10：搜寻省略信息

**何时用：** 论证看起来"好得不像真的"时。

**操作步骤：**
1. 提醒自己：几乎每个信息都有目的
2. 对任何论证问：
   - 反对的人会说什么？
   - 有没有被遗漏的定义？
   - 有没有遗漏的价值观/视角？
   - 有没有缺失的数字/数据？
   - **提议的行动有什么潜在的长期负面效果？**
   - 社会哪部分没有受益？谁蒙受损失？
3. 坚持要具体数字：看到"更大""更多""更高效" → 追问具体数值

---

### 技能11：得出多元结论

**何时用：** 准备做出最终判断时。

**操作步骤：**
1. 拒绝二分式思维（非此即彼）
2. 添加限定条件：
   - 什么时候这个结论成立？
   - 什么情况下成立？
   - 为什么/为了什么目的成立？
3. 使用条件句："如果…那么…"——不用假装知道我们不知道的
4. 重新表述问题：从"该不该做X？" → "我们该怎样处理Y？"
5. 列出多个备选结论，选择最契合最好理由的那个

## 日常练习方法

### 每日习惯
1. **阅读时**：对每篇文章至少找出论题、结论和一个关键歧义
2. **对话时**：练习"你是不是说…？"确认理解
3. **决策时**：写下至少2个替代原因和1个被省略的信息
4. **写作时**：确保结论明确、理由先行、术语无歧义

### 社交沟通策略
- 大声说出你的好奇心，而非摆出"抓住你了"的态度
- 弥合分歧：把对方最好的理由和自己的放一起，找新结论
- 寻找共同价值观作为对话起点
- 语气升温时提醒自己：目的是学习，不是赢辩论

### 批判性思维者的四大价值观
1. **自主决断**：从最多可能性中选择
2. **好奇心**：真心想了解不同观点
3. **谦恭有礼**：承认自己也会犯错
4. **尊重严密论证**：不论出处，只论逻辑

## 场景路由（Scene Routing）

根据你当前面对的问题类型，自动路由到最相关的技能组合：

### 路由1：论证分析（Argument Analysis）
**触发词：** "他说的对吗""这个论证成立吗""帮我分析这段话""这篇文章的逻辑"
```
输入 → Q1 定位论题和结论
     → Q2 挖掘理由
     → Q3 识别歧义
     → Q4 发掘隐藏假设
     → Q11 得出多元结论
输出 → 论证结构图：结论 ← 理由 ← 假设，附歧义标注
```
**操作要点：**
- 先画出论证骨架再评价，不要边读边判
- 理由不等于结论，必须分别列出
- 找不到假设时，试填空："要从理由推出结论，必须先相信____"

### 路由2：证据评估（Evidence Evaluation）
**触发词：** "这个证据可靠吗""数据说明了什么""研究可信吗""专家说的能信吗"
```
输入 → Q6 评估证据来源（直觉→专家，逐级升级）
     → Q7 评估研究/观察/类比质量
     → Q9 识破数据骗局
     → Q10 搜寻省略信息
输出 → 证据可信度评级（强/中/弱/不可信）+ 缺失证据清单
```
**操作要点：**
- 证据可靠性是阶梯式的，不能跳级（个人经历不能替代研究）
- 数据必须同时看绝对数和百分比
- 省略信息往往是决定性的——主动追问"反对者会怎么说"

### 路由3：逻辑谬误检测（Logical Fallacy Detection）
**触发词：** "这话哪里不对""是不是在偷换概念""感觉有问题但说不出来""对方在诡辩吗"
```
输入 → Q5 对照谬误速查表逐项排查
     → Q3 检查关键词歧义（偷换概念）
     → Q8 检查因果谬误（相关≠因果）
     → Q10 检查选择性省略（误导性呈现）
输出 → 谬误类型 + 具体位置 + 修正建议
```
**操作要点：**
- 先查假设是否荒谬，再查是否转移注意力，最后查是否循环论证
- 一个论证可能同时犯多种谬误
- 指出谬误时给出修正方向，不要只说"你错了"

### 路由4：假设质疑（Assumption Questioning）
**触发词：** "他默认了什么""前提是什么""为什么觉得理所当然""这个假设对吗"
```
输入 → Q4a 价值观假设：识别对立价值观，判断作者偏向
     → Q4b 描述性假设：找理由与结论之间的鸿沟
     → Q8 替代原因：假设不同则因果不同
     → Q3 歧义：假设往往藏在多义词的默认含义中
输出 → 假设清单（显性/隐性）+ 每条假设的可信度 + 推翻后果
```
**操作要点：**
- 隐性假设是最危险的，因为连作者自己都意识不到
- 从后果反推：如果假设不成立，结论会怎样？
- 常见隐性假设模板：世界是公正的 / 过去能预测未来 / 人是理性的 / 这个定义是唯一的

### 场景速查表

| 场景 | 主路由 | 辅助路由 | 重点提醒 |
|------|--------|----------|----------|
| 读新闻/社论 | 论证分析 | 证据评估 | 省略信息往往是关键 |
| 评估广告/营销 | 谬误检测 | 证据评估 | 情感词 = 红旗信号 |

## 失败模式与反模式（Failure Modes & Anti-Patterns）

### 失败模式：为什么批判性思维会失效

| 失败模式 | 表现 | 根因 | 修复 |
|----------|------|------|------|
| **确认偏误循环** | 只找支持自己观点的证据 | 情感先于理性 | 强制自己为对方找3个好理由 |
| **分析瘫痪** | 问题太多，无法得出结论 | 把11个问题全部应用到每个小事 | 区分"值得深挖"和"快速判断"——日常小事走速查即可 |
| **武器化提问** | 用问题攻击对方而非理解对方 | 把技能当辩论武器而非思考工具 | 每次提问前默念：目的是理解，不是赢 |
| **假中立** | 声称"两边都有道理"逃避判断 | 把谦恭误解为不表态 | 谦恭≠无立场，用条件句表态："如果X成立，那么Y更好" |
| **过度怀疑** | 对所有信息都不信 | 把批判性思维等同于否定一切 | 怀疑是手段不是目的，目标是找到更可靠的信念 |
| **表面提问** | 问了11个问题但答案都是"看起来还行" | 没有深入追问第二层 | 每个问题至少追问两层：第一层是"是什么"，第二层是"为什么这样" |

### 反模式：常见误用方式

```
❌ 反模式1："抓住你了"式提问
   表现：对方刚说完就指出谬误，语气像审讯
   ✅ 正确：先复述理解，再提出疑问，用"我注意到…"开头

❌ 反模式2：清单式机械应用
   表现：对每个信息都走完11步，耗时耗力
   ✅ 正确：根据场景路由选3-5个关键问题，深度优于广度

❌ 反模式3：只破不立
   表现：能指出10个问题但提不出1个替代方案
   ✅ 正确：每次批评后必须给出至少1个改进方向

❌ 反模式4：选择性批判
   表现：对不喜欢的观点严格审查，对喜欢的放水
   ✅ 正确：对自己同意的观点加倍严格——你越想相信的越要审视

❌ 反模式5：忽视情感维度
   表现：只分析逻辑，不考虑对方感受和语境
   ✅ 正确：先建立信任再提问，批判性思维是合作不是对抗
```

## 检查点（Checkpoints）

### 入口检查：是否需要启动批判性思维？

```
□ 这个信息会影响我的决策或行动吗？
  → 是 → 启动批判性思维
  → 否 → 可以先跳过（但记住以备后用）
□ 这个论证的利害关系有多大？
  → 高利害 → 走完整流程
  → 低利害 → 走速查模式
□ 我是否对这个话题有强烈的情感倾向？
  → 是 → ⚠️ 先确认偏误风险，强制为对方找好理由
```

### 过程检查：每步是否做到位？

```
理解阶段：
□ 我能用一句话说清结论吗？
□ 我能列出至少2个理由吗？
□ 我找出了至少1个关键歧义吗？
□ 我识别了至少1个隐藏假设吗？

评估阶段：
□ 我对照了谬误速查表吗？
□ 我评估了证据的可靠性等级吗？
□ 我至少想了2个替代原因吗？
□ 我追问了"什么被省略了"吗？
□ 我检查了数据的呈现方式吗？

结论阶段：
□ 我的结论是否用了条件句（如果…那么…）？
□ 我是否列出了至少2个备选结论？
□ 我是否明确了结论的适用范围和限制？
```

### 出口检查：结论是否可靠？

```
□ 经过完整审视后，我仍然相信这个结论吗？
□ 如果有新证据出现，我愿意修改结论吗？
□ 我能否向一个持反对意见的人解释我的推理？
□ 我是否避免了"非此即彼"的二分式结论？
□ 我是否对自己同意的观点也做了同样严格的审查？
```

### 快速决策检查点（日常小事用）

```
□ 来源是什么？（人/机构/媒体）
□ 证据类型是什么？（直觉/经历/研究/数据）
□ 有没有明显的谬误？（扫一眼谬误速查表）
□ 有什么明显被省略了？（花10秒想）
→ 以上4步在30秒内完成，足够应对日常判断
```

## 记忆口诀

**论理歧假，谬证替数省多**

- **论**题和结论是什么？
- **理**由是什么？
- 哪些词语有**歧**义？
- 什么**假**设？
- 有没有**谬**误？
- **证**据效力如何？
- 有没有**替**代原因？
- **数**据有没有欺骗？
- 有什么被**省**略了？
- 能得出什么**多**元结论？
