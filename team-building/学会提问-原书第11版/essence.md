# 学会提问（原书第11版）精华

**作者：** 尼尔·布朗、斯图尔特·基利
**核心主题：** 批判性思维——系统性提问的艺术

---

## 一、核心理念

### 批判性思维的三个维度
1. **意识**：认识到一整套环环相扣的关键问题
2. **能力**：在适当时机以适当方式提出并回答这些问题
3. **渴望**：积极主动地使用这些关键问题的强烈意愿

### 两种思维方式
- **海绵式思维**：被动吸收信息，不做判断。优点是知识积累快，致命缺陷是无法筛选信息的可信度。
- **淘金式思维**：主动与信息互动，不断提问，披沙拣金。核心是"互动性参与"——在作者和读者之间展开对话。

### 弱势 vs 强势批判性思维
- **弱势**：用批判性思维捍卫自己当前的信念（思想封闭）
- **强势**：用批判性思维评估所有断言，包括自己的信念（思想开放）

> "当我们改变自己的想法，接受一个更好的观点，我们应为此感到自豪，我们抵制住了诱惑，没有死心塌地去维护自己长期以来的信念。"——弗朗西斯·培根

## 二、批判性思维的障碍（减速带）

### 1. 思考过快（系统1思维）
- 大脑依赖"快思考"仓促决断
- 解药：**放慢**，训练"系统2思维"
- 核心习惯：问自己"我为什么要像现在这样想？"

### 2. 刻板印象
- 断言某人属于特定集体就必定具有某些特征
- 绕开了客观评价的艰难过程，提前关闭思路

### 3. 认知偏见
- **晕轮效应**：从一个人的某个优点/缺点推断其所有方面
- **信念固着**：死守已有信念，只接受确认性证据（确认性偏见）
- **可得性启发**：只根据手边最容易获得的信息形成结论
- **近因效应**：最新看到的信息权重过大
- **答非所问**：用自己的问题替代别人的问题

### 4. 自我中心
- 赋予自我世界以中心地位
- "知识的诅咒"：以为别人和我们知道的一样多

### 5. 一厢情愿（批判性思维最大的障碍）
- 希望什么是真的，就直接宣布它是真的
- 奇迹式思维：用奇迹解释无法理解的事物
- 治疗良药：积极主动使用批判性思维方式

## 三、论证的基本结构

### 论题
- **描述性论题**：关于世界是什么样的（"音乐学习是否提高数学能力？"）
- **规定性论题**：关于世界应该是什么样的（"该不该立法禁止……？"）

### 结论
- 作者或演说者希望你接受的信息
- 寻找线索：指示词（因此、由此可知、表明、证明）、位置（开头/结尾）、作者背景

### 理由
- 用来支撑结论的信念、证据、隐喻、类比和其他陈述
- 指示词：因为、由于、鉴于、研究显示、第一/第二/第三
- **先有理由，后有结论**——警惕"逆向逻辑"

> 论证 = 结论 + 理由。薄弱的理由必然导致薄弱的论证。

## 四、识别歧义

### 核心原则
- 词语有多重含义，在论证语境中含义不明确就是歧义
- 越抽象的词越容易产生歧义

### 寻找歧义的方法
1. 检查论题中的关键词
2. 在理由和结论中寻找关键术语
3. **反串法**：站在对立立场，看是否会用不同方式定义某些词
4. 在脑中想象词语代表的具体画面——想象不出则可能有歧义

### 歧义与感情色彩
- "增税" vs "免税"、"死亡税" vs "遗产税"——同义词激发不同情感反应
- 附加感情色彩的术语会让思维暂时性短路

> **谁想说服你，谁就要负责解释清楚。**

## 五、发现假设

### 假设的特征
1. 隐藏或没有明说出来
2. 作者认为理所当然
3. 对决定结论有较大影响
4. 可能有欺骗性

### 价值观假设
- 认为某些相互对立的价值观中一个比另一个更重要
- 典型冲突：忠诚 vs 诚实、竞争 vs 合作、自由 vs 安全、秩序 vs 言论自由
- 会随着情境改变而改变，不是固定的

**寻找线索：**
- 立论者的背景（他的身份决定了他的价值倾向）
- 可能发生的后果（为什么作者认为某些后果更可取？）
- 反串（如果采取相反立场会关心什么？）

### 描述性假设
- 对世界过去、现在或未来是什么样的信念（没有明说）
- 常见类型：
  - 发生在人身上的事主要都是自己选择的结果
  - 这个世界是公正的
  - 因为以前发生过，今后还会发生
  - 我的世界就是宇宙的中心

**寻找线索：**
- 不断思考理由和结论之间的鸿沟
- 寻找支撑理由的未明说想法
- 站在作者和反对者的立场

## 六、论证中的谬误

### 三类主要花招
1. 需要明显错误的假设才能成立的论证
2. 把无关信息弄得好像和结论有关
3. 看似找证据，实际预设了结论

### 常见谬误清单
| 谬误 | 本质 | 识别词 |
|------|------|--------|
| **人身攻击** | 攻击送信人而非信息 | — |
| **滑坡谬误** | 假设一系列不可控的连锁事件 | "如果…那么接下来…" |
| **追求完美方案** | 因不能完全解决就拒绝方案 | "不能彻底解决就没用" |
| **诉诸公众** | 大家都认为所以是对的 | "大家都…" |
| **诉诸可疑权威** | 引用不具备专业资格的权威 | "专家说…" |
| **诉诸感情** | 用情感代替证据 | 恐惧、希望、爱国 |
| **稻草人谬误** | 歪曲对方观点使其易受攻击 | — |
| **虚假两难** | 只展示两种选择 | "不是…就是…" |
| **乱扣帽子** | 给行为贴标签冒充解释 | "他就是个…" |
| **光环效应** | 用模糊的品德词汇赢得认同 | "伟大的""神圣的" |
| **转移话题** | 引入不相干话题转移注意力 | — |
| **循环论证** | 结论自己证明自己 | — |
| **事后归因** | 先后发生≠因果关系 | "在那之后" |

## 七、评估证据

### 证据类型与可靠性（由低到高）

| 证据类型 | 主要问题 |
|----------|----------|
| **直觉** | 私密性，别人无法判断可靠性 |
| **个人经历** | 样本太小，以偏概全 |
| **典型案例** | 诉诸情感，分散对证据价值的注意 |
| **当事人证言** | 选择性、个人利益、省略信息 |
| **专家意见** | 专家也会犯错、有偏见、意见不一 |
| **个人观察** | 受价值观、偏见、期待过滤 |
| **研究报告** | 最可靠，但需验证样本、方法、复制性 |
| **类比** | 看相关共同点与不同点的比例 |

### 评估研究报告的8个要点
1. 研究质量参差不齐
2. 研究成果常常互相矛盾——单一研究不够
3. 研究发现支持而非"证明"结论
4. 研究人员也有偏见和利益
5. 演说者常歪曲或简化研究结论
6. 研究"事实"会随时间改变
7. 人工环境可能扭曲结果
8. 经济利益等需求可能影响结果

### 评估类比的标准
- 两事物之间相同和不同的方式
- 相同点和不同点的**相关性**
- 强势类比 = 大量相关共同点 + 极少相关不同点

## 八、替代原因

### 核心教训
- 很多事件都可以用多种替代原因解释
- 专家常只提供自己喜欢的原因
- 替代原因越多，初始解释的确信度越低
- 想出替代原因是个创造性过程

### 相关性 ≠ 因果关系
任何相关性至少有4种解释：
1. 甲引起乙
2. 乙引起甲
3. 第三种因素丙同时引起甲和乙
4. 甲和乙相互影响

### 常见因果谬误
- **因果混淆**：将起因和结果混淆
- **忽略常见原因**：看不出第三因素在起作用
- **事后归因**：时间先后不等于因果
- **过度简化因果**：只强调一个原因而忽略多因素
- **基本归因错误**：高估个人倾向，低估环境因素

## 九、数据的欺骗性

### 三种平均值
- **平均数（mean）**：总和÷数量（易被极端值拉高）
- **中位数（median）**：排序后最中间的值
- **众数（mode）**：出现频率最高的值

> 还要关注**全距**和**数值分布**——平均值掩盖了个体差异。

### 数据欺骗的常见手法
1. **不知来历和带有偏见的数据**——问"怎么得来的？"
2. **用一件事的结论证明另一件事**——数据≠结论
3. **通过省略信息欺骗**——只给绝对数不给比率（或反之）
4. **用百分比制造错觉**——基数不明的百分比毫无意义

## 十、被省略的信息

### 为什么省略信息在所难免
1. 时空限制
2. 注意力持续时间有限
3. 立论者知识不全面
4. 有意欺骗
5. 价值观、信仰和态度不同

### 识别省略信息的提问清单
- 反对的人会提供什么理由？
- 如果关键词用另一种方式定义会怎样？
- 不同的价值观会产生什么论证？
- 这些"事实"的来源是什么？
- 提议的行动有什么**潜在的长期负面效果**？
- 社会的哪部分没有受益？谁蒙受损失？

## 十一、得出合理结论

### 摆脱二分式思维
- 重大问题很少能用"是"或"不是"回答
- 添加限定条件：**什么时候？什么地方？为什么/为了什么？**

### 条件句的衍生能力
- "如果税收减免针对低收入人群，那么…"
- 条件句让我们不用假装知道我们本来不知道的一切
- 保持谦卑：承认我们的知识总是不完全的

### 备选解决方法作为结论
- 把问题从"该不该做X？"重新表述为"我们该怎样处理Y问题？"
- 这会创造出多种可能的结论

> **理由很少只有一种含义。最值得相信的结论是那些最能契合最好的理由的结论。**

---

## 十二、批判性思维的价值观

- **自主决断**：从最多可能性中选择
- **好奇心**：多听多看，真正用心去听去看
- **谦恭有礼**："我唯一知道的事就是我一无所知"
- **尊重严密的论证**：不偏不倚信赖说理透彻的人

### 保持对话的策略
1. "你是不是说……？"确保理解对方
2. 问有没有什么证据可以让他改变想法
3. 弥合分歧，寻找共同价值观
4. 好奇心再强也要体贴他人
5. 表情和动作都要谦恭

---

## 一句话总结

> **批判性思维不是为了赢得辩论，而是为了追求更好的结论、更好的信念、更好的决定。它的本质是：在你思考重大事情的时候一定要"放慢"。**
