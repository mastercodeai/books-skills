## 🧭 场景路由（Scene Routing）

根据你当前面对的问题类型，选择最相关的技能组合：

| 场景 | 核心技能 | 辅助技能 |
|------|---------|---------|
| **投资决策** | ③能力圈自检 → ⑥等待好球 → ①多元思维模型 → ④心理检查清单 | ②逆向思考、⑧双轨分析 |
| **商业分析** | ①多元思维模型 → ⑧双轨分析 → ④心理检查清单 | ②逆向思考、③能力圈、⑤五何沟通 |
| **人生/职业抉择** | ②逆向思考 → ④心理检查清单 → ③能力圈自检 | ⑦终身学习、⑨说服法则 |
| **学习新领域** | ⑦终身学习操作系统 → ③能力圈自检 → ⑩司机知识自检 | ①多元思维模型 |
| **说服/沟通** | ⑨说服黄金法则 → ⑤五何沟通法 → ④心理检查清单 | ⑩司机知识自检 |
| **复盘/反思** | ②逆向思考 → ④心理检查清单 → ⑧双轨分析 | ①多元思维模型 |

**路由规则**：
1. 先确定场景类型
2. 按顺序执行核心技能，每步产出明确结论
3. 用辅助技能交叉验证
4. 如果多个场景重叠，以最紧迫的场景为主

## 1. 多元思维模型决策法

**何时用**：面对复杂决策、投资判断、商业分析

**操作步骤**：
1. 列出问题涉及的所有学科维度（经济、心理、数学、生物、工程等）
2. 从每个学科提取1-2个最相关的模型
3. 用多个模型同时分析同一问题
4. 寻找**lollapalooza效应**——多个因素同向叠加的极端结果
5. 如果只能用一个模型解决问题 → 警惕"铁锤人倾向"

**关键检查**：
- [ ] 我是否只从一个学科角度思考？
- [ ] 有没有反面证据被我忽略？
- [ ] 这个结果是否由多种心理倾向共同造成？

**失败模式**：
- ⚠️ 模型罗列但不整合：列出5个学科但各自独立分析，没有交叉验证
- ⚠️ 伪多元：表面用多个模型，实际都指向同一结论（确认偏误伪装）
- ⚠️ 过度复杂化：简单问题强套多元模型，浪费精力且模糊焦点

**反模式**：
- ❌ "我用经济学模型分析过了" → 只用一个学科
- ❌ 列出模型后直接跳到结论 → 跳过交叉验证步骤
- ❌ 找到lollapalooza效应就兴奋地行动 → 没有检查反面证据

## 2. 逆向思考清单

**何时用**：制定计划、评估决策、避免失败

**操作步骤**：
1. 明确你的目标（如：投资成功/幸福人生/企业增长）
2. **反过来问**：什么会导致失败/痛苦/衰败？
3. 列出所有可能导致失败的原因
4. 逐一检查自己是否正在犯这些错误
5. 优先**避免已知的愚蠢**，而非追求天才式创新

**芒格常用反问**：
- "我将来会死在什么地方？" → 不去那里
- "怎样才能损害这件事？" → 避免做那些事
- "什么会让这个投资永久亏损？" → 排除这些因素

**失败模式**：
- ⚠️ 逆向思考后没有行动：列出了失败原因但没有制定规避措施
- ⚠️ 只逆向不正向：过度关注风险导致完全不行动（分析瘫痪）
- ⚠️ 失败原因清单太泛：如"管理不善"而非"CEO有3次欺诈记录"

**反模式**：
- ❌ "想到了风险但觉得不会发生" → 概率忽视
- ❌ 列完清单就结束 → 没有转化为具体行动项
- ❌ 只问一次就满足 → 重要决策应多次、多角度逆向

## 3. 能力圈自检法

**何时用**：面对新机会、评估是否参与某事

**操作步骤**：
1. 问自己：这个领域我是否比99%的人了解得更深？
2. 我能否清楚说出这个领域的**边界**在哪里？
3. 我能否比最聪明的反对者更好地反驳自己的判断？
4. 如果不能 → 归类为"太难"，放弃

**决策三分法**：
- ✅ 可以（有明确优势）
- ❌ 不行（明确劣势）
- ⚠️ 太难（默认选项，大多数情况）

**失败模式**：
- ⚠️ 虚假自信：觉得自己懂，但无法通过"反驳测试"
- ⚠️ 圈子漂移：去年在圈内，今年行业变化后仍在圈内自居
- ⚠️ 过度保守：明明有优势却因恐惧归类为"太难"

**反模式**：
- ❌ "我在这个行业工作了10年" → 经验年限≠能力深度
- ❌ "大家都说这个好" → 外部共识≠你的能力圈
- ❌ "我已经研究了一个月" → 研究时长≠真正理解

**检查点**：
- [ ] 我能否画出这个领域的知识地图并标注边界？
- [ ] 我能否列举3个这个领域的反直觉事实？
- [ ] 如果一个专家质疑我，我能回应哪些？哪些回应不了？

## 4. 人类误判心理检查清单

**何时用**：重大决策前、评估他人建议、自我反思

**25项检查**（决策前逐项扫描）：

```
□ 1. 激励机制：谁因这个决定受益？他的建议是否受利益驱动？
□ 2. 喜爱偏差：我是否因喜欢某人而忽略其缺点/错误？
□ 3. 憎恨偏差：我是否因讨厌某人而忽略其优点/正确建议？
□ 4. 仓促决定：我是否因困惑/压力而急于下结论？
□ 5. 一致性陷阱：我是否因已有的承诺/身份而拒绝改变？
□ 6. 好奇心：我是否真正理解了问题的本质？
□ 7. 公平性：各方是否得到合理对待？
□ 8. 妒忌：我的决定是否受他人成功的刺激？
□ 9. 回馈：我是否因对方的小恩惠而做出不当让步？
□ 10. 联想：我是否将无关因素错误关联？
□ 11. 否认：我是否在逃避痛苦的现实？
□ 12. 自视过高：我是否高估了自己的能力/判断？
□ 13. 过度乐观：我是否忽略了负面概率？
□ 14. 损失厌恶：我是否因害怕失去而做出非理性决定？
□ 15. 从众：我是否因大家都这么做而跟风？
□ 16. 对比效应：我的参照系是否合理？
□ 17. 压力：当前压力水平是否影响了我的判断？
□ 18. 易得性：我是否过度依赖容易想到的信息？
□ 19. 生疏：我是否忽略了长期不用的技能/知识？
□ 20. 化学依赖：是否有物质影响了判断？
□ 21. 衰老：年龄是否影响了认知能力？
□ 22. 权威：我是否盲目信任权威人物？
□ 23. 废话：讨论中是否有大量无意义信息干扰？
□ 24. 理由充分：给出的理由是否真正合理？
□ 25. 叠加效应：以上哪些倾向在同时起作用？
```

**失败模式**：
- ⚠️ 走过场：快速勾完25项但没有真正反思（自我欺骗）
- ⚠️ 只检查别人：用清单找别人的偏误，忽略自己的
- ⚠️ 发现偏误后合理化："虽然有损失厌恶，但这次情况不同"

**反模式**：
- ❌ "我一向很理性" → #12 自视过高
- ❌ "这次不一样" → #5 一致性陷阱 + #11 否认
- ❌ "专家都这么说" → #22 权威偏误
- ❌ "我已经投入这么多了" → #14 损失厌恶 + #5 一致性陷阱

**检查点**：
- [ ] 哪3项偏误对我最危险？（应长期标注）
- [ ] 我是否有"偏误搭档"——能指出我盲点的人？
- [ ] 上次重大决策中我发现了哪些偏误？采取了什么措施？

## 5. "五何"沟通法

**来源**：卡尔·布劳恩工程公司

**规则**：每次沟通必须说明——**何人**因**何故**在**何时何地**做了**何事**

**应用场景**：
- 写邮件/报告时：必须包含原因
- 布置任务时：必须解释为什么
- 做决策时：必须记录决策理由

**效果**：人们理解原因后更愿意执行，即使不理解也会更倾向于服从

**失败模式**：
- ⚠️ 缺少"何故"：只说做什么，不说为什么 → 执行力打折
- ⚠️ 原因太抽象："为了公司发展" → 应具体到可验证的理由
- ⚠️ 只在大事用：日常小事不说明原因 → 团队养成"不问为什么"的习惯

## 6. 投资"等待好球"法

**何时用**：投资决策

**核心规则**：
1. 想象你有一张只有**20格**的考勤卡，一生只能投资20次
2. 大部分时间**什么也不做**
3. 只在发现**严重定价错误**时出手
4. 出手时**集中投入**，不要分散
5. 买入后**长期持有**，享受复利和税收优势

**好球的特征**：
- 企业有持久竞争优势（护城河）
- 管理层正直能干
- 价格远低于内在价值
- 在你的能力圈之内

**反面教材**：
- 每年花1-2%请投资顾问频繁操作
- 试图在每个行业都赚钱
- 因为股价上涨了一点点而错过（沃尔玛100亿美元教训）

**失败模式**：
- ⚠️ 永远在等待：用"等待好球"合理化惰性，错过真正的机会
- ⚠️ 降低标准：等太久后把"一般机会"强行定义为"好球"
- ⚠️ 集中过度：单次投入过大导致无法承受黑天鹅事件
- ⚠️ 买入即遗忘：不跟踪企业基本面变化，"长期持有"变成"盲目持有"

**反模式**：
- ❌ "这个机会难得一见" → 可能是 #13 过度乐观
- ❌ "我已经等了两年了" → #5 一致性陷阱 + #14 损失厌恶（沉没成本）
- ❌ "分散投资才安全" → 如果你真的在能力圈内，分散反而降低收益

**检查点**：
- [ ] 这个机会是否同时满足护城河+管理层+价格+能力圈四个条件？
- [ ] 如果买入后明天跌50%，我是否有信心加仓？
- [ ] 我的20格卡已经用了几格？还剩几格？

## 7. 终身学习操作系统

**日常习惯**：
- 每天起床争取比昨天**更聪明一点**
- **大量阅读**：传记、科学、心理学、经济学（芒格：聪明人没有不每天阅读的）
- 与"已逝的伟人交朋友"——通过阅读与杰出思想建立联系
- 每晚睡觉时比早晨聪明一点点

**学习原则**：
- 让知识在头脑中形成**思维框架**，而非死记硬背
- 掌握后必须**持续练习**（钢琴家一天不练琴就退步）
- 从**多种学科**学习，不局限于专业领域
- 不要害怕承认"我不知道"

**失败模式**：
- ⚠️ 阅读≠学习：读完不思考、不应用、不输出 → 信息过客
- ⚠️ 广度陷阱：什么都学一点，什么都不精 → 司机知识
- ⚠️ 输入成瘾：不断学习新东西但从不实践 → 知识囤积症

**反模式**：
- ❌ "我今年读了100本书" → 数量≠质量，读了什么比读了多少重要
- ❌ "这个领域我不懂所以不碰" → 逆向能力圈过度保守
- ❌ "我每天都在学习" → 学习需要输出和应用，不是阅读表演

**检查点**：
- [ ] 上周学到的最重要的一点是什么？我用了吗？
- [ ] 我的阅读是否覆盖了至少3个不同学科？
- [ ] 我能否向别人清晰解释最近学到的一个概念？

## 8. 双轨分析法

**何时用**：分析任何决策

**轨道一：理性分析**
- 真正控制涉及的利益的因素是什么？
- 真正的机会在哪里？
- 像打桥牌一样分析

**轨道二：潜意识分析**
- 我的大脑正在自动形成哪些潜意识因素？
- 哪些心理倾向在影响我的判断？
- 这些潜意识因素在当前情况下是否失灵？

**失败模式**：
- ⚠️ 只走轨道一：理性分析完美但忽略了自己的情绪状态
- ⚠️ 只走轨道二：过度关注心理偏误导致无法做决定
- ⚠️ 两轨不整合：分别得出结论但没有交叉验证

**检查点**：
- [ ] 轨道一和轨道二的结论是否一致？不一致时哪个更可信？
- [ ] 我当前的情绪状态是什么？它是否在影响我的分析？
- [ ] 如果我完全不带感情地看待这个决策，结论会不同吗？

## 9. 说服的黄金法则

**富兰克林原则**："诉诸利益，而非诉诸理性"

**操作要点**：
- 想说服别人做某事 → 告诉他**不做会失去什么**
- 即使动机高尚，也要用**利益框架**表达
- 人们最在意的是自己的利益，利用而非对抗这一点

**失败模式**：
- ⚠️ 只用利益操纵：短期有效但长期损害信任
- ⚠️ 忽略对方价值观：假设所有人只在乎物质利益
- ⚠️ 自己也被说服：在说服别人时反被对方的利益框架套住

**反模式**：
- ❌ "从逻辑上讲你应该…" → 诉诸理性而非利益
- ❌ "这对大家都好" → 太抽象，对方感受不到个人利益
- ❌ "你不做会后悔" → 如果对方不相信，反而产生逆反心理

## 10. 避免"司机知识"的自检

**普朗克知识**：真正理解，经过深入学习和思考
**司机知识**：鹦鹉学舌，能说但不真懂

**自检问题**：
- 我能否用自己的话向12岁小孩解释这个概念？
- 面对尖锐的反驳，我能否从容回应？
- 我的知识是否经得起实践检验？
- 如果去掉所有术语，我还剩下什么？

**失败模式**：
- ⚠️ 解释得很流利但经不起追问 → 表面流畅≠真正理解
- ⚠️ 用术语堆砌掩盖不懂 → "这个要看context、nuance和paradigm"
- ⚠️ 能解释但无法应用 → 知道≠会做

**反模式**：
- ❌ "简单来说就是…" → 如果只是背诵别人的简化版本，仍是司机知识
- ❌ "这个很复杂，三言两语说不清" → 可能是自己没真正理解
- ❌ "你不懂这个行业" → 用专业壁垒掩盖理解不足

**检查点**：
- [ ] 我能否用3种不同的方式解释同一个概念？
- [ ] 我能否举出一个反例来说明这个概念的边界？
- [ ] 如果明天要教别人，我准备好了吗？

## 🔴 全局反模式速查

在执行任何技能前，先扫描以下常见陷阱：

| 反模式 | 表现 | 对治 |
|--------|------|------|
| **确认偏误伪装** | 用多元模型包装单一结论 | 强制自己写出最强反面论点 |
| **分析瘫痪** | 无限分析不行动 | 设定决策截止时间 |
| **事后归因** | 成功归能力，失败归运气 | 事前写下预期，事后对比 |
| **过度自信** | "这次我不会犯错" | 用#4检查清单强制扫描 |
| **沉没成本** | "已经投入这么多了" | 问"如果从零开始，我还会做吗？" |
| **锚定效应** | 被第一个数字/信息锚定 | 主动寻找多个独立参考点 |
| **忙碌≠有效** | 做很多事但没有进展 | 每周问"这周最有价值的一件事是什么？" |

## 🏁 决策检查点（任何重大决策前必过）

```
□ 1. 场景路由：我属于哪个场景？该用哪些技能？
□ 2. 能力圈：这个决策在我的能力圈内吗？
□ 3. 逆向思考：什么会让这个决策失败？我正在犯这些错吗？
□ 4. 心理偏误：#4清单中最可能影响我的3个偏误是什么？
□ 5. 双轨验证：理性分析和潜意识分析的结论一致吗？
□ 6. 证据质量：我的信息来源可靠吗？有司机知识混入吗？
□ 7. 可逆性：这个决策可逆吗？不可逆决策需要更高标准
□ 8. 时间检验：10分钟后/10个月后/10年后我会怎么看这个决定？
□ 9. 反面论证：我能否写出一段有力的反对意见？
□ 10. 行动计划：决定后的第一步具体行动是什么？何时执行？
```
