> 从《创造时间：专注于每天最重要的事》中提炼的可执行技能
> 核心方法：挑选（Pick）→ 检验（Try）→ 重复（Repeat）

|------|----------|------|----------|

## 每日启动清单

```
□ 今天我的重点项是什么？（写下来）
□ 安排到日程表了吗？（封锁时间块）
□ 主要分心源已屏蔽？（至少一个屏障）
□ 身体状态如何？（运动/饮食/睡眠）
□ 睡前花2分钟做笔记
```

## 一、选择重点项的技能

### S1. 写下来
- 每天早上或前一晚，用便利贴/笔记本写下今日重点项
- 贴在电脑、冰箱或桌上，持续提醒自己

### S2. 偷天换日（重复昨天）
- 没想法？重复昨天的重点项——巩固势头或形成习惯
- 适用于：未完成的项目、正在学习的技能、令人愉悦的活动

### S3. 分级评定生活
1. 列出生活中3-10件大事（不限工作）
2. 逐一选出最重要的，排出1-5优先级
3. 圈出第一项，用清单辅助每日决策
4. 定期重排——优先级会随阶段变化

### S4. 打包小事件
- 把琐碎任务攒成一天集中处理（如"处理邮件日"）
- 偶尔为之，让平时放心忽略小事，专注重点项

### S5. 或办清单（替代待办清单）
- 将"可能会做的事"记在候选池中
- 每天从中挑选一项排入日程，作为重点项
- 分离"决定"与"行动"，避免择易而行

### S6. 火炉清单
- 前火炉：1个最优先项目 + 其任务列表（大量留白）
- 后火炉：1个次优先项目 + 其任务列表
- 水槽：其他杂项
- 每几天重制一次，一次只能有一件"前火炉"

### S7. 个人冲刺
- 连续多天专注同一重点项，保持"大脑内存"不卸载
- 适合写作、学技能、做项目——比分散几周更高效

## 二、为重点项创造时间的技能

### S8. 安排重点项
- 确定需要多少时间 → 确定何时做 → 排入日程表
- 日程表上的时间就是承诺，强迫你做出取舍

### S9. 封锁日程表
- 每天封锁1-2小时作为"请勿安排"时间
- 打主动进攻牌——用于重点项，而非仅仅回避会议
- 别人问起就说："我已经有安排了"

### S10. 推平日程表
- 压缩会议时长（1小时→30分钟）
- 把事情推到下午或集中到某几天
- 简短座谈会替代长会——皆大欢喜

### S11. 推掉不那么重要的事
- 忙到极致时，问自己能取消什么
- 诚实告知原因，必要时道歉

### S12. 只需说"不"
- 脚本示例：
  - "我真的有些重要项目在忙，没有时间揽新任务"
  - "我没有时间出色地完成这件事"
  - "酸糖防卫"：先拒绝，再推荐合适人选或表示感谢

### S13. 设计你的一天
- 按小时甚至半小时规划日程
- 晚间回顾：什么有效、什么无效
- 系统性规划反而带来"专注于当下"的自由

### S14. 成为晨型人
- 光线：醒来后开所有灯，看日出
- 咖啡仪式：缓慢的手冲过程帮大脑苏醒
- 前夜计划：确定睡眠时长，调暗灯光，远离屏幕

### S15. 夜间重点项时间
- 先补充精力（真正的休息）→ 离线（定时断网）→ 别忘了放松（按时上床）

### S16. 做完就停下
- 画出终点线——完成重点项即可收工
- 力竭前停下比硬撑"多做一件"更高效

## 三、聚焦的技能

### S17. 无干扰手机
- 卸载社交App、删除邮箱、移除浏览器
- 保留工具类App（地图、音乐、日历）
- 可作为24小时试验，不必永久

### S18. 退出登录
- 用完即退，下次需要时再登录
- 进阶：把密码改成记不住的复杂密码

### S19. 关闭消息推送
- 逐一关闭，只保留日历提醒和短信
- 新App一律选"不显示推送"

### S20. 清空主界面
- 所有App图标移到第二屏
- 首屏只留壁纸——每次解锁多一秒停顿

### S21. 戴腕表
- 看时间不必掏手机，避免被屏幕吸入

### S23. 跳过早晨签到
- 醒来后尽量推迟查看邮件/社交媒体
- 拖得越久，休息感越持久，越容易进入聚焦

### S24. 封锁分心氪石
- 识别你的"氪石"——那个一刷就停不下来的东西
- 退出登录/屏蔽网站/卸载App/断网

### S25. 忽视新闻
- 每周看一次即可（如《经济学人》周刊）
- 真正重要的新闻会自己找上门来

### S26. 完毕后归位
- 每天结束时关闭所有标签页、退出所有账户
- 次日打开电脑就是干净的桌面

### S28. 装置假日定时器
- 把路由器接到定时器上，设定自动断网时间
- 最低意志力成本的强制离线

### S30. 提防时间坑洞
- 一条推特→查动态→酝酿下一条 = 30分钟坑洞
- 识别并避开你的时间陨石坑

### S31. 假赢换成真赢
- 邮件、社交媒体的"完成感"是假赢
- 提醒自己：重点项才是真赢

### S32. 化分心为工具
- 想清楚为什么用某个App → 何时用 → 用多久
- 有目的地使用，而非无意识地刷

### S34. 在一天结束时处理邮件
- 黄金时段留给重点项
- 精力较低时处理邮件反而更高效（不易过度承诺）

### S35. 安排邮件时间
- 在日程表上设定固定邮件处理时段

### S37. 把邮件当信件
- 一天只投递一次的信件，你不会每分钟检查信箱

### S38. 慢些回复
- 不必秒回——那是在说"别人的优先项高于我的"
- 相信重点项、相信聚焦模式、相信他人会找到你

### S39. 重设期望值
- 告知同事："有重要项目在忙，紧急事务请发短信"

## 四、找到节奏的技能

### S48. 关门
- 关门或戴耳机 = 向自己和他人宣告"现在是聚焦时间"

### S49. 设定截止日期
- 报名比赛、发出邀约、让朋友监督
- 截止日期是设计冲刺的核心秘诀

### S50. 爆破重点项
- 把大任务拆成有动词的小步骤
- 大脑认为可操作时，积极性和方向感最强

### S51. 播放聚焦歌单
- 为不同类型的专注活动设定专属音乐
- 音乐成为习惯回路的暗示

### S52. 设置可视定时器
- 让时间"可见"——红色圆盘随时间消减
- 产生良好的紧迫感

### S54. 纸上工作
- 纸笔比电脑更不易分心，更易激发创意

### S55. 创建"随机问题"清单
- 想查手机时，把问题写下来，稍后再查

### S56. 感受呼吸
- 一次深呼吸足以重设注意力

### S57. 拥抱无聊
- 无聊时思绪漫游，往往到达有趣的地方
- 研究显示：无聊状态更擅长创造性解决问题

### S60. 全身心投入
- 疲劳的解药不一定是休息，有时是全身心投入
- 抛却忧虑，带着热情和真诚投入当下

## 五、保持活力的技能

### S61. 每天锻炼（但别逞能）
- 目标：约20分钟/天
- 10分钟也行，总比没有好
- 不完美没关系——放弃"真正的运动"执念

### S62. 走路
- 改变默认出行方式：提早一站下车、停远一点
- 走路时可思考、听播客、打电话

### S63. 制造不便
- 自己做饭、走楼梯、手提行李
- 把便利从"默认"变为"理性选择"

### S64. 超短时锻炼
- 7分钟训练或JZ的3×3（俯卧撑+深蹲+引体向上）
- 高强度间歇训练效果可能优于长时间温和锻炼

### S65. 吃真正的食物
- Michael Pollan的箴言：吃食物，别吃太多，多吃植物
- 厄克能认出的未加工食材

### S66. 餐盘中央公园化
- 先放沙拉（绿色区域），再围绕它添加其他食物

### S67. 保持饥饿（间歇性断食）
- 偶尔少吃一顿，让思维更清晰
- 适用于：晨间工作前不吃早餐

### S68. 像孩童一样吃零食
- 备好高质量零食（坚果、水果、花生酱）
- 饿了就吃真正的食物，而非垃圾食品

### S69. 吃黑巧克力
- 替代高糖甜食，稳定精力水平

### S70-76. 善用咖啡因
- **S70**: 9:30后喝第一杯（等皮质醇自然唤醒）
- **S71**: 在精力触底前30分钟摄入（别等累了才喝）
- **S72**: 咖啡因小憩——喝完咖啡立刻睡15分钟
- **S73**: 绿茶保持全天稳定能量
- **S74**: 为重点项涡轮加速——开始前冲杯咖啡
- **S75**: 找到你的"最后咖啡因时刻"（半衰期5-6小时）
- **S76**: 咖啡因和甜食分开

### S77. 拥抱自然
- 公园散步可提升20%认知能力
- 任何自然环境都对大脑电池有益

### S78. 巧入冥想
- 用引导型App，从3分钟开始
- 不必莲花坐——走路、坐公交时都可以
- 如果"冥想"让你不适，叫它"安静时间"

### S79. 把耳机放在家里
- 偶尔让大脑享受真正的安静

### S80. 真正的休息
- 无屏休息：望窗外、散步、吃零食、聊天
- 屏幕休息不会恢复精力，反而消耗更多

### S81. 与部落共度时光
- 列出"能量给予者"——每次见面都让你脚步轻快的人
- 抽出时间面对面交流，而非屏幕互动

### S82. 吃饭不看屏幕
- 一举三得：更健康、更社交、更放松

### S83-86. 穴居睡眠
- **S83**: 卧室只用于睡觉——移除所有电子设备
- **S84**: 伪造日落——晚餐后调暗灯光，设备开夜间模式
- **S85**: 偷闲打盹——10-20分钟即可恢复精神
- **S86**: 避免作息颠倒——每天同一时间起床

### S87. 先戴上自己的氧气罩
- 照顾他人之前先照顾好自己
- 照料好自己才有精力为重要事情创造时间

## 六、反思技能

### 每日笔记模板
```
日期：________
今日重点项：________
为重点项创造时间了吗？ □是 □否
聚焦程度：□高 □中 □低
能量水平：1-10 ____
今天用了什么策略？ ____
什么有效？ ____
什么无效？ ____
感恩时刻：________
```

### 实验心态
- 每个策略都是实验，不奏效就换一个
- 失败不是自责的理由，而是明天重试的数据点
- 完美不存在——灵活的习惯胜过修道院誓言

## ⚠️ 失败模式与对策

| 失败模式 | 典型表现 | 根因 | 对策 |
|----------|----------|------|------|
| **重点项从未发生** | 每天都写但从未执行 | 没排进日程表 / 时间块不够具体 | S8 安排重点项 + S9 封锁日程表 |
| **分心循环** | "就看一下"变成30分钟 | 缺乏物理屏障 | S24 封锁氪石 + S28 定时断网 |
| **虚假忙碌** | 整天处理邮件/消息，感觉很忙 | 假赢替代真赢 | S31 假赢换真赢 + S34 晚间处理邮件 |
| **精力崩溃** | 下午2点后脑雾、犯困 | 睡眠不足 / 缺乏运动 / 饮食不当 | S86 规律作息 + S61 每天锻炼 + S65 吃真正的食物 |
| **完美主义瘫痪** | 等"准备好了"才开始 | 重点项太大、太模糊 | S50 爆破重点项 + S7 个人冲刺 |
| **说"是"太快** | 答应了太多事，时间被瓜分 | 社交压力 / 害怕错过 | S12 说"不" + S11 推掉不重要的事 |
| **晨间沦陷** | 一醒来就刷手机，一天基调已定 | 手机在床头 / 没有晨间仪式 | S14 成为晨型人 + S23 跳过早晨签到 |
| **策略堆叠** | 一次引入太多策略，全部放弃 | 贪多嚼不烂 | 每次只实验1-2个策略，至少坚持一周 |

## 🚫 反模式（避免这些做法）

| 反模式 | 为什么有害 | 正确做法 |
|--------|-----------|----------|
| **待办清单当重点项** | 长清单导致择易而行，重要的永远排后面 | 用或办清单（S5），每天只选1项排入日程 |
| **用忙碌感衡量价值** | 邮件/会议的"完成感"是假赢 | 以重点项是否完成为唯一衡量标准 |
| **意志力硬扛分心** | 意志力是有限资源，终会耗尽 | 用物理屏障代替意志力（卸载、断网、定时器） |
| **屏幕休息** | 刷手机不是休息，是另一种消耗 | 无屏休息：散步、望窗、聊天（S80） |
| **完美晨间流程** | 2小时晨间仪式 = 迟到 + 焦虑 | 从1个简单动作开始（写重点项 + 开灯） |
| **永远"明天开始"** | 等待完美时机 = 永远不开始 | 今天就试一个策略，哪怕只做10分钟 |
| **周末补觉** | 打乱生物钟，周一更累 | 每天同一时间起床（S86），周末也一样 |
| **同时打两场仗** | 既想戒手机又想早起又想运动 | 一次只改变一个变量，稳定后再加 |

## ✅ 检查点（每周回顾）

每周日花10分钟回答以下问题：

### 本周数据
- [ ] 本周5个工作日中，重点项完成了几天？ /5
- [ ] 平均聚焦程度：高 / 中 / 低
- [ ] 平均能量水平：____/10
- [ ] 最常用的策略是哪个？

### 模式识别
- [ ] 是否出现失败模式？哪个？
- [ ] 是否掉入反模式？哪个？
- [ ] 什么时间段精力最好？最差？

### 下周调整
- [ ] 保留哪个策略？（继续有效）
- [ ] 调整哪个策略？（需要微调）
- [ ] 新试哪个策略？（从场景路由选一个）
- [ ] 下周的重点项主题是什么？

### 每月里程碑
- 第1周：找到你的重点项选择方式（S1-S7选一个）
- 第2周：建立时间封锁习惯（S9每天执行）
- 第3周：封锁至少一个分心源（S24）
- 第4周：回顾数据，确认哪些策略值得保留

## 快速启动方案（推荐新手）

| 维度 | 策略 | 说明 |
|------|------|------|
| 重点项 | S8 安排重点项 | 把今日最重要的一件事排进日程 |
| 聚焦 | S24 封锁分心氪石 | 识别并屏蔽你最上瘾的一个App |
| 活力 | S62 走路 | 多走几分钟，给大脑充电 |
| 反思 | 连续三晚做笔记 | 记录结果，看学到了什么 |

> **新手起步口诀**：每天早上写1个重点项 → 排进日程 → 屏蔽1个分心源 → 晚上写2分钟笔记。坚持一周后再扩展。
