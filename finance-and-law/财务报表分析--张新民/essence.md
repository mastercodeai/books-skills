# 财务报表分析（张新民 钱爱民）

> 张新民教授创立的"张氏财务分析框架"——从战略视角分析中国企业财务报表，超越了美国式比率分析的局限。

## 一、财务报表分析概论

### 1. 分析方法演进
- **信用分析**（19世纪末）→ 银行对借款企业资产负债表的偿债能力分析
- **投资分析**（1920s）→ 股东关注盈利能力，利润表成为更重要报表
- **内部分析**（二战后）→ 企业主动自我分析，面向管理决策

### 2. 四大基本分析方法

| 方法 | 核心思路 | 应用 |
|------|----------|------|
| **结构分析法** | 某总体指标=100%，算各项目占比 | 资产结构、利润结构 |
| **趋势分析法** | 连续多期对比，判断变动方向和幅度 | 环比、定基分析 |
| **比率分析法** | 关联项目对比，揭示经济活动特征 | 偿债/营运/盈利指标 |
| **比较分析法** | 与行业先进水平、竞争对手对比 | 判断竞争地位 |

**核心观点：财务报表分析的灵魂在于对比。**

### 3. 项目质量分析法（本书特色）
- 对资产负债表、利润表、现金流量表各项目展开质量分析
- 在项目质量基础上判断企业财务状况的整体质量
- 是比率分析法的有益补充

## 二、财务报表分析基础

### 1. 三张基本报表
- **资产负债表**：特定日期的财务状况（资产=负债+所有者权益）
- **利润表**：一定时期的经营成果
- **现金流量表**：一定时期的现金流入流出

### 2. 四大会计假设
| 假设 | 含义 | 对报表的影响 |
|------|------|------------|
| 会计主体 | 界定核算空间范围 | 决定报表反映的特定单位 |
| 持续经营 | 企业将持续经营下去 | 资产按历史成本而非清算价值计量 |
| 会计分期 | 将持续经营划分为期间 | 产生了权责发生制、应收应付等 |
| 货币计量 | 以货币为统一计量单位 | 忽略非量化信息（如团队能力） |

### 3. 八大会计原则
- **客观性**：如实反映，不偏不倚
- **相关性**：信息与决策需求相关
- **可比性**：口径一致，便于横向纵向比较
- **实质重于形式**：按经济实质而非法律形式处理
- **重要性**：重大项目单独反映
- **谨慎性**：不低估负债费用，不高估资产收益（但可能增加操纵空间）
- **及时性**：及时确认、计量、报告
- **配比原则**：收入与相关费用在同一期间确认

### 4. 确认基础与计量属性
- **权责发生制**（普遍采用）：按业务实际发生时间确认，产生"有利润没钱"现象
- **收付实现制**：按实收实付确认，含金量高但不反映真实业绩
- 现金流量表弥补权责发生制的不足

五种计量属性：历史成本 → 重置成本 → 可变现净值 → 现值 → 公允价值

## 三、资产负债表分析（第3章——本书核心）

### 1. 资产项目质量分析

#### 流动资产

| 项目 | 质量分析要点 |
|------|------------|
| **货币资金** | 规模恰当性（过多=低效，过少=紧张）、受限资金、与经营规模匹配 |
| **应收票据** | 银行承兑 > 商业承兑；到期能否收回 |
| **应收账款** | 规模真实性、周转性（账龄结构）、保值性（坏账准备）；客户集中度 |
| **预付款项** | 交易真实性的判断、与存货关系、关联交易风险 |
| **存货** | 构成及规模恰当性、周转性（存货周转率）、保值性（减值）；毛利率趋势 |
| **其他应收款** | "垃圾桶"项目——可能隐藏大股东资金占用、违规担保等 |

#### 非流动资产

| 项目 | 质量分析要点 |
|------|------------|
| **长期股权投资** | 盈利性（成本法vs权益法）、保值性、战略含义 |
| **固定资产** | 规模恰当性、分布配置合理性、盈利性、周转性（固定资产周转率） |
| **无形资产** | 专利/土地使用权的价值、商誉的减值风险 |
| **在建工程** | 转固进度、是否存在虚增（如万福生科造假案例） |

**资产质量三维特征：盈利性、周转性、保值性**

#### 资源配置战略（资产重分类）
- **经营性资产**：货币资金、应收、存货、固定资产等 → 主营业务相关
- **投资性资产**：长期股权投资、交易性金融资产等 → 资本运作相关
- 三类企业：经营主导型、投资主导型、经营与投资并重型

### 2. 负债质量分析

| 类型 | 内容 | 质量含义 |
|------|------|---------|
| 流动负债 | 短期借款、应付票据/账款、预收款项、应付职工薪酬等 | 短期偿债压力 |
| 非流动负债 | 长期借款、应付债券、长期应付款、预计负债等 | 长期偿债压力 |

**核心关注**：
- 预收款项=竞争力信号（下游占用客户资金）
- 应付票据/账款=议价能力（上游占用供应商资金）
- 预计负债=潜在风险（如诉讼、产品质量保证）

### 3. 资本结构质量分析

资本结构质量五维评价：
1. **资本成本与资产报酬率的对比关系**：资产报酬率 > 资本成本 → 净资产增加
2. **资本期限结构与资产结构的协调性**：长期资产用长期资金、短期波动用短期资金
3. **面对未来资金需求的财务弹性**：过高财务杠杆 → 融资弹性差
4. **控制权结构与治理结构的合理性**：股权结构 → 董事会 → 治理效率
5. **利益相关者之间的和谐性与公平性**

### 4. 资本引入战略分析（张氏框架核心创新）

将资产负债表右边重新分类为四类资本：

| 资本类型 | 来源 | 特征 | 战略类型 |
|---------|------|------|---------|
| **经营性负债资本** | 应付票据/账款、预收款项 | 零/低成本，锁定上下游 | 经营驱动型 |
| **金融性负债资本** | 借款、债券 | 有利息成本，偿还压力大 | 举债融资驱动型 |
| **股东投入资本** | 股本、资本公积 | 企业发展原动力 | 股东驱动型 |
| **企业留存资本** | 盈余公积、未分配利润 | 最稳定内部融资来源 | 利润驱动型 |

**经营驱动型企业**特征（如格力电器）：
- 占用上下游资金能力强
- 资本成本低
- 偿债压力较小
- 固化上下游经济联盟

## 四、利润表分析（第4章）

### 1. 利润结构层次

```
营业收入
- 营业成本
= 毛利（初始获利空间）

核心利润 = 毛利 - 税金及附加 - 销售费用 - 管理费用 - 研发费用 - 财务费用
（衡量主营业务的持续盈利能力）

营业利润 = 核心利润 + 其他收益 + 投资收益 + 公允价值变动 + 资产处置收益等

利润总额 = 营业利润 + 营业外收入 - 营业外支出

净利润 = 利润总额 - 所得税费用
```

**利润的三大支柱**：核心利润、投资收益、营业外收入

### 2. 营业收入质量分析（"五看"）

| 维度 | 分析要点 |
|------|---------|
| **卖什么** | 产品/品种构成、集中度、与战略吻合性 |
| **卖给谁（地区）** | 区域构成、国际化程度、地区经济前景 |
| **卖给谁（客户）** | 客户集中度、大客户依赖风险、回款质量 |
| **靠什么（关联交易）** | 关联交易对收入贡献、交易价格市场化程度 |
| **靠什么（政府补助）** | 行政手段对业绩的贡献、政策持续性 |

### 3. 期间费用质量分析

| 费用 | 分析要点 |
|------|---------|
| **销售费用** | 规模增长 vs 收入增长的匹配性；广告/促销费用的长期效应 |
| **管理费用** | 规模合理性、职工薪酬变化、对人的激励效应 |
| **研发费用** | 占收入比（与同行比较）、未来竞争力影响、滞后效应 |

**核心观点：期间费用不要追求最小化，要追求成本效益最大化。**

### 4. 利润质量恶化的外在表现
- 营业收入与营业成本变动趋势异常
- 核心利润与经营活动现金流持续背离
- 投资收益大幅波动
- 关联交易对利润贡献异常
- 资产减值损失大幅计提或转回
- 营业外收入成为利润主要来源
- 会计政策/估计变更对利润影响重大

## 五、现金流量表分析（第5章）

### 1. 三类现金流量

| 类别 | 流入 | 流出 | 战略含义 |
|------|------|------|---------|
| **经营活动** | 销售商品收到的现金 | 购买商品/支付职工/税费 | 自身造血能力 |
| **投资活动** | 收回投资/处置资产 | 购建资产/对外投资 | 扩张与收缩方向 |
| **筹资活动** | 吸收投资/取得借款 | 偿还债务/分配股利 | 融资能力与分红 |

### 2. 关键分析比率

| 指标 | 公式 | 含义 |
|------|------|------|
| **现金比率** | 现金/流动负债 | 最保守的短期偿债能力 |
| **经营活动净现金比率** | 经营现金流净额/平均流动负债 | 经营活动偿债保障程度 |
| **到期债务本息偿付比率** | 经营现金流净额/(到期债务+利息) | 到期债务的现金保障 |
| **核心利润获现率** | 经营现金流净额/核心利润 | 利润含金量（应≥1） |

### 3. 现金流量质量分析框架

**经营活动现金流量**：
- 流入量的充足性
- 流出量的合理性
- 净额的稳定性及对战略的支持

**投资活动现金流量**：
- 流出与企业战略的吻合性（扩张？收缩？）
- 流入的盈利性

**筹资活动现金流量**：
- 与经营/投资活动的适应性
- 筹资结构的合理性
- 对企业战略的支持

## 六、合并报表分析（第6章）

### 1. 合并报表的特征与局限
- 反映集团整体财务状况，但不反映个别企业实际状况
- 合并过程中内部交易抵销使部分信息消失
- 不同行业子公司合并后，比率指标可能失去意义
- 合并报表通常不能直接作为母子公司决策依据

### 2. 差量分析法（合并报表特有方法）

通过比较合并报表与母公司报表的差量，揭示集团管理特征：

| 分析内容 | 方法 |
|---------|------|
| **控制性投资的规模** | 母公司长期股权投资 + 其他应收款等"隐藏"投资 |
| **子公司整体资产规模** | 合并资产 - 母公司资产（越大=子公司占用资源越多） |
| **汇集资金规模** | 母公司其他应付款 - 合并其他应付款（差额=子公司汇交资金） |
| **母子公司业务关联度** | 对比母公司/合并的营收、成本、销售费用等差异 |
| **盈利能力与费用效率** | 比较母公司/合并的毛利率、费用率差异 |

### 3. 母子公司业务关联度三种模式
- **一致对外销售**：合并营收远大于母公司（如特变电工）
- **母公司生产→子公司销售**：母公司销售费用远小于合并（如贵州茅台）
- **子公司为母公司配套**：合并成本小于母公司成本（如卫光生物）

## 七、财务状况质量的综合分析

### 1. "八看"分析框架（张新民战略视角）

| 维度 | 看什么 | 透视什么 |
|------|-------|---------|
| 一看资产 | 资产规模与结构 | 资源配置战略 |
| 二看负债 | 负债规模与结构 | 经营风险 |
| 三看权益 | 所有者权益构成 | 资本引入战略 |
| 四看收入 | 收入构成与质量 | 核心竞争力 |
| 五看成本费用 | 成本费用结构与效率 | 管理质量 |
| 六看利润 | 利润结构与质量 | 可持续盈利能力 |
| 七看现金流 | 三类现金流质量 | 战略支撑能力 |
| 八看发展 | 趋势与前景 | 企业价值与风险 |

### 2. 杜邦分析体系（经典比率分析）

```
净资产收益率 = 净利润/净资产
= 销售净利率 × 总资产周转率 × 权益乘数
（盈利能力） （营运能力） （财务杠杆）
```

**核心启示**：不同企业提升ROE的路径不同
- 经营主导型企业：靠销售净利率和总资产周转率
- 投资主导型企业：靠权益乘数（杠杆）

### 3. 比率分析的陷阱
- 母公司报表和合并报表的比率可能截然不同
- 不同行业的子公司合并后比率失去可比性
- 存货周转率在多元化企业集团中可能毫无意义
- 会计政策选择和估计会严重影响比率的可比性
- 需结合项目质量分析法弥补比率分析的不足

## 核心案例：北陆药业（2019）

全书以北陆药业2019年年报贯穿始终，展示完整的分析过程：
- 资产负债表：货币资金充裕、无银行借款、经营性负债为主（经营驱动型）
- 利润表：对比剂占收入81%、核心利润率持续提高、销售费用大幅增长
- 现金流量表：经营活动现金流充足、核心利润获现率良好
- 战略：聚焦主业+精准医疗投资布局
