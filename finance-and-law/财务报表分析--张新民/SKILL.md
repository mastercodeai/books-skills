# SKILL.md — 财务报表分析（张新民框架）

> 基于张新民《财务报表分析》的实战分析技能体系。核心理念：超越美国式比率分析，以战略视角和项目质量分析法透视中国企业财务状况。

---

## 一、完整分析流程

```
Step 1: 审计报告 → 确认报表可信度
Step 2: 资产负债表分析 → 企业"底子"（资源与风险）
Step 3: 利润表分析 → 企业"面子"（盈利能力与质量）
Step 4: 现金流量表分析 → 企业"日子"（造血能力）
Step 5: 合并报表差量分析 → 集团管理特征
Step 6: 综合判断 → 财务状况整体质量评价
```

---

## 二、资产负债表分析技能

### 2.1 资产质量分析三维模型

对每个重要资产项目，从三个维度评估：

| 维度 | 含义 | 关键指标 |
|------|------|---------|
| **盈利性** | 资产能带来多少收益 | 毛利率、投资收益率 |
| **周转性** | 资产转化为收入的速度 | 周转率、周转天数 |
| **保值性** | 资产账面价值与可收回价值的差距 | 减值准备、账面vs公允 |

### 2.2 重点资产项目速查

**货币资金**：
- 规模是否恰当？过多=资金闲置低效；过少=流动性紧张
- 是否有受限资金？（保证金、质押存款）
- 与经营规模是否匹配？

**应收账款**：
- 账龄结构：1年内占比越高，质量越好
- 坏账准备计提是否充分？是否存在异常大额单项计提？
- 前五大欠款方集中度
- 周转率趋势：下降→回款变慢→警惕收入质量

**存货**：
- 构成分析：原材料/在产品/库存商品比例
- 毛利率趋势：异常波动→可能"低转成本"虚增利润
- 营业成本与期末存货的此消彼长关系
- 存货周转率：下降→产品滞销或囤积

**长期股权投资**：
- 成本法 vs 权益法的根本区别
  - 成本法：投资收益 = 实际收到的股利（实打实的现金）
  - 权益法：投资收益 = 被投资企业净利润 × 持股比例（"泡沫"资产）
- 年度内重大变动的含义：追加投资/收回/权益法确认收益
- 投资方向是否与核心竞争力一致

**固定资产**：
- 固定资产原值变化 → 透视企业战略（扩张/收缩/优化）
- 生产用 vs 非生产用比例
- 固定资产周转率 → 行业对比，过高或过低均异常
- 虚增固定资产的信号（如在建工程长期不转固）

**其他应收款**：
- ⚠️ "垃圾桶"项目——重点关注！
- 异常大额可能意味着：大股东资金占用、违规担保、隐性关联方交易
- 账龄异常、坏账准备集中100%计提 → 重大风险信号

### 2.3 资产负债表重分类——战略视角

**左边（资产）→ 资源配置战略**：
```
经营性资产（货币资金、应收、存货、固定资产、无形资产等）
    vs
投资性资产（长期股权投资、交易性金融资产等）

→ 经营主导型 / 投资主导型 / 并重型
```

**右边（资本）→ 资本引入战略**：
```
经营性负债资本（应付票据/账款、预收款项）→ 零成本，锁定上下游
金融性负债资本（借款、债券）→ 有息负债，偿债压力
股东投入资本（股本、资本公积）→ 企业原动力
企业留存资本（盈余公积、未分配利润）→ 最稳定内部融资

→ 经营驱动型 / 举债融资驱动型 / 股东驱动型 / 利润驱动型 / 并重驱动型
```

### 2.4 资本结构质量五维评估

1. **成本效益关系**：资产报酬率 > 加权资本成本？
2. **期限结构协调性**：长期资产配长期资金？有无"短融长投"？
3. **财务弹性**：当前杠杆下还能否获得新融资？
4. **控制权合理性**：股权集中度、董事会治理效率
5. **利益和谐性**：股东间、股东与管理层间是否有冲突？

## 三、利润表分析技能

### 3.1 利润结构层次分析

```
毛利率 = 毛利 / 营业收入    → 初始获利空间、产品竞争力
核心利润率 = 核心利润 / 营业收入  → 主营业务的持续盈利能力
营业利润率 = 营业利润 / 营业收入  → 含投资收益等的综合盈利能力
净利率 = 净利润 / 营业收入    → 最终获利能力
```

**核心利润** = 营业收入 - 营业成本 - 税金及附加 - 销售费用 - 管理费用 - 研发费用 - 财务费用

### 3.2 营业收入质量"五看"

1. **看品种构成**：主导产品集中度、多元化战略效果、与战略吻合性
2. **看地区构成**：区域分布、国际化程度、地区经济前景
3. **看客户构成**：集中度（前5大客户占比）、大客户依赖风险、议价能力
4. **看关联交易**：占收入比、价格市场化程度、持续性
5. **看政府补助**：占利润比、政策持续性、是否成"利润拐杖"

### 3.3 期间费用分析原则

> 不要追求费用最小化，要追求成本效益最大化。

- **销售费用**：增长 vs 收入增长的匹配性 → 费用有效性
- **管理费用**：职工薪酬变化 → 人才激励与保留
- **研发费用**：占收入比 vs 同行 → 未来竞争力（有滞后效应）
- **利息费用**：财务杠杆成本，与借款规模和利率匹配

### 3.4 利润质量恶化八大信号

1. 营收增长但经营现金流下降
2. 核心利润占比持续下降，投资收益/营业外收入占比上升
3. 关联交易对利润贡献异常增大
4. 应收账款增速远超营收增速
5. 存货异常积压，毛利率异常波动
6. 大额资产减值"洗大澡"（如乐视网）
7. 频繁会计政策/估计变更
8. 核心利润获现率持续低于1

## 四、现金流量表分析技能

### 4.1 三类现金流组合判断企业状态

| 经营 | 投资 | 筹资 | 企业状态 |
|------|------|------|---------|
| + | - | + | 高速成长期（经营赚钱+借钱+投资扩张） |
| + | - | - | 成熟期（经营赚钱+投资+还债/分红） |
| + | + | - | 衰退期/回收（经营+卖资产+还债） |
| - | + | + | 危机期（经营亏损+卖资产+借钱维持） |
| - | - | + | 初创期/烧钱期（经营未盈利+投资+融资支撑） |
| - | + | - | 面临破产（三类全部恶化） |

### 4.2 关键现金流量比率

| 指标 | 计算 | 判断标准 |
|------|------|---------|
| 核心利润获现率 | 经营现金流净额 / 核心利润 | ≥1 为优 |
| 经营活动净现金比率 | 经营现金流净额 / 平均流动负债 | 越高越好 |
| 到期债务本息偿付比率 | 经营现金流净额 / (到期债务+利息) | ≥1 为安全 |
| 现金比率 | 期末现金 / 流动负债 | 过高=闲置，过低=紧张 |

### 4.3 现金流量质量判断

**经营活动现金流**：
- ✅ 持续为正、规模稳定增长
- ❌ 季度波动剧烈（深交所问询的典型信号）
- ❌ 与核心利润长期背离（利润高但现金流差）

**投资活动现金流**：
- ✅ 大额流出 = 战略扩张（购建固定资产/对外投资）
- ✅ 大额流入 = 回收投资/战略收缩
- ⚠️ 大量"收到其他与投资活动有关的现金"→ 可能是理财产品

**筹资活动现金流**：
- ✅ 借款能力反映银行对企业的信心
- ⚠️ 过度依赖借款 → 财务风险
- ✅ 大额分红 → 盈利能力强、现金流充裕

## 五、合并报表分析技能

### 5.1 差量分析法核心步骤

```
1. 控制性投资规模 = 母公司长期股权投资（合并报表该数字很小或为零）
   → 差额反映母公司对子公司投资的总规模

2. 子公司整体资产规模 = 合并总资产 - 母公司总资产
   → 差额越大，子公司撬动的资源越多

3. 汇集资金规模 = 母公司负债相关项目 - 合并报表同项目
   → 差额为子公司向母公司汇交的资金

4. 业务关联度：比较母公司与合并的营收、成本、销售费用差异
   → 判断母子公司是"一致对外"还是"内部流转"
```

### 5.2 母公司报表 vs 合并报表选择

| 分析目的 | 应使用 |
|---------|-------|
| 评价母公司自身偿债能力 | 母公司报表 |
| 评价集团整体盈利能力 | 合并报表 |
| 分析母公司对子公司的控制力 | 母公司报表+合并报表对比 |
| 分析集团的资源配置效率 | 合并报表 |
| 评价母公司管理层业绩 | 母公司报表 |
| 了解集团多元化经营全貌 | 合并报表 |

## 六、快速识别企业风险的清单

### 资产负债表红旗 🚩
- [ ] 其他应收款异常大额或大幅增长
- [ ] 应收账款增速 >> 营业收入增速
- [ ] 存货异常积压，周转率持续下降
- [ ] 商誉占净资产比例过高（>30%需警惕）
- [ ] 在建工程长期不转固定资产
- [ ] 资产负债率持续攀升，且以金融性负债为主
- [ ] "短融长投"——流动负债支撑长期资产

### 利润表红旗 🚩
- [ ] 核心利润占净利润比例持续下降
- [ ] 政府补助/营业外收入成为利润主要来源
- [ ] 投资收益大幅波动
- [ ] 关联交易贡献利润占比异常
- [ ] 资产减值损失大额计提（"洗大澡"后转回）
- [ ] 毛利率异常高于同行且无合理解释

### 现金流量表红旗 🚩
- [ ] 经营活动现金流持续为负
- [ ] 核心利润获现率长期远低于1
- [ ] 各季度经营现金流波动剧烈
- [ ] 大量资金通过"收到其他与投资/筹资活动有关的现金"流入
- [ ] 借新债还旧债循环，筹资活动现金流大进大出

## 七、行业差异分析要点

| 行业 | 关注重点 |
|------|---------|
| **制造业** | 固定资产规模与产能利用率、存货周转、原材料成本波动 |
| **医药** | 研发费用投入及占比、销售费用（学术推广）、政府补助 |
| **房地产** | 预收款项（合同负债）、存货（开发成本）、现金流季节性 |
| **零售/快消** | 应付账款议价能力、存货周转效率、经营性现金流 |
| **金融/银行** | 不良贷款率、拨备覆盖率、净息差 |
| **互联网** | 用户增长指标、研发投入资本化比例、现金消耗率 |

## 八、核心公式速查

### 偿债能力
```
流动比率 = 流动资产 / 流动负债          （一般 ≥ 2）
速动比率 = 速动资产 / 流动负债          （一般 ≥ 1）
资产负债率 = 负债总额 / 资产总额         （行业差异大）
权益乘数 = 总资产 / 所有者权益
利息保障倍数 = (利润总额 + 利息费用) / 利息费用
```

### 营运能力
```
应收账款周转率 = 营业收入 / 应收账款平均余额
存货周转率 = 营业成本 / 存货平均余额
固定资产周转率 = 营业收入 / 固定资产原值平均余额
总资产周转率 = 营业收入 / 总资产平均余额
```

### 盈利能力
```
毛利率 = 毛利 / 营业收入
核心利润率 = 核心利润 / 营业收入
销售净利率 = 净利润 / 营业收入
总资产报酬率 = 利润总额 / 总资产平均余额
净资产收益率(ROE) = 净利润 / 所有者权益平均余额
```

### 杜邦分析
```
ROE = 销售净利率 × 总资产周转率 × 权益乘数
    = (净利润/营收) × (营收/总资产) × (总资产/所有者权益)
```

### 现金流量
```
核心利润获现率 = 经营活动现金流量净额 / 核心利润 × 100%
经营活动净现金比率 = 经营活动现金流量净额 / 平均流动负债
现金比率 = 期末现金及现金等价物 / 流动负债
```
