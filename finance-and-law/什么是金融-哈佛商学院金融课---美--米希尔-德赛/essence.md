## 第一部分：金融的基石

### 1. 财务报表——企业的语言

**三大报表**：
- **利润表（Income Statement）**：收入 − 成本 = 利润。关注 EBIT（息税前利润）、EBITDA（息税折旧摊销前利润）和净利润。
- **资产负债表（Balance Sheet）**：资产 = 负债 + 所有者权益。反映某一时点的财务状况。
- **现金流量表（Cash Flow Statement）**：经营活动、投资活动、融资活动三类现金流。

**核心比率框架——杜邦分析（DuPont Framework）**：
```
ROE = 净利润率 × 资产周转率 × 权益乘数
    = (净利润/收入) × (收入/资产) × (资产/权益)
```
- **ROE（净资产收益率）**：衡量股东回报的终极指标。
- **ROA（总资产收益率）**：衡量资产运用效率。
- **EBITDA 利润率**：衡量核心经营盈利能力。

**其他关键指标**：
- 流动比率（Current Ratio）、存货周转率、应收账款周转天数
- 利息覆盖倍数（Interest Coverage Ratio）
- 资产负债率、权益乘数

**要点**：财务报表分析不是记公式，而是理解每个数字背后的商业逻辑。不同行业的"正常"比率差异巨大。

### 2. 货币的时间价值与估值基础

**核心原则**：今天的一块钱比明天的一块钱更值钱。

**折现（Discounting）**：
```
PV = FV / (1 + r)^n
```
- PV = 现值，FV = 终值，r = 折现率，n = 期数

**净现值（NPV）**：
- 将未来所有现金流折现到今天，减去初始投资。
- NPV > 0 → 项目创造价值。

**内部收益率（IRR）**：
- 使 NPV = 0 的折现率。
- IRR > 资本成本 → 值得投资。

**永续年金与增长永续年金**：
- 永续年金：PV = C / r
- 增长永续年金：PV = C / (r − g)（g 为增长率）

**估值方法**：
- **可比公司法（Comps）**：用同行的市盈率(P/E)、EV/EBITDA 等倍数。
- **现金流折现法（DCF）**：预测自由现金流，用加权平均资本成本(WACC)折现。

**要点**：估值是金融的核心技能。所有估值都建立在对未来的假设之上，因此理解假设比计算本身更重要。

## 第二部分：风险与资本结构

### 3. 风险与回报

**核心关系**：高风险高回报，但并非所有风险都应获得补偿。

**风险的分类**：
- **特有风险（Idiosyncratic Risk）**：公司特有，可通过分散化消除。
- **系统性风险（Systematic Risk / Market Risk）**：影响整个市场，无法分散化消除。

**Beta（β）**：
- 衡量个股相对于市场整体的波动性。
- β = 1：与市场同步；β > 1：波动更大；β < 1：波动更小。
- 金融学认为：只有系统性风险（β）才应获得回报。

**资本资产定价模型（CAPM）**：
```
E(R) = Rf + β × (Rm − Rf)
```
- Rf = 无风险利率，Rm = 市场预期回报率
- (Rm − Rf) = 市场风险溢价

**分散化的力量**：
- 持有 20-30 只相关性低的股票，即可消除大部分特有风险。
- 这就是为什么机构投资者和指数基金如此重要。

**要点**：风险不是"波动"那么简单。理解哪些风险应该承担、哪些应该分散，是投资决策的关键。

### 4. 资本结构——债务与股权的权衡

**基本问题**：公司应该用多少债务、多少股权来融资？

**债务的优势**：
- 利息可抵税（税盾效应，Tax Shield）。
- 债务成本通常低于股权成本。
- 过度举债 → 财务困境成本（Cost of Financial Distress）。

**资本结构理论**：
- **MM 定理（Modigliani-Miller）**：在完美市场中，资本结构无关紧要。
- **权衡理论（Trade-off Theory）**：最优资本结构 = 税盾收益与财务困境成本的平衡。
- **啄序理论（Pecking Order Theory）**：公司偏好内部融资 → 债务 → 股权。

**加权平均资本成本（WACC）**：
```
WACC = (E/V) × Re + (D/V) × Rd × (1 − T)
```
- E/V = 股权比例，D/V = 债务比例
- Re = 股权成本，Rd = 债务成本，T = 税率

**杠杆收购（LBO）**：
- 用大量债务收购公司，通过改善经营和债务税盾创造价值。
- Scopia Capital 等案例展示了资本结构决策的实际影响。

**要点**：资本结构不是简单的"借钱便宜就多借"，而是要在税收优势、财务灵活性和破产风险之间找到平衡。

## 第三部分：市场与决策

### 5. 金融市场与投资者

**市场参与者**：
- **机构投资者**：养老基金、捐赠基金、共同基金、对冲基金。
- **投资银行**：承销、并购顾问、做市。
- **被动 vs 主动投资**：指数基金（被动）vs 主动管理基金。

**市场效率假说（EMH）**：
- 弱式：价格反映历史信息。
- 半强式：价格反映所有公开信息。
- 强式：价格反映所有信息（含内幕信息）。
- 现实中市场"大致有效"，但存在系统性偏差。

**2008 年金融危机的教训**：
- 过度杠杆 + 复杂衍生品 + 评级机构失职 = 系统性崩溃。
- 金融市场既是价值发现的工具，也是风险放大的机器。

**被动投资的崛起**：
- 大多数主动基金长期跑不赢指数。
- 费用率差异对长期回报影响巨大。

### 6. 公司财务决策——CEO 与 CFO 的视角

**CFO 的核心职责**：
- 资本预算（Capital Budgeting）：决定投资什么项目。
- 资本结构：决定如何融资。
- 资金管理：现金、分红、回购。
- 风险管理与对冲（Hedging）。

**资本配置（Capital Allocation）**：
- 公司赚到的钱怎么用？再投资、分红、回购、收购？
- 优秀 CFO 的核心能力是资本配置效率。

**收购与兼并（M&A）**：
- 战略收购 vs 财务收购。
- 协同效应（Synergy）的期望 vs 现实。
- 尽职调查（Due Diligence）的重要性。

**代理问题与公司治理**：
- 管理层利益 vs 股东利益的冲突。
- 董事会的监督角色。
- 激励机制设计：期权、限制性股票等。

**案例**：
- Alphabet/Google 的资本配置策略。
- 某些公司 CFO 与 CEO 的分工与协作。
- H.J. Heinz 的私有化案例。

## 全书核心思想

1. **金融是语言**：财务报表和金融概念是理解商业世界的通用语言。
2. **时间有价值**：今天的钱比明天的钱更值钱，这是一切估值的基石。
3. **风险与回报对称**：理解哪些风险有补偿、哪些没有，是投资和融资决策的核心。
4. **市场大致有效**：不要轻易认为自己比市场聪明，但理解市场的局限性同样重要。
5. **资本配置是终极技能**：对个人投资者和企业高管都一样——如何分配稀缺的资本，决定了创造或毁灭多少价值。
