## 一、财务分析技能

### 1.1 快速读懂一家公司
1. **先看利润表**：收入是否增长？利润率趋势如何？EBITDA 利润率是多少？
2. **再看资产负债表**：负债率高不高？现金储备够不够？资产结构是否合理？
3. **最后看现金流**：经营现金流是否为正？自由现金流能否覆盖投资需求？

### 1.2 杜邦分析三步法
```
ROE = 净利润率 × 资产周转率 × 权益乘数
```
- **利润率驱动型**（如奢侈品）：高毛利，低周转
- **周转率驱动型**（如沃尔玛）：薄利多销
- **杠杆驱动型**（如银行）：高负债放大回报

**应用**：当 ROE 变化时，分解三个因素，定位是哪个环节在改善或恶化。

### 1.3 关键财务指标清单

| 指标 | 公式 | 意义 |
|------|------|------|
| 毛利率 | (收入−COGS)/收入 | 产品竞争力 |
| EBITDA 利润率 | EBITDA/收入 | 核心经营盈利能力 |
| ROE | 净利润/股东权益 | 股东回报 |
| ROA | 净利润/总资产 | 资产运用效率 |
| 流动比率 | 流动资产/流动负债 | 短期偿债能力 |
| 利息覆盖倍数 | EBIT/利息费用 | 债务安全性 |
| 存货周转天数 | 365/(COGS/平均存货) | 运营效率 |

## 二、估值技能

### 2.1 DCF 估值五步法
1. **预测自由现金流（FCF）**：未来 5-10 年
2. **确定折现率（WACC）**
3. **计算终值（Terminal Value）**：用增长永续年金公式
4. **折现求和**：所有未来现金流的现值
5. **敏感性分析**：改变关键假设，看估值区间

```
WACC = (E/V)×Re + (D/V)×Rd×(1−T)
终值 = FCF_n × (1+g) / (WACC − g)
```

### 2.2 可比公司估值法
1. 选 5-10 家同行业可比公司
2. 计算 P/E、EV/EBITDA、P/B 等倍数
3. 取中位数，乘以目标公司对应指标
4. 注意：可比公司的选择比计算本身更重要

### 2.3 估值的核心原则
- 估值 = 假设 + 数学。**改变假设比优化公式影响大得多**。
- 永远做敏感性分析：给关键变量 ±10%~20% 的变动。
- 不同方法交叉验证：DCF、可比公司、可比交易。

### 2.4 估值检查点
- [ ] FCF 预测是否基于历史趋势 + 合理增长假设？
- [ ] WACC 中的 Beta 是否用了同行业可比公司均值？
- [ ] 终值是否占总估值 < 60%？（超过则过度依赖永续假设）
- [ ] 增长率 g 是否 ≤ 长期 GDP 增速？（否则不可持续）
- [ ] 是否做了至少 3 组敏感性分析（乐观/基准/悲观）？
- [ ] DCF 结果与可比公司法偏差是否 < 30%？

## 三、投资决策技能

### 3.1 NPV vs IRR 决策规则
| 方法 | 规则 | 局限 |
|------|------|------|
| NPV | NPV > 0 则投资 | 依赖折现率假设 |
| IRR | IRR > 资本成本则投资 | 非常规现金流可能有多个 IRR |
| 回收期 | 越短越好 | 忽略时间价值和后续现金流 |

**优先使用 NPV**，IRR 作为辅助参考。

### 3.2 风险评估框架
1. **识别风险类型**：市场风险（系统性）vs 公司特有风险
2. **评估 Beta**：该投资相对市场波动如何？
3. **分散化检查**：该投资在组合中是否提供了分散化收益？
4. **用 CAPM 定价**：该风险水平下合理回报应该是多少？

```
E(R) = 无风险利率 + β × 市场风险溢价
```

### 3.3 资本结构决策
- **目标**：找到使 WACC 最小的债务/股权比例
- **债务优势**：税盾（利息抵税）
- **债务劣势**：财务困境风险、灵活性下降
- **经验法则**：
  - 稳定现金流行业（如公用事业）→ 可多借债
  - 高波动行业（如科技）→ 宜少借债

### 3.4 投资决策检查点
- [ ] 项目的 NPV 是否为正？IRR 是否 > WACC？
- [ ] 现金流预测是否排除了沉没成本？
- [ ] 是否考虑了机会成本（替代方案的 NPV）？
- [ ] 风险调整后的回报率是否合理（用 CAPM 校验）？
- [ ] 是否做了情景分析（最好/最差/最可能）？

## 四、公司治理与高管决策技能

### 4.1 CFO 资本配置决策树
```
公司有超额现金
├── 内部再投资 → 回报率 > WACC？
│   ├── 是 → 投资
│   └── 否 → 考虑其他选项
├── 分红 → 稳定、可预期
├── 股票回购 → 股价被低估时
└── 收购 → 协同效应明确时
```

### 4.2 并购评估清单
- [ ] 战略逻辑是否清晰？
- [ ] 协同效应是否可量化、可实现？
- [ ] 收购溢价是否合理？
- [ ] 尽职调查是否充分（财务、法律、运营）？
- [ ] 整合计划是否详细？
- [ ] 管理层是否有整合经验？

### 4.3 激励机制设计原则
- 将管理层利益与股东长期利益对齐
- 避免短期主义：限制性股票 > 短期期权
- 董事会独立性：独立董事占比、审计委员会权力
- 透明的信息披露是治理的基石

## 五、金融市场认知技能

### 5.1 被动 vs 主动投资决策
| 维度 | 被动（指数基金） | 主动管理 |
|------|-----------------|---------|
| 费用 | 低（0.03-0.2%） | 高（1-2%） |
| 长期表现 | ~多数跑赢主动 | ~多数跑输指数 |
| 适合 | 大多数投资者 | 有信息优势的专业投资者 |

### 5.2 市场效率的实际应用
- **短期**：很难持续跑赢市场，不要频繁交易
- **长期**：市场存在系统性偏差（动量、价值效应等）
- **实践中**：假设市场大致有效，但保持对极端定价的警觉

### 5.3 金融危机的教训
- 过度杠杆是系统性风险的放大器
- 复杂金融产品往往隐藏着未被识别的风险
- 评级机构存在利益冲突
- 流动性可以在瞬间消失

## 六、关键公式速查

```text
NPV = Σ [FCF_t / (1+r)^t] − 初始投资
IRR：使 NPV = 0 的 r
永续年金 PV = C / r
增长永续年金 PV = C / (r − g)
CAPM：E(R) = Rf + β(Rm − Rf)
WACC = (E/V)Re + (D/V)Rd(1−T)
ROE = 净利润率 × 资产周转率 × 权益乘数
Enterprise Value = 市值 + 净债务
FCF = EBIT(1−T) + 折旧 − ΔNWC − CapEx
```

## 七、失败模式与反模式

### 7.1 估值失败模式

| 失败模式 | 表现 | 根因 | 纠正 |
|---------|------|------|------|
| 垃圾进垃圾出 | DCF 结果精确但偏离实际 | 增长率/折现率假设不切实际 | 用行业基准锚定假设，做敏感性分析 |
| 终值占比过高 | 终值占估值 70%+ | 对永续增长过于乐观 | 延长显式预测期，降低 g |
| 可比公司选错 | 估值偏差巨大 | 业务模式/规模/增速不匹配 | 按 3 维度（行业、规模、增长）筛选 |
| 锚定效应 | 估值总是接近"感觉" | 心理锚定干扰判断 | 先独立估值，再看市场价格 |

### 7.2 资本结构反模式

| 反模式 | 为什么错 | 正确做法 |
|--------|---------|---------|
| "越多杠杆越好" | 忽视财务困境成本 | 计算 WACC 最优点 |
| 模仿同行杠杆率 | 每家公司现金流稳定性不同 | 基于自身现金流波动决定 |
| 低利率就多借 | 利率低≠风险低，周期会反转 | 以经济周期中位利率测试偿债能力 |
| 用短期债投长期项目 | 期限错配引发流动性危机 | 长期资产用长期资金匹配 |

### 7.3 投资决策反模式

| 反模式 | 表现 | 正确做法 |
|--------|------|---------|
| 沉没成本谬误 | "已经投了这么多，不能放弃" | 只看未来增量现金流 |
| 用 IRR 替代 NPV | 多互斥项目选错 | 以 NPV 为主要决策标准 |
| 忽视机会成本 | 只算项目本身收益 | 与最优替代方案比较 |
| 过度自信预测 | 收入增长 30% 持续 10 年 | 用均值回归思维，参考行业均值 |

### 7.4 并购反模式

| 反模式 | 表现 | 正确做法 |
|--------|------|---------|
| 帝国建设 | 为规模而收购，不看回报 | 仅在 NPV > 0 时收购 |
| 协同效应幻觉 | 预估协同 20%，实现 5% | 用历史并购数据校准 |
| 赢家诅咒 | 竞标中出价过高 | 设定价格上限，超过就退出 |
| 忽视整合 | 交割即终点 | 整合计划在交割前完成 |

### 7.5 心理陷阱清单
- **锚定**：先入为主的数据主导判断 → 先独立分析，后看参考
- **确认偏误**：只找支持自己结论的证据 → 主动寻找反面论据
- **损失厌恶**：亏损时死扛不卖 → 设定止损规则并执行
- **从众效应**：别人都投我也投 → 回到 NPV/DCF 做独立判断
- **过度自信**：高估自己预测准确度 → 做敏感性分析，扩大假设区间

## 八、检查点总览

### 8.1 财务分析检查点
- [ ] 三张报表是否交叉验证？（利润表净利润 ≈ 现金流表净利润）
- [ ] 是否至少看了 3 年趋势，而非单一年度？
- [ ] 同行业对比基准是否合理？
- [ ] 是否识别了一次性项目并剔除？

### 8.2 估值检查点（详见 §二.4）
- [ ] 假设是否合理且有据可查？
- [ ] 敏感性分析是否覆盖关键变量？
- [ ] 多种方法是否交叉验证？

### 8.3 投资决策检查点（详见 §三.4）
- [ ] NPV 是否为正？
- [ ] 是否排除了沉没成本？
- [ ] 风险调整后回报是否合理？

### 8.4 并购检查点（详见 §四.2）
- [ ] 战略逻辑 + 协同效应 + 整合计划三者是否齐备？

### 8.5 资本结构检查点
- [ ] 当前杠杆率 vs 行业中位数？
- [ ] 利息覆盖倍数是否 > 3x？
- [ ] 经济下行情景下偿债能力是否充足？
- [ ] 债务期限结构是否匹配资产期限？

## 九、实践行动建议

### 对个人投资者
1. 理解财务报表是看懂公司故事的基础
2. 分散化投资，不要把鸡蛋放在一个篮子里
3. 低费用的指数基金是大多数人的最优选择
4. 长期持有，避免频繁交易

### 对企业决策者
1. 用 NPV 而非直觉做投资决策
2. 资本结构要在税盾和财务安全之间平衡
3. 分红和回购要基于长期价值，而非短期股价
4. 公司治理质量直接影响资本成本

### 对金融从业者
1. 估值的核心是假设，不是公式
2. 理解风险的本质比计算风险更重要
3. 市场参与者的行为往往比理论模型更复杂
4. 2008 年的教训：永远不要忘记尾部风险
