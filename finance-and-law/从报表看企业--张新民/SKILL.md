# 从财务报表看企业——分析技能清单

来源：《从报表看企业——数字背后的秘密》（张新民）

---

## 🧭 场景路由（Scene Routing）

根据分析目标，选择对应报表和技能组合：

| 场景 | 主要报表 | 核心技能 | 辅助技能 |
|------|---------|---------|---------|
| **评估企业战略与家底** | 资产负债表 | 技能一（战略类型）、技能二（扩张效应） | 技能三（资本引入） |
| **判断真实偿债风险** | 资产负债表 | 技能六（金融负债率） | 技能三（资本来源结构） |

**快捷路径：**

- 三表齐全 + 母公司报表 → 全部技能均可执行

## 技能一：识别企业战略类型

**方法：** 以母公司资产负债表为基础，区分经营资产与投资资产的占比。

**操作步骤：**
1. 找出母公司的经营资产：货币资金、应收票据/账款、存货、固定资产、无形资产
2. 找出母公司的投资资产：交易性金融资产、可供出售金融资产、持有至到期投资、长期股权投资
3. 比较两者占比，归类为经营主导型/投资主导型/并重型

**判断标准：**
- 经营资产占主体 → 经营主导型（专业化，如格力电器）
- 投资资产占主体 → 投资主导型（多元化，如美的集团）
- 两者大致均衡 → 并重型（如特变电工）

---

## 技能二：计算控制性投资的扩张效应

**公式：**
- 控制性投资 ≈ 母公司(长期股权投资 + 其他应收款 + 预付款项) - 合并报表对应项目
- 扩张效应 = 合并资产总计 - 母公司资产总计

**解读：**
- 扩张效应 / 控制性投资 > 1 → 有撬动效应，值越大说明扩张效果越好
- 子公司吸纳少数股东入资、自身贷款、经营性负债和利润积累都是扩张来源

---

## 技能三：分析资本引入战略

**方法：** 将负债和股东权益按来源分为四类，计算各自占比。

**四类资源：**
1. 经营性资源 = 应付票据 + 应付账款 + 预收款项（+ 应付职工薪酬 + 应交税费）
2. 金融性资源 = 短期借款 + 交易性金融负债 + 一年内到期非流动负债 + 长期借款 + 应付债券 + 长期应付款
3. 股东入资资源 = 股本（实收资本）+ 资本公积
4. 利润留剩资源 = 盈余公积 + 未分配利润

**判断：**
- 经营性资源为主 → 经营驱动型（竞争优势强，"两头吃"能力突出）
- 金融性资源为主 → 债务融资驱动型（快速扩张期）
- 股东入资为主 → 股东驱动型（初创阶段或经营不善）
- 利润积累为主 → 利润驱动型（成熟企业）

---

## 技能四：评估核心利润质量

**核心利润公式：**
核心利润 = 营业收入 - 营业成本 - 税金及附加 - 销售费用 - 管理费用 - 财务费用

**质量检验（经验法则）：**
- 存货周转 > 2次/年时，经营净现金应 ≥ 核心利润 × 1.2~1.5
- 经营净现金 < 核心利润 → 利润质量堪忧，可能有存货积压或应收膨胀

**利润结构分析：**
- 看三大支柱（核心利润、投资收益、营业外收入）各占净利润比重
- 核心利润占比高 → 可持续；营业外收入/投资收益占比高 → 警惕

---

## 技能五：评估上下游关系竞争力

**采购端（看"吃上游"能力）：**
- 合并报表应付票据+应付账款远大于存货 → 议价能力强
- 预付款项越合并越小 → 可能是向子公司提供资金，非真正预付

**销售端（看"吃下游"能力）：**
- 应收票据占比高 → 回款质量好（银行承兑汇票有保障）
- 年末应收比年初大幅下降 → 回款管理加强
- 预收款项增长 → 预收款能力增强

**综合判断：** 应收结构中票据多、应付结构中票据少 → 上下游均有竞争力

---

## 技能六：识别金融负债率（真实财务风险）

**公式：**
金融负债率 = (短期借款 + 交易性金融负债 + 一年内到期非流动负债 + 长期借款 + 应付债券 + 长期应付款) / 资产总计

**关键区别：**
- 资产负债率高 ≠ 风险高（经营性负债可能占主体，如格力电器85%负债率但金融负债率极低）
- 金融负债率才是真正的偿债风险指标
- 预收款项包含毛利因素，实际负债被夸大

---

## 技能七：识别利润操纵迹象

**存货积压操纵：**
- 存货大幅增加 + 核心利润增长 + 经营净现金不足 → 可能通过存货积压虚增利润
- 关系式：期初存货 + 本期采购 = 营业成本 + 期末存货。期末存货↑ → 营业成本↓ → 毛利↑

**异常减值准备：**
- 减值准备计提异常高 → 可能是"洗大澡"，为下年扭亏做准备
- 减值准备计提异常低 → 可能隐藏资产质量问题

**公允价值变动收益：**
- 最虚的利润，能看不能用，没有现金流量跟进

**小项目大贡献：**
- 营业外收入、投资收益支撑净利润 → 核心业务盈利能力存疑
- 变卖资产维持盈利 → 不可持续

---

## 技能八：现金流质量分析

**经营现金流充分性：**
1. 与核心利润比较（1.2~1.5倍经验标准）
2. 经营净现金须满足：补偿折旧摊销 + 支付利息 + 支付现金股利 + 扩大再生产

**投资现金流的补偿机制：**
- 购建固定资产的现金流出 → 由未来使用期内经营活动净现金逐渐补偿
- 对外投资的现金流出 → 由未来"收回投资"或"取得投资收益"的现金补偿

**筹资现金流关注点：**
- 看流入量来源（股东入资 vs 贷款）和支持方向

---

## 技能九：合并报表分析要点

**可用比率（合并报表）：**
- 净资产收益率、总资产报酬率、毛利率、核心利润率、销售费用率、管理费用率、经营净现金/核心利润

**慎用比率（合并报表）：**
- 存货周转率、固定资产周转率、流动比率、资产负债率（这些在合并口径下可能失去意义）

**"越合并越小"的项目：**
- 其他应收款/预付款项 → 子公司占用资金规模
- 长期股权投资/投资收益 → 控制性投资规模/子公司分红
- 存货/营业收入 → 内部交易规模

**"越合并越大"的项目：**
- 固定资产/在建工程 → 子公司拥有独立生产体系
- 预付款项 → 子公司独立对外支付

---

## 技能十：综合前景预测框架

**第一步：经营活动前景**
- 经营资产结构优化程度 → 核心利润增长潜力 → 经营净现金充足度

**第二步：投资活动前景**
- 控制性投资的扩张效应 → 子公司盈利能力 → 投资回报

**第三步：融资/重组/并购前景**
- 增量融资空间（金融负债率、盈利能力、融资环境）
- 并购的可能性与风险（高商誉并购需警惕）

**红旗信号清单：**
- [ ] 营业收入增长但应收账款大幅膨胀
- [ ] 核心利润增长但经营净现金为负或远低于核心利润
- [ ] 金融负债率持续上升
- [ ] 固定资产/在建工程增速远超营业收入增速
- [ ] 合并报表出现巨额商誉
- [ ] 利润严重依赖营业外收入或投资收益
- [ ] 其他应收款年末比年初异常增长
- [ ] 控股股东通过其他应收款/预付款项占用资金

## ⚠️ 失败模式（Failure Modes）

### F1：只看资产负债率就下结论
- **症状：** 看到资产负债率80%+就判定"高风险"
- **根因：** 未区分经营性负债与金融性负债
- **修正：** 必须计算金融负债率（技能六），经营性负债（应付账款、预收款）本质是竞争优势的体现

### F2：用合并报表算周转率
- **症状：** 用合并存货/合并营业成本计算存货周转率
- **根因：** 合并报表已抵消内部交易，存货与营业成本口径不匹配
- **修正：** 周转率类指标只能用母公司单体报表计算

### F3：忽略母公司与合并的差异
- **症状：** 只看合并报表就得出结论
- **根因：** 合并报表掩盖了母公司与子公司之间的资金关系
- **修正：** 始终对比母公司与合并报表（技能二、技能九），差异本身就是分析对象

### F4：利润高就等于好企业
- **症状：** 净利润增长即判定业绩改善
- **根因：** 未拆解利润来源结构
- **修正：** 检查核心利润占比（技能四），警惕营业外收入/投资收益/公允价值变动撑起的利润

### F5：现金流好就万事大吉
- **症状：** 经营净现金充裕就不再深挖
- **根因：** 可能是拖欠供应商（应付账款暴增）或预收了客户的钱但尚未交付
- **修正：** 结合技能五分析现金流充裕的原因，区分"真强"和"假强"

### F6：跨行业直接对比财务指标
- **症状：** 用制造业标准判断互联网企业
- **根因：** 不同行业资产结构、负债特征、现金流模式差异巨大
- **修正：** 先确认行业特征，再设定合理的比较基准

## 🚫 反模式（Anti-Patterns）

### A1：孤立看单一指标
- **错误：** "毛利率40%，好公司！"
- **正确：** 毛利率需结合核心利润率、经营净现金/核心利润、存货周转一起看

### A2：只看一年数据
- **错误：** 基于单年报表得出趋势判断
- **正确：** 至少看3年数据，观察趋势变化方向和速度

### A3：混淆"能力"与"结果"
- **错误：** 应付账款多说明企业缺钱
- **正确：** 应付账款多可能是议价能力强（"吃上游"），需结合行业地位判断

### A4：忽视会计政策变更
- **错误：** 直接对比前后两年数据
- **正确：** 检查附注中会计政策/估计变更，调整后再对比

### A5：把商誉当资产
- **错误：** 商誉计入总资产，放大了资产规模
- **正确：** 分析时应关注"去商誉"后的净资产，评估真实家底

### A6：过度依赖财务比率
- **错误：** 罗列一堆比率就写完分析报告
- **正确：** 比率是线索不是结论，必须结合业务逻辑解释"为什么"

## ✅ 分析检查点（Checkpoints）

### 阶段一：数据准备
- [ ] 确认报表类型（合并/母公司）及年份范围（至少3年）
- [ ] 检查会计政策是否有重大变更（看附注）
- [ ] 标记母公司与合并报表的关键差异项

### 阶段二：资产负债表分析
- [ ] 完成技能一：识别战略类型（经营/投资/并重）
- [ ] 完成技能二：计算控制性投资与扩张效应
- [ ] 完成技能三：拆解四类资本来源占比
- [ ] 完成技能五：评估上下游议价能力
- [ ] 完成技能六：计算金融负债率，与资产负债率对比

### 阶段三：利润表分析
- [ ] 完成技能四：计算核心利润，检验经营净现金充分性
- [ ] 完成技能四：拆解利润三大支柱占比
- [ ] 完成技能七：逐项排查利润操纵迹象

### 阶段四：现金流分析
- [ ] 完成技能八：评估经营现金流与核心利润的匹配度
- [ ] 完成技能八：分析投资现金流的补偿来源
- [ ] 完成技能八：识别筹资现金流的驱动因素

### 阶段五：综合判断
- [ ] 完成技能九：合并报表特殊项目检查
- [ ] 完成技能十：三表联动前景预测
- [ ] 逐项过红旗信号清单
- [ ] 输出结论：一句话概括企业财务健康度 + 三个关键发现 + 一个最大风险
